# <a name="pidtagproofofsubmission-canonical-property"></a>Propriedade canônica PidTagProofOfSubmission

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Essa propriedade contém TRUE se o remetente da mensagem solicita prova de que o sistema de transferência de mensagem enviou uma mensagem para entrega ao destinatário pretendido originalmente.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_PROOF_OF_SUBMISSION  <br/> |
|Identificador:  <br/> |0x0E26  <br/> |
|Tipo de dados:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |MAPI não transmittable  <br/> |
