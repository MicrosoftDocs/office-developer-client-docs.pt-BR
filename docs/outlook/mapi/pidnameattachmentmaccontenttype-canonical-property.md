# <a name="pidnameattachmentmaccontenttype-canonical-property"></a>Propriedade canônica PidNameAttachmentMacContentType

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém um valor de campo de cabeçalho Content-Type de [RFC3282] para anexos usados por clientes de email baseados em Macintosh.
  
|||
|:-----|:-----|
|Nomes amigáveis:  <br/> |Nenhum  <br/> |
|Conjunto de propriedades:  <br/> |PSETID_Attachment  <br/> |
|Nome da propriedade:  <br/> |AttachmentMacContentType  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Anexo de mensagem  <br/> |
