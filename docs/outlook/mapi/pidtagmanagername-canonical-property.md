# <a name="pidtagmanagername-canonical-property"></a>Propriedade canônica PidTagManagerName

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém o nome do gerente do destinatário. 
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_MANAGER_NAME, PR_MANAGER_NAME_A, PR_MANAGER_NAME_W  <br/> |
|Identificador:  <br/> |0x3A4E  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Área:  <br/> |Endereço  <br/> |
   
## <a name="remarks"></a>Comentários

Essas propriedades fornecem informações de identificação e acesso de um destinatário. Eles são definidos pelo destinatário e sua organização.
