# <a name="cbflatmtsidlist"></a>CbFLATMTSIDLIST

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Calcula o número de bytes em uma estrutura [FLATMTSIDLIST](flatmtsidlist.md) existente. 
  
|||
|:-----|:-----|
|Arquivo de cabeçalho:  <br/> |Mapidefs. h  <br/> |
|Estrutura relacionada:  <br/> |**FLATMTSIDLIST** <br/> |
   
```cpp
CbFLATMTSIDLIST (_lplist)
```

## <a name="parameters"></a>Parâmetros

 __lplist_
  
> Ponteiro para a estrutura **FLATMTSIDLIST** existente.
