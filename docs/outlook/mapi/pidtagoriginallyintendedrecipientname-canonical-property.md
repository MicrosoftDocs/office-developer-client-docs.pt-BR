# <a name="pidtagoriginallyintendedrecipientname-canonical-property"></a>Propriedade canônica PidTagOriginallyIntendedRecipientName

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém o nome codificado do destinatário originalmente pretendido de uma mensagem com origem automática.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_ORIGINALLY_INTENDED_RECIPIENT_NAME  <br/> |
|Identificador:  <br/> |0x0020  <br/> |
|Tipo de dados:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Servidor  <br/> |
   
## <a name="remarks"></a>Comentários

A **PR_ORIGINALLY_INTENDED_RECIPIENT_NAME** propriedade deve ser definida pelo agente automático que encaminhou a mensagem.
