# <a name="pidlidmeetingworkspaceurl-canonical-property"></a>Propriedade canônica PidLidMeetingWorkspaceUrl

  
  
**Aplica-se a**: Outlook 
  
Especifica a URL do espaço de trabalho de reunião que está associado um objeto calendar.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidMWSURL  <br/> |
|Propriedade definida:  <br/> |PSETID_Appointment  <br/> |
|ID de longo (LID):  <br/> |0x00008209  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Reuniões  <br/> |
   
## <a name="remarks"></a>Comentários

Essa propriedade não é necessária.
