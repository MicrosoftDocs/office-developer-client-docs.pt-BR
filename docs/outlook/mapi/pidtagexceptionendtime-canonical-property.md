# <a name="pidtagexceptionendtime-canonical-property"></a>Propriedade canônica PidTagExceptionEndTime

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Indica a data de término e a hora da exceção no fuso horário local do computador quando a exceção é criada.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_EXCEPTION_ENDTIME  <br/> |
|Identificador:  <br/> |0x7FFC  <br/> |
|Tipo de dados:  <br/> |PT_SYSTIME  <br/> |
|Área:  <br/> |Mensagem definida de classe não transmittable  <br/> |
   
## <a name="remarks"></a>Comentários

> [!NOTE]
> Essa propriedade é informativa e não deve ser confiáveis para informações essenciais.
