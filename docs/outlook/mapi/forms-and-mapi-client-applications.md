# <a name="forms-and-mapi-client-applications"></a>Formulários e aplicativos de cliente MAPI

**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Como todas as mensagens MAPI são instâncias de formulários, os aplicativos cliente exibem e interagem com os objetos de formulário como fazem com outros objetos de mensagem.
  
Aplicativos cliente que usam as interfaces de formulário MAPI podem fornecer adicional interação com objetos de formulário. Aplicativos cliente podem usar qualquer verbos especiais definidos por uma forma de fornecer aos usuários com a gama completa de interações destinado pelo criador do formulário. Os clientes também podem usar as propriedades personalizadas que o servidor de formulário expõe para ajudar os usuários classificar, visualizar ou caso contrário, gerenciar formulários na interface do cliente.
  
Para obter mais informações, consulte [Manipulação de formulários de MAPI](handling-mapi-forms.md).
