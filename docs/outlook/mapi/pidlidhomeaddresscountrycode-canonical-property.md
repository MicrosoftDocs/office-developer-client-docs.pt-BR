# <a name="pidlidhomeaddresscountrycode-canonical-property"></a>Propriedade canônica PidLidHomeAddressCountryCode

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Especifica a parte de código do país/região do endereço residencial do contato.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidHomeAddressCountryCode  <br/> |
|Conjunto de propriedades:  <br/> |PSETID_Address  <br/> |
|Long ID (LID):  <br/> |0x000080DA  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Contato  <br/> |
