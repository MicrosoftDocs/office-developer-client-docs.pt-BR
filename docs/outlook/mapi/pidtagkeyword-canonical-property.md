# <a name="pidtagkeyword-canonical-property"></a>Propriedade canônica PidTagKeyword

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém uma palavra-chave que identifica o destinatário para o administrador do sistema do destinatário.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_KEYWORD, PR_KEYWORD_A, PR_KEYWORD_W  <br/> |
|Identificador:  <br/> |0x3A0B  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Área:  <br/> |Endereço  <br/> |
   
## <a name="remarks"></a>Comentários

Essas propriedades fornecem informações de identificação e acesso de um destinatário. Eles são definidos pelo destinatário e sua organização.
