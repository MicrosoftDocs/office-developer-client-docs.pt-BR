# <a name="pidtagkeyword-canonical-property"></a>Propriedade canônica PidTagKeyword

  
  
**Aplica-se a**: Outlook 
  
Contém uma palavra-chave que identifica o destinatário ao administrador do sistema do destinatário.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_KEYWORD, PR_KEYWORD_A, PR_KEYWORD_W  <br/> |
|Identificador:  <br/> |0x3A0B  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Área:  <br/> |Endereço  <br/> |
   
## <a name="remarks"></a>Comentários

Essas propriedades fornecem identificação e acessar informações para um destinatário. Eles são definidos por destinatário e suas respectivas organizações.
