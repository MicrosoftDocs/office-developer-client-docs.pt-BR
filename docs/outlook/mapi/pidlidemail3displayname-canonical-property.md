# <a name="pidlidemail3displayname-canonical-property"></a>Propriedade canônico de PidLidEmail3DisplayName

  
  
**Aplica-se a**: Outlook 
  
Especifica o nome de exibição legíveis de usuário para o terceiro endereço de email.

|:-----|:-----|
|Propriedades associadas:  <br/> |dispidEmail3DisplayName  <br/> |
|Propriedade definida:  <br/> |PSETID_Address  <br/> |
|ID de longo (LID):  <br/> |0x000080A0  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE  <br/> |
