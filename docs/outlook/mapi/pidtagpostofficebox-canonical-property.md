# <a name="pidtagpostofficebox-canonical-property"></a>Propriedade canônica PidTagPostOfficeBox

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém o número ou o identificador da caixa postal do destinatário.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_POST_OFFICE_BOX, PR_POST_OFFICE_BOX_A, PR_POST_OFFICE_BOX_W, PR_BUSINESS_ADDRESS_POST_OFFICE_BOX, PR_BUSINESS_ADDRESS_POST_OFFICE_BOX_A, PR_BUSINESS_ADDRESS_POST_OFFICE_BOX_W  <br/> |
|Identificador:  <br/> |0x3A2B  <br/> |
|Tipo de dados:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Área:  <br/> |Usuário de email MAPI  <br/> |
   
## <a name="remarks"></a>Comentários

Essas propriedades fornecem identificação e acessar informações para um destinatário. Eles são definidos por destinatário e suas respectivas organizações.
