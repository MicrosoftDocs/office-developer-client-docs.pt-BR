# <a name="pidlidtasklastupdate-canonical-property"></a>Propriedade canônica PidLidTaskLastUpdate

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Indica o Tempo Universal Coordenado (UTC) e a data da alteração mais recente que foi feita na tarefa e indicada pela propriedade **dispidTaskHistory** ([PidLidTaskHistory](pidlidtasklastupdate-canonical-property.md)).
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidTaskLastUpdate  <br/> |
|Conjunto de propriedades:  <br/> |PSETID_Task  <br/> |
|Long ID (LID):  <br/> |0x00008115  <br/> |
|Tipo de dados:  <br/> |PT_SYSTIME  <br/> |
|Área:  <br/> |Tarefas  <br/> |
