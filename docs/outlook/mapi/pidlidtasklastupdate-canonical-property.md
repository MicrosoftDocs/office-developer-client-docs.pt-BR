# <a name="pidlidtasklastupdate-canonical-property"></a>Propriedade canônico de PidLidTaskLastUpdate

  
  
**Aplica-se a**: Outlook 
  
Indica o tempo Universal Coordenado (UTC) e data da alteração mais recente que tenha sido feita à tarefa e indicada pela propriedade **dispidTaskHistory** ([PidLidTaskHistory](pidlidtasklastupdate-canonical-property.md)).
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidTaskLastUpdate  <br/> |
|Propriedade definida:  <br/> |PSETID_Task  <br/> |
|ID de longo (LID):  <br/> |0x00008115  <br/> |
|Tipo de dados:  <br/> |PT_SYSTIME  <br/> |
|Área:  <br/> |Tarefa  <br/> |
