# <a name="pidtagread-canonical-property"></a>Propriedade canônica PidTagRead

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Indica se uma linha da tabela foi lido.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_READ  <br/> |
|Identificador:  <br/> |0x0E69  <br/> |
|Tipo de dados:  <br/> |PT_BOOLEAN  <br/> |
|Área:  <br/> |MAPI não transmittable  <br/> |
