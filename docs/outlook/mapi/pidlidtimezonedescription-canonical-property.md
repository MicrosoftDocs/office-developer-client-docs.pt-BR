# <a name="pidlidtimezonedescription-canonical-property"></a>Propriedade canônica PidLidTimeZoneDescription

  
  
**Aplica-se a**: Outlook 
  
Especifica uma descrição de cadeia de caracteres do fuso horário.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidTimeZoneDesc  <br/> |
|Propriedade definida:  <br/> |PSETID_Appointment  <br/> |
|ID de longo (LID):  <br/> |0x00008234  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Calendário  <br/> |
   
## <a name="remarks"></a>Comentários

Esta propriedade especifica uma descrição legíveis do fuso horário que é representado pelos dados na propriedade **dispidTimeZoneStruct** ([PidLidTimeZoneStruct](pidlidtimezonestruct-canonical-property.md)).
