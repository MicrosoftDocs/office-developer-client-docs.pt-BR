# <a name="pidlidtimezonedescription-canonical-property"></a>Propriedade canônica PidLidTimeZoneDescription

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Especifica uma descrição de cadeia de caracteres do fuso horário.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidTimeZoneDesc  <br/> |
|Conjunto de propriedades:  <br/> |PSETID_Appointment  <br/> |
|Long ID (LID):  <br/> |0x00008234  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Calendário  <br/> |
   
## <a name="remarks"></a>Comentários

Esta propriedade especifica uma descrição acessível para humanos do fuso horário representado pelos dados na propriedade **dispidTimeZoneStruct** ([PidLidTimeZoneStruct](pidlidtimezonestruct-canonical-property.md)).
