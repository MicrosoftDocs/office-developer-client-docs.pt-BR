# <a name="pidlidprivate-canonical-property"></a>Propriedade canônica PidLidPrivate

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Indica se o usuário final deseja que esse objeto de mensagem seja ocultado de outros usuários que tenham acesso ao objeto de mensagem.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidPrivate  <br/> |
|Conjunto de propriedades:  <br/> |PSETID_Common  <br/> |
|Long ID (LID):  <br/> |0x00008506  <br/> |
|Tipo de dados:  <br/> |PT_BOOLEAN  <br/> |
|Área:  <br/> |Envio de mensagens gerais  <br/> |
