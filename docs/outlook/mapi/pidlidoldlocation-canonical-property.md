# <a name="pidlidoldlocation-canonical-property"></a>Propriedade canônica PidLidOldLocation

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Indica o valor original da **propriedade dispidLocation** ([PidLidLocation](pidlidlocation-canonical-property.md)) antes de uma atualização de reunião.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidOldLocation  <br/> |
|Conjunto de propriedades:  <br/> |PSETID_Meeting  <br/> |
|Long ID (LID):  <br/> |0x00000028  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Reuniões  <br/> |
   
## <a name="remarks"></a>Comentários

Essa propriedade não é necessária.
