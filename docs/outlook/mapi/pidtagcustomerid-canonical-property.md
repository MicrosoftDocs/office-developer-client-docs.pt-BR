# <a name="pidtagcustomerid-canonical-property"></a>Propriedade canônica PidTagCustomerId

  
  
**Aplica-se a**: Outlook 
  
Contém o número de identificação do cliente do contato.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_CUSTOMER_ID, PR_CUSTOMER_ID_A, PR_CUSTOMER_ID_W  <br/> |
|Identificador:  <br/> |0x3A4A  <br/> |
|Tipo de dados:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Área:  <br/> |Contato  <br/> |
   
## <a name="remarks"></a>Comentários

O conteúdo dessas propriedades é definido pelo usuário ou a organização do usuário.
