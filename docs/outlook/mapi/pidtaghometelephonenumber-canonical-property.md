# <a name="pidtaghometelephonenumber-canonical-property"></a>Propriedade canônica PidTagHomeTelephoneNumber

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém o número de telefone principal da casa do destinatário. 
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_HOME_TELEPHONE_NUMBER, PR_HOME_TELEPHONE_NUMBER_A, PR_HOME_TELEPHONE_NUMBER_W  <br/> |
|Identificador:  <br/> |0x3A09  <br/> |
|Tipo de dados:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Área:  <br/> |Usuário de email MAPI  <br/> |
   
## <a name="remarks"></a>Comentários

Essas propriedades fornecem a identificação e o acesso de um destinatário. Eles são definidos pelo destinatário e sua organização.
