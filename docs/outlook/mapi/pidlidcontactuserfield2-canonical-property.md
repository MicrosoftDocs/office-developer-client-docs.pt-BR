# <a name="pidlidcontactuserfield2-canonical-property"></a>Propriedade canônica PidLidContactUserField2

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Especifica o segundo campo no contato que destina-se ao uso de diversos para o contato.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidContactUserField2  <br/> |
|Propriedade definida:  <br/> |PSETID_Address  <br/> |
|ID de longo (LID):  <br/> |0x00008050  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Contato  <br/> |
   
## <a name="remarks"></a>Comentários

Essa propriedade contém o texto que não está relacionado a qualquer outra propriedade específicos para contatos.
