# <a name="pidtaglocaleid-canonical-property"></a>Propriedade canônica PidTagLocaleId

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Estabelece a localidade do idioma para traduzir mensagens geradas pelo sistema, como notificações de entrega.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_LOCALE_ID  <br/> |
|Identificador:  <br/> |0x66A1  <br/> |
|Tipo de dados:  <br/> |PT_LONG  <br/> |
|Área:  <br/> |Diversos  <br/> |
   
## <a name="remarks"></a>Comentários

Essa propriedade está disponível em um logon de caixa de correio privada.
