**Aplica-se a**: Outlook 2013 | Outlook 2016

|**Priority**|**Valor MAPI**|**Valor TNEF**|
|:-----|:-----|:-----|
|normal  <br/> |-1  <br/> |3  <br/> |
|alta  <br/> |0  <br/> |2  <br/> |
|baixa  <br/> |1  <br/> |1  <br/> |
