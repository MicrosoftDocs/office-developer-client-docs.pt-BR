# <a name="pidtagpstpath-canonical-property"></a>Propriedade canônica PidTagPstPath

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém o local do objeto no repositório de pasta pessoal para criar ou configurar.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_PST_PATH, PR_PST_PATH_A, PR_PST_PATH_W  <br/> |
|Identificador:  <br/> |0x6700  <br/> |
|Tipo de dados:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Área:  <br/> |Tabela de armazenamento pessoal (. pst) interna  <br/> |
