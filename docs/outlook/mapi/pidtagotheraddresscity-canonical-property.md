# <a name="pidtagotheraddresscity-canonical-property"></a>Propriedade canônica PidTagOtherAddressCity

  
  
**Aplica-se a**: Outlook 
  
Contém o nome do email outra localidade do usuário, como a cidade ou cidade.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_OTHER_ADDRESS_CITY, PR_OTHER_ADDRESS_CITY_A, PR_OTHER_ADDRESS_CITY_W  <br/> |
|Identificador:  <br/> |0x3A5F  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Área:  <br/> |Endereço  <br/> |
   
## <a name="remarks"></a>Comentários

Essas propriedades são definidas pelo usuário ou a organização do usuário.
