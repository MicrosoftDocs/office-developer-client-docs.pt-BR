# <a name="pidtaghobbies-canonical-property"></a>Propriedade canônica PidTagHobbies

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém os nomes dos Hobbies do usuário de mensagens.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_HOBBIES, PR_HOBBIES_A, PR_HOBBIES_W  <br/> |
|Identificador:  <br/> |0x3A43  <br/> |
|Tipo de dados:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Área:  <br/> |Usuário de email MAPI  <br/> |
   
## <a name="remarks"></a>Comentários

Essas propriedades fornecem informações de identificação e acesso sobre um usuário de mensagens e o conteúdo é definido pela organização do usuário.
