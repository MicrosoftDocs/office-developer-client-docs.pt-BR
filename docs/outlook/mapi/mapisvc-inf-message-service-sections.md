# <a name="mapisvcinf-message-service-sections"></a>Seções do serviço de mensagens MapiSvc.inf

  
  
**Aplica-se a**: Outlook 
  
Mapisvc inclui uma seção de serviço de mensagem para cada uma das entradas listadas na seção **[Services]** . Existem dois tipos de entradas nestas seções: um para a definição de determinadas propriedades e o outro para listar os nomes das seções que estão relacionados ao serviço de mensagem que está sendo configurado.
