# <a name="pidtagspamtrustedsenders-canonical-property"></a>Propriedade canônica PidTagSpamTrustedSenders

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém uma lista delimitada por ponto-e-vírgula de endereços de email e domínios compostos por remetentes confiáveis.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_SPAM_TRUSTED_SENDERS_W  <br/> |
|Long ID (LID):  <br/> |0x0418  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Spam  <br/> |
