# <a name="pidtaginternetlines-canonical-property"></a>Propriedade canônica PidTagInternetLines

  
  
**Aplica-se a**: Outlook

|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_INTERNET_LINES  <br/> |
|Identificador:  <br/> |0x1034  <br/> |
|Tipo de dados:  <br/> |PT_LONG  <br/> |
|Área:  <br/> |MIME  <br/> |
