# <a name="pidtagsupplementaryinfo-canonical-property"></a>Propriedade canônica PidTagSupplementaryInfo

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém informações adicionais para uso em um relatório.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_SUPPLEMENTARY_INFO, PR_SUPPLEMENTARY_INFO_A, PR_SUPPLEMENTARY_INFO_W  <br/> |
|Identificador:  <br/> |0x0C1B  <br/> |
|Tipo de dados:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Área:  <br/> |Destinatário MAPI  <br/> |
   
## <a name="remarks"></a>Comentários

Essas propriedades contêm informações geradas pelo agente de transferência de mensagens ou pelo provedor de transporte relacionado ao relatório. Normalmente é usado para texto de relatório de entrega ou não entrega originado com o sistema de mensagens subjacente.
