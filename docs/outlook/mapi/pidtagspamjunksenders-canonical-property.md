# <a name="pidtagspamjunksenders-canonical-property"></a>Propriedade canônica PidTagSpamJunkSenders

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém uma lista delimitada por ponto-e-vírgula de endereços de email e domínios que compõem os envios bloqueados.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_SPAM_JUNK_SENDERS_W  <br/> |
|Long ID (LID):  <br/> |0x041A  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Spam  <br/> |
