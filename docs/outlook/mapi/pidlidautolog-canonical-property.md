# <a name="pidlidautolog-canonical-property"></a>Propriedade canônica PidLidAutoLog

  
  
**Aplica-se a**: Outlook 
  
Especifica se é necessário criar um diário para cada ação associada a esse contato.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidAutoLog  <br/> |
|Propriedade definida:  <br/> |PSETID_Address  <br/> |
|ID de longo (LID):  <br/> |0x00008025  <br/> |
|Tipo de dados:  <br/> |PT_BOOLEAN  <br/> |
|Área:  <br/> |Contato  <br/> |
