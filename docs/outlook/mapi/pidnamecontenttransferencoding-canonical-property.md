# <a name="pidnamecontenttransferencoding-canonical-property"></a>Propriedade canônica PidNameContentTransferEncoding

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém um valor de campo de título [RFC3282] Content-Transfer-Encoding.
  
|||
|:-----|:-----|
|Nomes amigáveis:  <br/> |Nenhum  <br/> |
|Conjunto de propriedades:  <br/> |PS_INTERNET_HEADERS  <br/> |
|Nome da propriedade:  <br/> |Content-Transfer-Encoding  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Email  <br/> |
