# <a name="pidtagcontactaddressbookdisplaynames-canonical-property"></a>Propriedade canônica PidTagContactAddressBookDisplayNames

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém uma lista de nomes de exibição para usuários de mensagens no livro de endereços de contato.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_CONTAB_DISPLAY_NAMES, PR_CONTAB_DISPLAY_NAMES_A, PR_CONTAB_DISPLAY_NAMES_W  <br/> |
|Identificador:  <br/> |0x6624  <br/> |
|Tipo de dados:  <br/> |PT_MV_UNICODE, PT_MV_STRING8  <br/> |
|Área:  <br/> |Contact address book  <br/> |
   
## <a name="remarks"></a>Comentários

Essas propriedades são definidas pelo usuário e pela organização do usuário.
