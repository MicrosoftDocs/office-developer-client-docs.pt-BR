# <a name="pidtagreplytime-canonical-property"></a>Propriedade canônica PidTagReplyTime

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém o prazo final em uma mensagem. 
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_REPLY_TIME  <br/> |
|Identificador:  <br/> |0x0030  <br/> |
|Tipo de dados:  <br/> |PT_SYSTIME  <br/> |
|Área:  <br/> |Envelope MAPI  <br/> |
