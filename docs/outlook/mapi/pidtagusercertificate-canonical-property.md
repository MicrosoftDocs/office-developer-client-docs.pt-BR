# <a name="pidtagusercertificate-canonical-property"></a>Propriedade canônica PidTagUserCertificate

  
  
**Aplica-se a**: Outlook 
  
Contém um certificado de autenticação de ASN. 1 para um usuário de mensagens. 
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_USER_CERTIFICATE  <br/> |
|Identificador:  <br/> |0x3A22  <br/> |
|Tipo de dados:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Usuário de email MAPI  <br/> |
   
## <a name="remarks"></a>Comentários

Um certificado de autenticação é semelhante a uma assinatura digital. Várias propriedades MAPI fornecem certificados ASN. 1.
