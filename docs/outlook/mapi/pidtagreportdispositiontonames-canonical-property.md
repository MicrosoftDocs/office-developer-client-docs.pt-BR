# <a name="pidtagreportdispositiontonames-canonical-property"></a>Propriedade canônica PidTagReportDispositionToNames

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Indica os nomes dos destinatários do relatório para mensagens que solicitam recibos.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_REPORT_DISPOSITION_TO_NAMES, PR_REPORT_DISPOSITION_TO_NAMES_A, PR_REPORT_DISPOSITION_TO_NAMES_W  <br/> |
|Identificador:  <br/> |0x0083  <br/> |
|Tipo de dados:  <br/> |PT_MV_STRING8, PT_UNICODE  <br/> |
|Área:  <br/> |Envelope MAPI  <br/> |
