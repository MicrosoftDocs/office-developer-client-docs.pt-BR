# <a name="pidtagphysicaldeliveryreportrequest-canonical-property"></a>Propriedade canônica PidTagPhysicalDeliveryReportRequest

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém o modo de um relatório a ser entregue a um destinatário de mensagem específico após a conclusão da entrega ou entrega de mensagens físicas pelo sistema de tratamento de mensagens.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_PHYSICAL_DELIVERY_REPORT_REQUEST  <br/> |
|Identificador:  <br/> |0x0C0C  <br/> |
|Tipo de dados:  <br/> |PT_LONG  <br/> |
|Área:  <br/> |Destinatário MAPI  <br/> |
