# <a name="pidtagsmtpaddress-canonical-property"></a>Propriedade canônica PidTagSmtpAddress

  
  
**Aplica-se a**: Outlook 
  
Contém o endereço de SMTP para o objeto de catálogo de endereços.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_SMTP_ADDRESS, PR_SMTP_ADDRESS_A, PR_SMTP_ADDRESS_W  <br/> |
|Identificador:  <br/> |0x39FE  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Área:  <br/> |Endereço  <br/> |
