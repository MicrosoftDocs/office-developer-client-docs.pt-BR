# <a name="pidtagorganizationalidnumber-canonical-property"></a>Propriedade canônica PidTagOrganizationalIdNumber

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém um número de identificação organizacional do contato, como um número de identificação de funcionário.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_ORGANIZATIONAL_ID_NUMBER, PR_ORGANIZATIONAL_ID_NUMBER_A, PR_ORGANIZATIONAL_ID_NUMBER_W  <br/> |
|Identificador:  <br/> |0x3A10  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Área:  <br/> |Endereço  <br/> |
   
## <a name="remarks"></a>Comentários

Estas são propriedades opcionais. Seu uso é determinado pelo usuário ou organização mensagens.
