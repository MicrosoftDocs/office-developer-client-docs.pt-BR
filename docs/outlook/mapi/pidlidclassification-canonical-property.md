# <a name="pidlidclassification-canonical-property"></a>Propriedade canônica PidLidClassification

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Indica o nome da classificação de mensagem.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidClassification  <br/> |
|Propriedade definida:  <br/> |PSETID_Common  <br/> |
|ID de longo (LID):  <br/> |0x000085B6  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Soluções gerais de mensagens  <br/> |
