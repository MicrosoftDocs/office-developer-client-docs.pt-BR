# <a name="attreciptable"></a>attRecipTable

**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Quando uma tabela de destinatários está sendo codificada, cada destinatário é codificado como uma linha de propriedades MAPI. O formato é o seguinte: 
  
_Row_Seq:_
  
>  _linha-contagem_ _Property_Sequence,_... 
    
O formato de _Property_Seq_ é o mesmo que no atributo [attMAPIProps](attmapiprops.md) .
