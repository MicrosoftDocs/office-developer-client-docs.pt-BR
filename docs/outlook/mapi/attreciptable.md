# <a name="attreciptable"></a>attRecipTable

**Aplica-se a**: Outlook 
  
Quando uma tabela de destinatário está sendo codificada, cada destinatário é codificado como uma linha de propriedades MAPI. O formato é da seguinte maneira: 
  
_Row_Seq:_
  
>  _contagem de linhas_ _Property_Sequence,_... 
    
O formato do _Property_Seq_ é o mesmo que o atributo [attMAPIProps](attmapiprops.md) .
