# <a name="pidlidlogflags-canonical-property"></a>Propriedade canônica PidLidLogFlags

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém metadados sobre o diário.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidLogFlags  <br/> |
|Conjunto de propriedades:  <br/> |PSETID_Log  <br/> |
|Long ID (LID):  <br/> |0x0000870C  <br/> |
|Tipo de dados:  <br/> |PT_LONG  <br/> |
|Área:  <br/> |Diário  <br/> |
   
## <a name="remarks"></a>Comentários

O campo de bits que contém metadados sobre o diário deve ser zero ou "0x40000000".
