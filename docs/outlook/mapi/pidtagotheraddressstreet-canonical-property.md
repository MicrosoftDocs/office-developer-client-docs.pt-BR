# <a name="pidtagotheraddressstreet-canonical-property"></a>Propriedade canônica PidTagOtherAddressStreet

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém o outro endereço do usuário de email.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_OTHER_ADDRESS_STREET, PR_OTHER_ADDRESS_STREET_A, PR_OTHER_ADDRESS_STREET_W  <br/> |
|Identificador:  <br/> |0x3A63  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Área:  <br/> |Endereço  <br/> |
   
## <a name="remarks"></a>Comentários

Essas propriedades são definidas pelo usuário ou pela organização do usuário.
