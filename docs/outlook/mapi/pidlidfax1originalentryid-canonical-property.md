# <a name="pidlidfax1originalentryid-canonical-property"></a>Propriedade canônica PidLidFax1OriginalEntryId

  
  
**Aplica-se a**: Outlook 
  
Especifica a EntryID original do endereço de fax comercial do contato.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidFax1OriginalEntryID  <br/> |
|Propriedade definida:  <br/> |PSETID_Address  <br/> |
|ID de longo (LID):  <br/> |0x000080B5  <br/> |
|Tipo de dados:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Contato  <br/> |
   
## <a name="remarks"></a>Comentários

Essa propriedade, se houver, deve especificar a EntryId único que corresponde a esse endereço de fax.
