# <a name="pidlidfax1originalentryid-canonical-property"></a>Propriedade canônica PidLidFax1OriginalEntryId

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Especifica a EntryID original do endereço de fax comercial do contato.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidFax1OriginalEntryID  <br/> |
|Conjunto de propriedades:  <br/> |PSETID_Address  <br/> |
|Long ID (LID):  <br/> |0x000080B5  <br/> |
|Tipo de dados:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Contato  <br/> |
   
## <a name="remarks"></a>Comentários

Esta propriedade, se presente, deve especificar a EntryID única que corresponde a esse endereço de fax.
