# <a name="pidlidrecalltime-canonical-property"></a>Propriedade canônica PidLidRecallTime

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Indica a hora em que uma rechamada de mensagem foi iniciada.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidRecallTime  <br/> |
|Conjunto de propriedades:  <br/> |PSETID_Common  <br/> |
|Long ID (LID):  <br/> |0x00008549  <br/> |
|Tipo de dados:  <br/> |PT_SYSTIME  <br/> |
|Área:  <br/> |Envio de mensagens gerais  <br/> |
