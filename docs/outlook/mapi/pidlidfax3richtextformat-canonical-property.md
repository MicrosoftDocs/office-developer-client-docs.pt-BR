# <a name="pidlidfax3richtextformat-canonical-property"></a>Propriedade canônica PidLidFax3RichTextFormat

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Especifica o que outro endereço de fax do formato rich text do contato.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidFax3RTF  <br/> |
|Propriedade definida:  <br/> |PSETID_Address  <br/> |
|ID de longo (LID):  <br/> |0x000080D6  <br/> |
|Tipo de dados:  <br/> |PT_BOOLEAN  <br/> |
|Área:  <br/> |Contato  <br/> |
