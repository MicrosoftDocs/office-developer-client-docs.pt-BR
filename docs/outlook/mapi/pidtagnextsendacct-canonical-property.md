# <a name="pidtagnextsendacct-canonical-property"></a>Propriedade canônica PidTagNextSendAcct

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Especifica o servidor que o cliente está tentando usar para enviar emails.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_NEXT_SEND_ACCT  <br/> |
|Identificador:  <br/> |0x0E29  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Aplicativo do Outlook  <br/> |
   
## <a name="remarks"></a>Comentários

O formato dessa propriedade é dependente de implementação. Essa propriedade pode ser usada pelo cliente para determinar em qual servidor direcionar o email, mas é opcional e o valor não tem significado para o servidor.
