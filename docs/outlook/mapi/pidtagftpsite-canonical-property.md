# <a name="pidtagftpsite-canonical-property"></a>Propriedade canônica PidTagFtpSite

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém a URL do protocolo FTP do contato. FTP é um protocolo usado para transferir dados, conforme especificado em [RFC959].
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_FTP_SITE, PR_FTP_SITE_A, PR_FTP_SITE_W  <br/> |
|Identificador:  <br/> |0x3A4C  <br/> |
|Tipo de dados:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Área:  <br/> |Usuário de email MAPI  <br/> |
