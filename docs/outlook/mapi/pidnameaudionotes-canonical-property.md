# <a name="pidnameaudionotes-canonical-property"></a>Propriedade canônica PidNameAudioNotes

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Especifica as observações textuais que estejam anexadas a uma mensagem de voz.
  
|||
|:-----|:-----|
|Nomes amigáveis:  <br/> |UMAudioNotes  <br/> |
|Propriedade definida:  <br/> |PSETID_UnifiedMessaging  <br/> |
|Nome da propriedade:  <br/> |UMAudioNotes  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Unificação de mensagens  <br/> |
   
## <a name="remarks"></a>Comentários

Para habilitar um usuário final, leia e editar notas de áudio diretamente em uma mensagem de voz, um cliente fornece uma caixa de edição onde o usuário pode digitar um conjunto de notas que são adicionados a essa propriedade do objeto de mensagem de voz.
