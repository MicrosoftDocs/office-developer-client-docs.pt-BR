# <a name="pidnameaudionotes-canonical-property"></a>Propriedade canônica PidNameAudioNotes

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Especifica as anotações textuais anexadas a uma mensagem de voz.
  
|||
|:-----|:-----|
|Nomes amigáveis:  <br/> |UMAudioNotes  <br/> |
|Conjunto de propriedades:  <br/> |PSETID_UnifiedMessaging  <br/> |
|Nome da propriedade:  <br/> |UMAudioNotes  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |UM (Unificação de mensagens)  <br/> |
   
## <a name="remarks"></a>Comentários

Para permitir que um usuário final leia e edite anotações de áudio diretamente em uma mensagem de voz, um cliente fornece uma caixa de edição onde o usuário pode digitar um conjunto de anotações que são adicionadas a essa propriedade do objeto de mensagem de voz.
