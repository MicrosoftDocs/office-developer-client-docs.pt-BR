# <a name="pidtagchildrensnames-canonical-property"></a>Propriedade canônica PidTagChildrensNames

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém uma lista dos nomes dos filhos
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_CHILDRENS_NAMES, PR_CHILDRENS_NAMES_A, PR_CHILDRENS_NAMES_W  <br/> |
|Identificador:  <br/> |0x3A58  <br/> |
|Tipo de dados:  <br/> |PT_MV_UNICODE, PT_MV_STRING8  <br/> |
|Área:  <br/> |Contato  <br/> |
   
## <a name="remarks"></a>Comentários

Essas propriedades são definidas pelo usuário ou a organização do usuário.
