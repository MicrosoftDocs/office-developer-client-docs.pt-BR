# <a name="pidtagabdefaultpab-canonical-property"></a>Propriedade canônica PidTagAbDefaultPab

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém o identificador de entrada do contêiner de catálogo de endereços a ser usado como o catálogo de endereços pessoal (PAB). 
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_AB_DEFAULT_PAB  <br/> |
|Identificador:  <br/> |0x3D07  <br/> |
|Tipo de dados:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Catálogo de endereços  <br/> |
   
## <a name="remarks"></a>Comentários

Não use essa propriedade. É reservado para uso por MAPI.
