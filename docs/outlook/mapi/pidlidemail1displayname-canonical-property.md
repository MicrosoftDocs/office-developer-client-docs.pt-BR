# <a name="pidlidemail1displayname-canonical-property"></a>Propriedade canônica PidLidEmail1DisplayName

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Especifica o nome de exibição acessível pelo usuário para o primeiro endereço de email.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidEmail1DisplayName  <br/> |
|Conjunto de propriedades:  <br/> |PSETID_Address  <br/> |
|Long ID (LID):  <br/> |0x00008080  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Contato  <br/> |
