# <a name="pidlidemail1displayname-canonical-property"></a>Propriedade canônica PidLidEmail1DisplayName

  
  
**Aplica-se a**: Outlook 
  
Especifica o nome de exibição legíveis de usuário para o primeiro endereço de email.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidEmail1DisplayName  <br/> |
|Propriedade definida:  <br/> |PSETID_Address  <br/> |
|ID de longo (LID):  <br/> |0x00008080  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Contato  <br/> |
