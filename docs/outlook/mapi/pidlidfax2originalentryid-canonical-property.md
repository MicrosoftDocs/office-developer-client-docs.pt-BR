# <a name="pidlidfax2originalentryid-canonical-property"></a>Propriedade canônica PidLidFax2OriginalEntryId

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Especifica a EntryID original do endereço de fax residencial do contato.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidFax2OriginalEntryID  <br/> |
|Conjunto de propriedades:  <br/> |PSETID_Address  <br/> |
|Long ID (LID):  <br/> |0x000080C5  <br/> |
|Tipo de dados:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Contato  <br/> |
   
## <a name="remarks"></a>Comentários

Essa propriedade, se presente, deve especificar a EntryId única que corresponde a esse endereço de fax.
