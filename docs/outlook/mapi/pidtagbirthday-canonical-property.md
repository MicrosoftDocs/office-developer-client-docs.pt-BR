# <a name="pidtagbirthday-canonical-property"></a>Propriedade canônica PidTagBirthday

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém o aniversário do contato.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_BIRTHDAY  <br/> |
|Identificador:  <br/> |0x3A42  <br/> |
|Tipo de dados:  <br/> |PT_SYSTIME  <br/> |
|Área:  <br/> |Contato  <br/> |
   
## <a name="remarks"></a>Comentários

Essa propriedade é uma das propriedades que fornece informações de identificação e acesso sobre um contato. Essas propriedades são definidas pelo usuário e pela organização do usuário.
