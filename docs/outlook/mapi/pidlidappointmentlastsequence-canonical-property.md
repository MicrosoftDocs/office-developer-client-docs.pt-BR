# <a name="pidlidappointmentlastsequence-canonical-property"></a>Propriedade canônico de PidLidAppointmentLastSequence

  
  
**Aplica-se a**: Outlook 
  
Indica o último número de sequência que foi enviado para qualquer participante do organizador.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidApptLastSequence  <br/> |
|Propriedade definida:  <br/> |PSETID_Appointment  <br/> |
|ID de longo (LID):  <br/> |0x00008203  <br/> |
|Tipo de dados:  <br/> |PT_LONG  <br/> |
|Área:  <br/> |Reuniões  <br/> |
