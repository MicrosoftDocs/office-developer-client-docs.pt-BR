# <a name="pidlidcompanies-canonical-property"></a>Propriedade canônica PidLidCompanies

  
  
**Aplica-se a**: Outlook 
  
Contém os nomes das empresas que estão associados um item.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidCompanies  <br/> |
|Propriedade definida:  <br/> |PSETID_Common  <br/> |
|ID de longo (LID):  <br/> |0x00008539  <br/> |
|Tipo de dados:  <br/> |PT_MV_UNICODE  <br/> |
|Área:  <br/> |Soluções gerais de mensagens  <br/> |
