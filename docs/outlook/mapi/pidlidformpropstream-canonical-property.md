# <a name="pidlidformpropstream-canonical-property"></a>Propriedade canônica PidLidFormPropStream

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Representa os dados nas propriedades personalizadas que são salvas com a mensagem.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidFormPropStream  <br/> |
|Propriedade definida:  <br/> |PSETID_Common  <br/> |
|ID de longo (LID):  <br/> |0x0000851B  <br/> |
|Tipo de dados:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Configuração de tempo de execução  <br/> |
