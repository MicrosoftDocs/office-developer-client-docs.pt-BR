# <a name="pidtagcartelephonenumber-canonical-property"></a>Propriedade canônica PidTagCarTelephoneNumber

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém o número de telefone do carro do destinatário. 
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_CAR_TELEPHONE_NUMBER, PR_CAR_TELEPHONE_NUMBER_A, PR_CAR_TELEPHONE_NUMBER_W  <br/> |
|Identificador:  <br/> |0x3A1E  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Área:  <br/> |Contato  <br/> |
   
## <a name="remarks"></a>Comentários

Essas propriedades fornecem identificação e acessar informações sobre um destinatário. Eles são definidos por destinatário e a organização do destinatário.
