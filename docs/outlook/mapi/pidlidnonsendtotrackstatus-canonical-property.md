# <a name="pidlidnonsendtotrackstatus-canonical-property"></a>Propriedade canônica PidLidNonSendToTrackStatus

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém o valor de cada participante listado na **propriedade dispidNonSendableTo** ([PidLidNonSendableTo](pidlidnonsendableto-canonical-property.md)).
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidNonSendToTrackStatus  <br/> |
|Conjunto de propriedades:  <br/> |PSETID_Common  <br/> |
|Long ID (LID):  <br/> |0x00008543  <br/> |
|Tipo de dados:  <br/> |PT_MV_LONG  <br/> |
|Área:  <br/> |Envio de mensagens gerais  <br/> |
   
## <a name="remarks"></a>Comentários

Essa propriedade é necessária somente quando a **propriedade dispidNonSendableTo** está definida. O número de valores nesta propriedade deve ser igual ao número de valores em **dispidNonSendableTo**. Cada PT_LONG valor nessa propriedade corresponde ao participante na **propriedade dispidNonSendableTo** no mesmo índice.
