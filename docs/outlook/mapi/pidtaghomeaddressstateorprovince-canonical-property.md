# <a name="pidtaghomeaddressstateorprovince-canonical-property"></a>Propriedade canônica PidTagHomeAddressStateOrProvince

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém a parte do estado ou província do endereço de um usuário.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_HOME_ADDRESS_STATE_OR_PROVINCE, PR_HOME_ADDRESS_STATE_OR_PROVINCE_A, PR_HOME_ADDRESS_STATE_OR_PROVINCE_W  <br/> |
|Identificador:  <br/> |0x3A5C  <br/> |
|Tipo de dados:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Área:  <br/> |Usuário de email MAPI  <br/> |
   
## <a name="remarks"></a>Comentários

Essas propriedades são definidas pelo usuário ou pela organização do usuário.
