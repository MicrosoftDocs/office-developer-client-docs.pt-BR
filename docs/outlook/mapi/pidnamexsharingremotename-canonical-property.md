# <a name="pidnamexsharingremotename-canonical-property"></a>Propriedade canônica PidNameXSharingRemoteName

  
  
**Aplica-se a**: Outlook 
  
Representa o valor da propriedade **dispidSharingRemoteName** ([PidLidSharingRemoteName](pidlidsharingremotename-canonical-property.md)).
  
|||
|:-----|:-----|
|Nomes amigáveis:  <br/> |Nenhum  <br/> |
|Propriedade definida:  <br/> |PS_INTERNET_HEADERS  <br/> |
|Nome da propriedade:  <br/> |X-compartilhamento-Remote-Name  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Sharing  <br/> |
