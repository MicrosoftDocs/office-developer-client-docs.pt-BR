# <a name="pidtagpstpasswordsznew-canonical-property"></a>Propriedade canônica PidTagPstPasswordSzNew

  
  
**Aplica-se a**: Outlook 
  
Contém a nova senha a ser usada para acesso futuro para o repositório de pasta pessoal.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_PST_PW_SZ_NEW, PR_PST_PW_SZ_NEW_A, PR_PST_PW_SZ_NEW_W  <br/> |
|Identificador:  <br/> |0x6704  <br/> |
|Tipo de dados:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Área:  <br/> |Tabela de armazenamento pessoal (. pst) interna  <br/> |
