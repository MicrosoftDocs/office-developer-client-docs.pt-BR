# <a name="pidlidsharingproviderguid-canonical-property"></a>Propriedade canônica PidLidSharingProviderGuid

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Especifica o identificador global exclusivo (GUID) do provedor de compartilhamento. Esta é uma propriedade de uma mensagem de compartilhamento.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidSharingProviderGuid  <br/> |
|Conjunto de propriedades:  <br/> |PSETID_Sharing  <br/> |
|Long ID (LID):  <br/> |0x00008A01  <br/> |
|Tipo de dados:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Compartilhamento  <br/> |
   
## <a name="remarks"></a>Comentários

O valor dessa propriedade deve ser definido como "%xAE.F0.06.00.00.00.00.00.00.C0.00.00.00.00.00.00.46".
