# <a name="pidtagcontactversion-canonical-property"></a>Propriedade canônica PidTagContactVersion

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém a versão das informações de contato de um destinatário.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_CONTACT_VERSION  <br/> |
|Identificador:  <br/> |0x3A52  <br/> |
|Tipo de dados:  <br/> |PT_CLSID  <br/> |
|Área:  <br/> |Contato  <br/> |
   
## <a name="remarks"></a>Comentários

Essa propriedade é uma das propriedades que fornece informações de identificação e acesso sobre um destinatário. Essas propriedades são definidas pelo destinatário e pela organização do destinatário.
