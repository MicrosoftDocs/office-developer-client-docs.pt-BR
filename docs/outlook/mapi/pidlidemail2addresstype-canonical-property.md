# <a name="pidlidemail2addresstype-canonical-property"></a>Propriedade canônica PidLidEmail2AddressType

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Especifica o tipo de endereço do segundo endereço de email.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidEmail2AddrType  <br/> |
|Conjunto de propriedades:  <br/> |PSETID_Address  <br/> |
|Long ID (LID):  <br/> |0x00008092  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Contato  <br/> |
   
## <a name="remarks"></a>Comentários

Se presente, esse valor de propriedade deve ser um tipo de endereço válido.
