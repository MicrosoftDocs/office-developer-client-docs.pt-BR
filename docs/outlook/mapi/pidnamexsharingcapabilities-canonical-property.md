# <a name="pidnamexsharingcapabilities-canonical-property"></a>Propriedade canônica PidNameXSharingCapabilities

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém um valor que indica os recursos de compartilhamento do item.
  
|||
|:-----|:-----|
|Nomes amigáveis:  <br/> |Nenhum  <br/> |
|Conjunto de propriedades:  <br/> |PS_INTERNET_HEADERS  <br/> |
|Nome da propriedade:  <br/> |Recursos de compartilhamento X  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Compartilhamento  <br/> |
   
## <a name="remarks"></a>Comentários

Essa propriedade contém um valor que indica os recursos de compartilhamento do objeto de mensagem.
