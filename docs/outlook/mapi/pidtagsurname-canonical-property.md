# <a name="pidtagsurname-canonical-property"></a>Propriedade canônica PidTagSurname

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém a última ou o sobrenome do destinatário.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_SURNAME, PR_SURNAME_A, PR_SURNAME_W  <br/> |
|Identificador:  <br/> |0x3A11  <br/> |
|Tipo de dados:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Área:  <br/> |Usuário de email MAPI  <br/> |
   
## <a name="remarks"></a>Comentários

Essas propriedades fornecem identificação e acessar informações para um destinatário. Eles são definidos por destinatário e a organização do destinatário.
