# <a name="pidlidappointmentproposalnumber-canonical-property"></a>Propriedade canônica PidLidAppointmentProposalNumber

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Especifica o número de participantes que enviaram propostas de contadores que não foram aceitas ou rejeitadas pelo organizador.

|:-----|:-----|
|Propriedades associadas:  <br/> |dispidApptProposalNum  <br/> |
|Conjunto de propriedades:  <br/> |PSETID_Appointment  <br/> |
|Long ID (LID):  <br/> |0x00008259  <br/> |
|Tipo de dados:  <br/> |PT_LONG  <br/> |
|Área:  <br/> |Meetings  <br/> |
