# <a name="pidtaginternetarticlenumber-canonical-property"></a>Propriedade canônica PidTagInternetArticleNumber

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Um número associado a um item em um armazenamento de mensagens.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_INTERNET_ARTICLE_NUMBER  <br/> |
|Identificador:  <br/> |0x0E23  <br/> |
|Tipo de dados:  <br/> |PT_LONG  <br/> |
|Área:  <br/> |MIME  <br/> |
   
## <a name="remarks"></a>Comentários

Em repositórios de mensagem IMAPI, essa propriedade corresponde ao UID IMAPI.
