# <a name="olflagicon"></a>OlFlagIcon

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Especifica se um item está sinalizado e, se for, a cor associada ao item sinalizado.
  
## <a name="quick-info"></a>Informações rápidas

```cpp
enum OlFlagIcon { 
    olNoFlagIcon=0, 
    olPurpleFlagIcon=1, 
    olOrangeFlagIcon=2, 
    olGreenFlagIcon=3, 
    olYellowFlagIcon=4, 
    olBlueFlagIcon=5, 
    olRedFlagIcon=6, 
}; 

```

## <a name="members"></a>Membros

 _olNoFlagIcon_
  
> Nenhum sinalizador foi associado ao item.
    
 _olPurpleFlagIcon_
  
> Roxo foi associado ao item sinalizado.
    
 _olOrangeFlagIcon_
  
> O laranja foi associado ao item sinalizado.
    
 _olGreenFlagIcon_
  
> Green foi associado ao item sinalizado.
    
 _olYellowFlagIcon_
  
> Amarelo foi associado ao item sinalizado.
    
 _olBlueFlagIcon_
  
> Blue foi associado ao item sinalizado.
    
 _olRedFlagIcon_
  
> O vermelho foi associado ao item sinalizado.
