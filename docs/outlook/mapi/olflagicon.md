# <a name="olflagicon"></a>OlFlagIcon

  
  
**Aplica-se a**: Outlook 
  
Especifica se um item foi sinalizado e, se estiver, a cor associada ao item sinalizado.
  
## <a name="quick-info"></a>Informações rápidas

```cpp
enum OlFlagIcon { 
    olNoFlagIcon=0, 
    olPurpleFlagIcon=1, 
    olOrangeFlagIcon=2, 
    olGreenFlagIcon=3, 
    olYellowFlagIcon=4, 
    olBlueFlagIcon=5, 
    olRedFlagIcon=6, 
}; 

```

## <a name="members"></a>Members

 _olNoFlagIcon_
  
> Sem sinalizador foi associado ao item.
    
 _olPurpleFlagIcon_
  
> Roxo foi associado ao item sinalizado.
    
 _olOrangeFlagIcon_
  
> Laranja foi associada ao item sinalizado.
    
 _olGreenFlagIcon_
  
> Verde foi associado ao item sinalizado.
    
 _olYellowFlagIcon_
  
> Amarelo foi associado ao item sinalizado.
    
 _olBlueFlagIcon_
  
> Azul foi associado ao item sinalizado.
    
 _olRedFlagIcon_
  
> Vermelho foi associado ao item sinalizado.
