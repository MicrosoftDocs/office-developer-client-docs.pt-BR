# <a name="pidtagscheduleinfodontmaildelegates-canonical-property"></a>Propriedade canônica PidTagScheduleInfoDontMailDelegates

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Conterá TRUE se o representante não deseja receber atualizações.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_SCHDINFO_DONT_MAIL_DELEGATES  <br/> |
|Identificador:  <br/> |0x6843  <br/> |
|Tipo de dados:  <br/> |PT_BOOLEAN  <br/> |
|Área:  <br/> |Informações de disponibilidade  <br/> |
   
## <a name="remarks"></a>Comentários

Essa propriedade deverá ser definida no objeto de informações do representante.
