# <a name="pidtagotheraddresspostofficebox-canonical-property"></a>Propriedade canônica PidTagOtherAddressPostOfficeBox

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém a caixa postal do outro endereço de um contato.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_OTHER_ADDRESS_POST_OFFICE_BOX, PR_OTHER_ADDRESS_POST_OFFICE_BOX_A, PR_OTHER_ADDRESS_POST_OFFICE_BOX_W  <br/> |
|Identificador:  <br/> |0x3A64  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Área:  <br/> |Endereço  <br/> |
   
## <a name="remarks"></a>Comentários

Essas propriedades são definidas pelo usuário ou pela organização do usuário.
