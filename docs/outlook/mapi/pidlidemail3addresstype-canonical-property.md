# <a name="pidlidemail3addresstype-canonical-property"></a>Propriedade canônica PidLidEmail3AddressType

  
  
**Aplica-se a**: Outlook 
  
Especifica o tipo de endereço do terceiro endereço de email.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidEmail3AddrType  <br/> |
|Propriedade definida:  <br/> |PSETID_Address  <br/> |
|ID de longo (LID):  <br/> |0x000080A2  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Contato  <br/> |
   
## <a name="remarks"></a>Comentários

Se presente, o valor dessa propriedade deve ser um tipo de endereço válido.
