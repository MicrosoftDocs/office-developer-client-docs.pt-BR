# <a name="pidtagregisteredmailtype-canonical-property"></a>Propriedade canônica PidTagRegisteredMailType

  
  
**Aplica-se a**: Outlook 
  
Essa propriedade contém o tipo de registro usado para execução física de uma mensagem.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_REGISTERED_MAIL_TYPE  <br/> |
|Identificador:  <br/> |0x0C16  <br/> |
|Tipo de dados:  <br/> |PT_LONG  <br/> |
|Área:  <br/> |Destinatário MAPI  <br/> |
