# <a name="pidtagpagertelephonenumber-canonical-property"></a>Propriedade canônica PidTagPagerTelephoneNumber

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém o número do pager do destinatário.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_PAGER_TELEPHONE_NUMBER, PR_PAGER_TELEPHONE_NUMBER_A, PR_PAGER_TELEPHONE_NUMBER_W, PR_BEEPER_TELEPHONE_NUMBER, PR_BEEPER_TELEPHONE_NUMBER_A, PR_BEEPER_TELEPHONE_NUMBER_W  <br/> |
|Identificador:  <br/> |0x3A21  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Área:  <br/> |Endereço  <br/> |
   
## <a name="remarks"></a>Comentários

Essas propriedades fornecem identificação e acessar informações para um destinatário. Eles são definidos por destinatário e suas respectivas organizações.
