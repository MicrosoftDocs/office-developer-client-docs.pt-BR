# <a name="pidlidtaskrecurrence-canonical-property"></a>Propriedade canônica PidLidTaskRecurrence

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém uma estrutura RecurrencePattern que fornece informações sobre tarefas recorrentes.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidTaskRecur  <br/> |
|Propriedade definida:  <br/> |PSETID_Task  <br/> |
|ID de longo (LID):  <br/> |0x00008116  <br/> |
|Tipo de dados:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Task  <br/> |
   
## <a name="remarks"></a>Comentários

Para obter informações sobre como criar e especificar uma estrutura RecurrencePattern, consulte [criar um Item de tarefa recorrente simples](how-to-create-a-simple-recurrent-task-item.md).
