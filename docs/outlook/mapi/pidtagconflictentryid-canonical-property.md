# <a name="pidtagconflictentryid-canonical-property"></a>Propriedade canônica PidTagConflictEntryId

  
  
**Aplica-se a**: Outlook 
  
Contém a EntryID da mensagem de resolução de conflito.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_CONFLICT_ENTRYID  <br/> |
|Identificador:  <br/> |0x3FF0  <br/> |
|Tipo de dados:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Sincronização de alteração incremental (ICS)  <br/> |
