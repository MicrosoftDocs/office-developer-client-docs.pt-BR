# <a name="pidtagcontentlength-canonical-property"></a>Propriedade canônica PidTagContentLength

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém um comprimento de mensagem, em bytes, passado para um aplicativo cliente ou provedor de serviços para determinar se uma mensagem desse tamanho pode ser entregue. 
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_CONTENT_LENGTH  <br/> |
|Identificador:  <br/> |0x0009  <br/> |
|Tipo de dados:  <br/> |PT_LONG  <br/> |
|Área:  <br/> |Exchange  <br/> |
