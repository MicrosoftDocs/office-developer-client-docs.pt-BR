# <a name="pidlidfax3entryid-canonical-property"></a>Propriedade canônica PidLidFax3EntryId

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Especifica a EntryID do outro endereço de fax do contato.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidFax3EntryID  <br/> |
|Conjunto de propriedades:  <br/> |PSETID_Address  <br/> |
|Long ID (LID):  <br/> |0x000080D1  <br/> |
|Tipo de dados:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Contato  <br/> |
