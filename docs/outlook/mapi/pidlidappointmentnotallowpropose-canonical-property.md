# <a name="pidlidappointmentnotallowpropose-canonical-property"></a>Propriedade canônica PidLidAppointmentNotAllowPropose

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Indica se os participantes não poderão propor uma nova data/hora para a reunião.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidApptNotAllowPropose  <br/> |
|Propriedade definida:  <br/> |PSETID_Appointment  <br/> |
|ID de longo (LID):  <br/> |0x0000825A  <br/> |
|Tipo de dados:  <br/> |PT_BOOLEAN  <br/> |
|Área:  <br/> |Reuniões  <br/> |
   
## <a name="remarks"></a>Comentários

Um valor falso, ou a ausência dessa propriedade indica que os participantes poderão propor uma nova data/hora.
