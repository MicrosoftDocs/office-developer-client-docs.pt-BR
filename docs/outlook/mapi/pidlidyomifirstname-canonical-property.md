# <a name="pidlidyomifirstname-canonical-property"></a>Propriedade canônica PidLidYomiFirstName

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Especifica a pronúncia fonética do nome do contato.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidYomiFirstName  <br/> |
|Propriedade definida:  <br/> |PSETID_Address  <br/> |
|ID de longo (LID):  <br/> |0x0000802C  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Contato  <br/> |
