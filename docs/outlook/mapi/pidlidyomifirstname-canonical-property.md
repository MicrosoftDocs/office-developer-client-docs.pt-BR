# <a name="pidlidyomifirstname-canonical-property"></a>Propriedade canônica PidLidYomiFirstName

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Especifica a pronúncia phoneética do nome determinado do contato.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidYomiFirstName  <br/> |
|Conjunto de propriedades:  <br/> |PSETID_Address  <br/> |
|Long ID (LID):  <br/> |0x0000802C  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Contato  <br/> |
