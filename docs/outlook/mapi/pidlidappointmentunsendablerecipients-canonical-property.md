# <a name="pidlidappointmentunsendablerecipients-canonical-property"></a>Propriedade canônica PidLidAppointmentUnsendableRecipients

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém uma lista de participantes de outro.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidApptUnsendableRecips  <br/> |
|Propriedade definida:  <br/> |PSETID_Appointment  <br/> |
|ID de longo (LID):  <br/> |0x0000823D  <br/> |
|Tipo de dados:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Reuniões  <br/> |
   
## <a name="remarks"></a>Comentários

Esta propriedade não é obrigatório, mas deve ser definida. Seu formato é detalhado nas [[MS-OXOCAL]](https://msdn.microsoft.com/library/09861fde-c8e4-4028-9346-e7c214cfdba1%28Office.15%29.aspx).
