# <a name="pidlidappointmentunsendablerecipients-canonical-property"></a>Propriedade canônica PidLidAppointmentUnsendableRecipients

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém uma lista de participantes nãoendáveis.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidApptUnsendableRecips  <br/> |
|Conjunto de propriedades:  <br/> |PSETID_Appointment  <br/> |
|Long ID (LID):  <br/> |0x0000823D  <br/> |
|Tipo de dados:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Reuniões  <br/> |
   
## <a name="remarks"></a>Comentários

Essa propriedade não é necessária, mas deve ser definida. Seu formato é detalhado em [[MS-OXOCAL]](https://msdn.microsoft.com/library/09861fde-c8e4-4028-9346-e7c214cfdba1%28Office.15%29.aspx).
