# <a name="pidlidtaskowner-canonical-property"></a>Propriedade canônica PidLidTaskOwner

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém o nome do proprietário da tarefa.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidTaskOwner  <br/> |
|Conjunto de propriedades:  <br/> |PSETID_Task  <br/> |
|Long ID (LID):  <br/> |0x0000811F  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE  <br/> |
