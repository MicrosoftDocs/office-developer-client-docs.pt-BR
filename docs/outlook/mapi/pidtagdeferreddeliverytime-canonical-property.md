# <a name="pidtagdeferreddeliverytime-canonical-property"></a>Propriedade canônica PidTagDeferredDeliveryTime

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém a data e a hora em que um remetente de mensagem deseja receber uma mensagem. 
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_DEFERRED_DELIVERY_TIME  <br/> |
|Identificador:  <br/> |0x000F  <br/> |
|Tipo de dados:  <br/> |PT_SYSTIME  <br/> |
|Área:  <br/> |Envelope MAPI  <br/> |
   
## <a name="remarks"></a>Comentários

O MAPI não realiza a entrega adiada; é uma opção do sistema de mensagens subjacente para lidar com a entrega adiada.
