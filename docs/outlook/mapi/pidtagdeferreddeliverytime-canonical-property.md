# <a name="pidtagdeferreddeliverytime-canonical-property"></a>Propriedade canônica PidTagDeferredDeliveryTime

  
  
**Aplica-se a**: Outlook 
  
Contém a data e hora quando um remetente da mensagem quiser uma mensagem entregue. 
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_DEFERRED_DELIVERY_TIME  <br/> |
|Identificador:  <br/> |0x000F  <br/> |
|Tipo de dados:  <br/> |PT_SYSTIME  <br/> |
|Área:  <br/> |Envelope MAPI  <br/> |
   
## <a name="remarks"></a>Comentários

MAPI não realiza a entrega adiada; é uma opção de sistema de mensagens subjacente para lidar com a entrega adiada.
