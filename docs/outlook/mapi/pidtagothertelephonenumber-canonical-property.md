# <a name="pidtagothertelephonenumber-canonical-property"></a>Propriedade canônica PidTagOtherTelephoneNumber

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém um número de telefone alternativo para o destinatário.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_OTHER_TELEPHONE_NUMBER, PR_OTHER_TELEPHONE_NUMBER_A, PR_OTHER_TELEPHONE_NUMBER_W  <br/> |
|Identificador:  <br/> |0x3A1F  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Área:  <br/> |Endereço  <br/> |
   
## <a name="remarks"></a>Comentários

Essas propriedades fornecem identificação e acessar informações para um destinatário. Eles são definidos por destinatário e suas respectivas organizações. 
  
Essas propriedades são usadas para um número de telefone diferente no lugar do destinatário de negócios, residencial ou do office.
