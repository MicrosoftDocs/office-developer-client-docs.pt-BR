# <a name="pidtagreplyrequested-canonical-property"></a>Propriedade canônica PidTagReplyRequested

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém TRUE se um remetente de mensagem solicita uma resposta de um destinatário.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_REPLY_REQUESTED  <br/> |
|Identificador:  <br/> |0x0C17  <br/> |
|Tipo de dados:  <br/> |PT_BOOLEAN  <br/> |
|Área:  <br/> |Destinatário MAPI  <br/> |
