# <a name="pidlidagingdontageme-canonical-property"></a>Propriedade canônica PidLidAgingDontAgeMe

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Especifica um valor true se o item não será arquivado automaticamente.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidAgingDontAgeMe  <br/> |
|Conjunto de propriedades:  <br/> |PSETID_Common  <br/> |
|Long ID (LID):  <br/> |0x0000850E  <br/> |
|Tipo de dados:  <br/> |PT_BOOLEAN  <br/> |
|Área:  <br/> |Comum  <br/> |
