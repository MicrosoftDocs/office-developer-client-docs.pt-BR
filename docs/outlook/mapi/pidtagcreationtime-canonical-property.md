# <a name="pidtagcreationtime-canonical-property"></a>Propriedade canônica PidTagCreationTime

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém a data e hora de criação de uma mensagem. 
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_CREATION_TIME  <br/> |
|Identificador:  <br/> |0x3007  <br/> |
|Tipo de dados:  <br/> |PT_SYSTIME  <br/> |
|Área:  <br/> |Hora da mensagem  <br/> |
   
## <a name="remarks"></a>Comentários

Um repositório de mensagens define essa propriedade para cada mensagem que ela cria.
