# <a name="pidtagrecipienttrackstatustime-canonical-property"></a>Propriedade canônica PidTagRecipientTrackStatusTime

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém a data e hora de quando o participante respondido.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_RECIPIENT_TRACKSTATUS_TIME  <br/> |
|Identificador:  <br/> |0x5FFB  <br/> |
|Tipo de dados:  <br/> |PT_SYSTIME  <br/> |
|Área:  <br/> |Destinatário de transporte  <br/> |
   
## <a name="remarks"></a>Comentários

O valor deve ser especificado no tempo Universal Coordenado (UTC).
