# <a name="pidtagpstpasswordszold-canonical-property"></a>Propriedade canônica PidTagPstPasswordSzOld

  
  
**Aplica-se a**: Outlook 
  
Contém a senha da tabela de armazenamento pessoal (arquivo. pst) que está sendo configurada.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_PST_PW_SZ_OLD, PR_PST_PW_SZ_OLD_A, PR_PST_PW_SZ_OLD_W  <br/> |
|Identificador:  <br/> |0x6703  <br/> |
|Tipo de dados:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Área:  <br/> |Tabela de armazenamento pessoal (. pst) interna  <br/> |
