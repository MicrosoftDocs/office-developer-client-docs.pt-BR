# <a name="pidtagstreetaddress-canonical-property"></a>Propriedade canônica PidTagStreetAddress

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém o endereço do destinatário. 
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_STREET_ADDRESS, PR_STREET_ADDRESS_A, PR_STREET_ADDRESS_W, PR_BUSINESS_ADDRESS_STREET, PR_BUSINESS_ADDRESS_STREET_A, PR_BUSINESS_ADDRESS_STREET_W  <br/> |
|Identificador:  <br/> |0x3A29  <br/> |
|Tipo de dados:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Área:  <br/> |Usuário de email MAPI  <br/> |
   
## <a name="remarks"></a>Comentários

Essas propriedades fornecem informações de identificação e acesso de um destinatário. Eles são definidos pelo destinatário e pela organização do destinatário.
