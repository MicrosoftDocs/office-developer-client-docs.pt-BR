# <a name="pidlidfax2originaldisplayname-canonical-property"></a>Propriedade canônico de PidLidFax2OriginalDisplayName

  
  
**Aplica-se a**: Outlook 
  
Especifica o nome para exibição original do endereço de fax residencial do contato.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidFax2OriginalDisplayName  <br/> |
|Propriedade definida:  <br/> |PSETID_Address  <br/> |
|ID de longo (LID):  <br/> |0x000080C4  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Contato  <br/> |
