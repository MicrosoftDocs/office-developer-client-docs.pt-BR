# <a name="pidlidresendtime-canonical-property"></a>Propriedade canônica PidLidResendTime

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Se definido em um relatório de não entrega, indica a hora em que a mensagem foi enviada novamente.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidResendTime  <br/> |
|Conjunto de propriedades:  <br/> |PSETID_Report  <br/> |
|Long ID (LID):  <br/> |0x00008D00  <br/> |
|Tipo de dados:  <br/> |PT_SYSTIME  <br/> |
|Área:  <br/> |Envio de mensagens gerais  <br/> |
