# <a name="pidlidotheraddresscountrycode-canonical-property"></a>Propriedade canônica PidLidOtherAddressCountryCode

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Especifica a parte do código do país do outro endereço do contato.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidOtherAddressCountryCode  <br/> |
|Conjunto de propriedades:  <br/> |PSETID_Address  <br/> |
|Long ID (LID):  <br/> |0x000080DC  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Contato  <br/> |
