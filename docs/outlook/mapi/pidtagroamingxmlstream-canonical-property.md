# <a name="pidtagroamingxmlstream-canonical-property"></a>Propriedade canônica PidTagRoamingXmlStream

  
  
**Aplica-se a**: Outlook 
  
Contém um fluxo XML arbitrário.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_ROAMING_XMLSTREAM  <br/> |
|Identificador:  <br/> |0x7C08  <br/> |
|Tipo de dados:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Configuração  <br/> |
   
## <a name="remarks"></a>Comentários

Essa propriedade contém um fluxo de arbitrário de dados XML. Outras propriedades na mensagem podem implicam esquemas específicas para usar essa propriedade.
