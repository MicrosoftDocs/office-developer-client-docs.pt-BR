# <a name="pidtagroamingxmlstream-canonical-property"></a>Propriedade canônica PidTagRoamingXmlStream

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém um Stream XML arbitrário.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_ROAMING_XMLSTREAM  <br/> |
|Identificador:  <br/> |0x7C08  <br/> |
|Tipo de dados:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Configuração  <br/> |
   
## <a name="remarks"></a>Comentários

Esta propriedade contém um fluxo arbitrário de dados XML. Outras propriedades da mensagem podem sugerir que esquemas específicos sejam usados nesta propriedade.
