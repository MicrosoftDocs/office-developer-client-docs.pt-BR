# <a name="outlook-mapi-reference"></a>Referência de MAPI do Outlook

 
  
**Aplica-se a**: Outlook 
  
Bem-vindo à Referência da Messaging API (MAPI) do Microsoft Outlook 2013. A MAPI fornece a arquitetura de mensagens do Microsoft Outlook 2013. A MAPI para Outlook 2013 fornece um conjunto de interfaces, funções e outros tipos de dados para facilitar o desenvolvimento de aplicativos de mensagens do Outlook 2013. Aplicativos e transportes usam MAPI para manipular dados de email, para criar mensagens de email e as pastas para armazená-las e para oferecer suporte a notificações de alterações em dados relacionados a MAPI existentes.
  
> [!NOTE]
> Se você está começando a desenvolver soluções para o Outlook, confira [Selecionando uma API e tecnologia para desenvolver soluções para o Outlook 2013](http://msdn.microsoft.com/pt-BR/library/jj900714.aspx) para identificar as APIs e tecnologias mais apropriadas para suas necessidades.
