# <a name="pidtaghomeaddressstreet-canonical-property"></a>Propriedade canônica PidTagHomeAddressStreet

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém a parte da rua do endereço do usuário.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_HOME_ADDRESS_STREET, PR_HOME_ADDRESS_STREET_A, PR_HOME_ADDRESS_STREET_W  <br/> |
|Identificador:  <br/> |0x3A5D  <br/> |
|Tipo de dados:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Área:  <br/> |Usuário de email MAPI  <br/> |
   
## <a name="remarks"></a>Comentários

Essas propriedades são definidas pelo usuário ou a organização do usuário.
