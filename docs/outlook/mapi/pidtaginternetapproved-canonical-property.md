# <a name="pidtaginternetapproved-canonical-property"></a>Propriedade canônica PidTagInternetApproved

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016

|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_INTERNET_APPROVED, PR_INTERNET_APPROVED_A, PR_INTERNET_APPROVED_W  <br/> |
|Identificador:  <br/> |0x1030  <br/> |
|Tipo de dados:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Área:  <br/> |MIME  <br/> |
