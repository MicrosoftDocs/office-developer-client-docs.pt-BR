# <a name="pidlidfax1addresstype-canonical-property"></a>Propriedade canônica PidLidFax1AddressType

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Especifica o tipo de endereço para o endereço de fax comercial de um contato.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidFax1AddrType  <br/> |
|Propriedade definida:  <br/> |PSETID_Address  <br/> |
|ID de longo (LID):  <br/> |0x000080B2  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Contato  <br/> |
   
## <a name="remarks"></a>Comentários

Essa propriedade, se houver, deve ser definida como "FAX".
