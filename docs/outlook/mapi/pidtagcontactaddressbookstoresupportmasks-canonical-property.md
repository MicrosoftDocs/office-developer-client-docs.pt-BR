# <a name="pidtagcontactaddressbookstoresupportmasks-canonical-property"></a>Propriedade canônica PidTagContactAddressBookStoreSupportMasks

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém sinalizadores que indicam os recursos com suporte da loja.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_CONTAB_STORE_SUPPORT_MASKS  <br/> |
|Identificador:  <br/> |0x6621  <br/> |
|Tipo de dados:  <br/> |PT_MV_LONG  <br/> |
|Área:  <br/> |Contact address book  <br/> |
   
## <a name="remarks"></a>Comentários

Essa propriedade é obtida dos armazenamentos que contêm as pastas Contatos. O provedor do Livro de Endereços de Contato o utiliza para avaliar a adequação dos recursos com suporte do armazenamento. É uma propriedade em uma seção de perfil do Livro de Endereços de Contato.
