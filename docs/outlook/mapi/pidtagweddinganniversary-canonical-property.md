# <a name="pidtagweddinganniversary-canonical-property"></a>Propriedade canônica PidTagWeddingAnniversary

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém a data de aniversário de chá de um usuário.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_WEDDING_ANNIVERSARY  <br/> |
|Identificador:  <br/> |0x3A41  <br/> |
|Tipo de dados:  <br/> |PT_SYSTIME  <br/> |
|Área:  <br/> |Usuário de email MAPI  <br/> |
   
## <a name="remarks"></a>Comentários

Essa propriedade é definida pelo usuário ou a organização do usuário.
