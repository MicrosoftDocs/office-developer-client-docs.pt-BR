# <a name="pidtagrulename-canonical-property"></a>Propriedade canônica PidTagRuleName

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Especifica o nome da regra que possa ser entendido pelo usuário.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_RULE_NAME  <br/> |
|Identificador:  <br/> |0x6682  <br/> |
|Tipo de dados:  <br/> |PT_STRING8  <br/> |
|Área:  <br/> |Regras no servidor  <br/> |
