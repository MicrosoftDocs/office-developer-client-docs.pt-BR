# <a name="pidtagtransportproviders-canonical-property"></a>Propriedade canônica PidTagTransportProviders

  
  
**Aplica-se a**: Outlook 
  
Contém uma lista de identificadores de provedores de transporte no perfil atual.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_TRANSPORT_PROVIDERS  <br/> |
|Identificador:  <br/> |0x3D02  <br/> |
|Tipo de dados:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Perfil MAPI  <br/> |
   
## <a name="remarks"></a>Comentários

Não use essa propriedade. Ele é reservado para uso pelo MAPI.
