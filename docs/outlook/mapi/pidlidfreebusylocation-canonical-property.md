# <a name="pidlidfreebusylocation-canonical-property"></a>Propriedade canônica PidLidFreeBusyLocation

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Especifica um caminho de URL a partir do qual um cliente pode recuperar informações de disponibilidade para o contato como um arquivo iCal, conforme especificado em [[MS-OXCICAL]](https://msdn.microsoft.com/library/a685a040-5b69-4c84-b084-795113fb4012%28Office.15%29.aspx).
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidFreeBusyLocation  <br/> |
|Conjunto de propriedades:  <br/> |PSETID_Address  <br/> |
|Long ID (LID):  <br/> |0x000080D8  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Contato  <br/> |
