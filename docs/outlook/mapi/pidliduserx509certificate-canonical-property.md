# <a name="pidliduserx509certificate-canonical-property"></a>Propriedade canônica PidLidUserX509Certificate

  
  
**Aplica-se a**: Outlook 
  
Contém os certificados de segurança x. 509 versão 3 para um usuário de mensagens.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidEMSAbX509Cert  <br/> |
|Propriedade definida:  <br/> |PSETID_Address  <br/> |
|ID de longo (LID):  <br/> |0x000080D9  <br/> |
|Tipo de dados:  <br/> |PT_MV_BINARY  <br/> |
|Área:  <br/> |Contato  <br/> |
