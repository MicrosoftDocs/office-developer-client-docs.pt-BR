# <a name="pidtagtransportkey-canonical-property"></a>Propriedade canônica PidTagTransportKey

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém um valor usado pelo spooler MAPI para controlar o progresso de uma mensagem de saída por meio dos provedores de transporte de saída.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_TRANSPORT_KEY  <br/> |
|Identificador:  <br/> |0x0E16  <br/> |
|Tipo de dados:  <br/> |PT_LONG  <br/> |
|Área:  <br/> |MAPI não transmitível  <br/> |
   
## <a name="remarks"></a>Comentários

Não use essa propriedade. Ele é reservado para uso pelo MAPI.
