# <a name="pidtagttytddphonenumber-canonical-property"></a>Propriedade canônica PidTagGfTddPhoneNumber

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém o número de telefone do TTY (telefone de texto) ou dispositivo de telecomunicações para pessoas com deficiência auditiva (TDD).
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_TTYTDD_PHONE_NUMBER, PR_TTYTDD_PHONE_NUMBER_A, PR_TTYTDD_PHONE_NUMBER_W  <br/> |
|Identificador:  <br/> |0x3A4B  <br/> |
|Tipo de dados:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Área:  <br/> |Contato  <br/> |
