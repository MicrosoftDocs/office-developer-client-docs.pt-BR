# <a name="pidtagrpcoverhttpproxyserver-canonical-property"></a>Propriedade canônica PidTagRpcOverHttpProxyServer

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém a URL do servidor proxy usada para conectar o Microsoft Outlook ao Microsoft Exchange Server sem o protocolo HTTP ou o protocolo HTTPS (Hypertext Transfer Protocol Secure) que vem antes da URL.
  
## 

|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_ROH_PROXY_SERVER  <br/> |
|Identificador:  <br/> |0x6622  <br/> |
|Tipo de propriedade:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Diversos  <br/> |
