# <a name="pidtagnickname-canonical-property"></a>Propriedade canônica PidTagNickname

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém o apelido do contato.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_NICKNAME, PR_NICKNAME_A, PR_NICKNAME_W  <br/> |
|Identificador:  <br/> |0x3A4F  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Área:  <br/> |Endereço  <br/> |
   
## <a name="remarks"></a>Comentários

Essas propriedades são opcionais. Seu conteúdo é definido pelo usuário.
