# <a name="pidlidattendeecriticalchange-canonical-property"></a>Propriedade canônica PidLidAttendeeCriticalChange

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Especifica a data e hora em que o objeto de reuniões foi enviado.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |LID_ATTENDEE_CRITICAL_CHANGE  <br/> |
|Propriedade definida:  <br/> |PSETID_Meeting  <br/> |
|ID de longo (LID):  <br/> |0x00000001  <br/> |
|Tipo de dados:  <br/> |PT_SYSTIME  <br/> |
|Área:  <br/> |Reuniões  <br/> |
   
## <a name="remarks"></a>Comentários

O valor deve ser especificado no tempo Universal Coordenado (UTC).
