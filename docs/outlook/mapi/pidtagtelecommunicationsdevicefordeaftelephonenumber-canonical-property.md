# <a name="pidtagtelecommunicationsdevicefordeaftelephonenumber-canonical-property"></a>Propriedade canônica PidTagTelecommunicationsDeviceForDeafTelephoneNumber

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém o dispositivo de telecomunicações do usuário de mensagens para o número de telefone surdo (TTYTDD).
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_TTYTDD_PHONE_NUMBER, PR_TTYTDD_PHONE_NUMBER_A, PR_TTYTDD_PHONE_NUMBER_W  <br/> |
|Identificador:  <br/> |0x3A4B  <br/> |
|Tipo de dados:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Área:  <br/> |Usuário de email MAPI  <br/> |
   
## <a name="remarks"></a>Comentários

Essas propriedades fornecem informações de identificação e acesso sobre um usuário de mensagens. O conteúdo é definido pelo usuário de mensagens e pela organização do usuário de mensagens.
