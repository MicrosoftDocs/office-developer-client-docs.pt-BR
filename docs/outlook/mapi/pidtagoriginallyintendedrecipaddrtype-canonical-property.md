# <a name="pidtagoriginallyintendedrecipaddrtype-canonical-property"></a>Propriedade canônica PidTagOriginallyIntendedRecipAddrtype

  
  
**Aplica-se a**: Outlook 
  
Contém o tipo de endereço do destinatário pretendido originalmente de uma mensagem de autoforwarded.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_ORIGINALLY_INTENDED_RECIP_ADDRTYPE, PR_ORIGINALLY_INTENDED_RECIP_ADDRTYPE_A, PR_ORIGINALLY_INTENDED_RECIP_ADDRTYPE_W  <br/> |
|Identificador:  <br/> |0x007B  <br/> |
|Tipo de dados:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Área:  <br/> |Servidor  <br/> |
   
## <a name="remarks"></a>Comentários

Essas propriedades são uma das propriedades de endereço do destinatário pretendido originalmente da mensagem. Ele deve ser definido pelo agente de automática que encaminhou a mensagem.
