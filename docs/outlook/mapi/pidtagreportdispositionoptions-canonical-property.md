# <a name="pidtagreportdispositionoptions-canonical-property"></a>Propriedade canônica PidTagReportDispositionOptions

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Mapeia para o Disposition-Options mime para mensagens que solicitam recibos. O valor dessa propriedade deve ser uma cadeia de caracteres vazia.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_REPORT_DISPOSITION_OPTIONS, PR_REPORT_DISPOSITION_OPTIONS_A, PR_REPORT_DISPOSITION_OPTIONS_W  <br/> |
|Identificador:  <br/> |0x0085  <br/> |
|Tipo de dados:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Área:  <br/> |Envelope MAPI  <br/> |
