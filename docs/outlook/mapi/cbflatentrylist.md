# <a name="cbflatentrylist"></a>CbFLATENTRYLIST

  
  
**Aplica-se a**: Outlook 
  
Calcula o número de bytes em uma estrutura [FLATENTRYLIST](flatentrylist.md) existente. 
  
|||
|:-----|:-----|
|Arquivo de cabeçalho:  <br/> |Mapidefs.h  <br/> |
|Estrutura relacionada:  <br/> |**FLATENTRYLIST** <br/> |
   
```cpp
CbFLATENTRYLIST(_lplist)
```

## <a name="parameters"></a>Parâmetros

 __lplist_
  
> Ponteiro para a estrutura **FLATENTRYLIST** em questão.
