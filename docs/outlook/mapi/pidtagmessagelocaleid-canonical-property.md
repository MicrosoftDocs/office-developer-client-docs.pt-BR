# <a name="pidtagmessagelocaleid-canonical-property"></a>Propriedade canônica PidTagMessageLocaleId

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém o LCID do Windows do usuário final que criou essa mensagem.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_MESSAGE_LOCALE_ID  <br/> |
|Identificador:  <br/> |0x3FF1  <br/> |
|Tipo de dados:  <br/> |PT_LONG  <br/> |
|Área:  <br/> |Diversos  <br/> |
