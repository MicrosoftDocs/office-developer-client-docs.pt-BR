# <a name="pidtaginternetcodepage-canonical-property"></a>Propriedade canônica PidTagInternetCodepage

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Indica a página de código usada **para PR_BODY** ([PidTagBody](pidtagbody-canonical-property.md)) ou PR_BODY_HTML **(** [PidTagBodyHtml](pidtagbodyhtml-canonical-property.md)) propriedades.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_INTERNET_CPID  <br/> |
|Identificador:  <br/> |0x3FDE  <br/> |
|Tipo de dados:  <br/> |PT_LONG  <br/> |
|Área:  <br/> |Diversos  <br/> |
