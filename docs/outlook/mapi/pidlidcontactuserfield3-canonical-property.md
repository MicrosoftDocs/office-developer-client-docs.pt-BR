# <a name="pidlidcontactuserfield3-canonical-property"></a>Propriedade canônica PidLidContactUserField3

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Especifica o terceiro campo do contato destinado ao uso diversos do contato.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidContactUserField3  <br/> |
|Conjunto de propriedades:  <br/> |PSETID_Address  <br/> |
|Long ID (LID):  <br/> |0x00008051  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Contato  <br/> |
   
## <a name="remarks"></a>Comentários

Essa propriedade contém texto que não está relacionado a qualquer outra propriedade específica do contato.
