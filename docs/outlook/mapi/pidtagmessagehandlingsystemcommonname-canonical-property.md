# <a name="pidtagmessagehandlingsystemcommonname-canonical-property"></a>Propriedade canônica PidTagMessageHandlingSystemCommonName

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém o nome comum do sistema de manipulação de mensagens.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_MHS_COMMON_NAME, PR_MHS_COMMON_NAME_A, PR_MHS_COMMON_NAME_W  <br/> |
|Identificador:  <br/> |0x3A0F  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Área:  <br/> |Endereço  <br/> |
   
## <a name="remarks"></a>Comentários

Essas propriedades contêm o nome comum do sistema de manipulação de mensagens.
