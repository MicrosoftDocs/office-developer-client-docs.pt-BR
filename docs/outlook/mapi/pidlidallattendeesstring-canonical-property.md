# <a name="pidlidallattendeesstring-canonical-property"></a>Propriedade canônica PidLidAllAttendeesString

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Especifica uma lista de todos os participantes, exceto o organizador, incluindo recursos e participantes não enviados.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidAllAttendeesString  <br/> |
|Conjunto de propriedades:  <br/> |PSETID_Appointment  <br/> |
|Long ID (LID):  <br/> |0x00008238  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Meetings  <br/> |
   
## <a name="remarks"></a>Comentários

O valor de cada participante é o nome de exibição do participante. Entradas separadas devem ser delimitadas por ponto-e-vírgula seguido por um espaço. Esta propriedade não é obrigatória.
