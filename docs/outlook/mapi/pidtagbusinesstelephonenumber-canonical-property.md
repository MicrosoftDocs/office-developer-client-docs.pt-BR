# <a name="pidtagbusinesstelephonenumber-canonical-property"></a>Propriedade canônica PidTagBusinessTelephoneNumber

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém o número de telefone principal do local do destinatário de negócios. 
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_BUSINESS_TELEPHONE_NUMBER, PR_BUSINESS_TELEPHONE_NUMBER_A, PR_BUSINESS_TELEPHONE_NUMBER_W, PR_OFFICE_TELEPHONE_NUMBER, PR_OFFICE_TELEPHONE_NUMBER_A, PR_OFFICE_TELEPHONE_NUMBER_W  <br/> |
|Identificador:  <br/> |0x3A08  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Área:  <br/> |Contato  <br/> |
   
## <a name="remarks"></a>Comentários

Essas propriedades são exemplos das propriedades que fornece acesso a informações sobre um destinatário e identificação. Essas propriedades são definidas por destinatário e a organização do destinatário.
