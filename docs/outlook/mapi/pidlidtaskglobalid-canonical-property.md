# <a name="pidlidtaskglobalid-canonical-property"></a>Propriedade canônica PidLidTaskGlobalId

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Localiza uma tarefa existente, usando um GUID, no recebimento de uma atualização de tarefa ou uma resposta de tarefa.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidTaskGlobalObjId  <br/> |
|Propriedade definida:  <br/> |PSETID_Common  <br/> |
|ID de longo (LID):  <br/> |0x00008519  <br/> |
|Tipo de dados:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Task  <br/> |
   
## <a name="remarks"></a>Comentários

Essa propriedade é da esquerda não definida para tarefas não atribuídas.
