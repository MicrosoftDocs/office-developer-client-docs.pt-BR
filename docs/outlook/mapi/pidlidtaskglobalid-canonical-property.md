# <a name="pidlidtaskglobalid-canonical-property"></a>Propriedade canônica PidLidTaskGlobalId

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Localiza uma tarefa existente, usando um GUID, após o recebimento de uma resposta de tarefa ou de uma atualização de tarefa.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidTaskGlobalObjId  <br/> |
|Conjunto de propriedades:  <br/> |PSETID_Common  <br/> |
|Long ID (LID):  <br/> |0x00008519  <br/> |
|Tipo de dados:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Tarefa  <br/> |
   
## <a name="remarks"></a>Comentários

Esta propriedade é esquerda desdefinida para tarefas não atribuídas.
