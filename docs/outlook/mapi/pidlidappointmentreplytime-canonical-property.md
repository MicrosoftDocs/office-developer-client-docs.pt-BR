# <a name="pidlidappointmentreplytime-canonical-property"></a>Propriedade canônica PidLidAppointmentReplyTime

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Especifica a data e a hora em que o participante respondeu a uma solicitação de reunião recebida ou atualização da reunião.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidApptReplyTime  <br/> |
|Conjunto de propriedades:  <br/> |PSETID_Appointment  <br/> |
|Long ID (LID):  <br/> |0x00008220  <br/> |
|Tipo de dados:  <br/> |PT_SYSTIME  <br/> |
|Área:  <br/> |Reuniões  <br/> |
   
## <a name="remarks"></a>Comentários

O valor deve ser especificado em UTC (Tempo Universal Coordenado).
