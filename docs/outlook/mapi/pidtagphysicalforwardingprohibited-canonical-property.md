# <a name="pidtagphysicalforwardingprohibited-canonical-property"></a>Propriedade canônica PidTagPhysicalForwardingProhibited

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém TRUE se um remetente de mensagem proíbe o encaminhamento de mensagens físicas para um destinatário específico.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_PHYSICAL_FORWARDING_PROHIBITED  <br/> |
|Identificador:  <br/> |0x0C0F  <br/> |
|Tipo de dados:  <br/> |PT_BOOLEAN  <br/> |
|Área:  <br/> |Destinatário MAPI  <br/> |
