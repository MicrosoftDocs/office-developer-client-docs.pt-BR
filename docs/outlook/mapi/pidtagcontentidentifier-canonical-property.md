# <a name="pidtagcontentidentifier-canonical-property"></a>Propriedade canônica PidTagContentIdentifier

  
  
**Aplica-se a**: Outlook 
  
Contém um valor de chave que permite que o destinatário da mensagem identificar o seu conteúdo.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_CONTENT_IDENTIFIER, PR_CONTENT_IDENTIFIER_A, PR_CONTENT_IDENTIFIER_W  <br/> |
|Identificador:  <br/> |0x0008  <br/> |
|Tipo de dados:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Área:  <br/> |Exchange  <br/> |
