# <a name="pidlidworkaddress-canonical-property"></a>Propriedade canônica PidLidWorkAddress

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Especifica o endereço comercial completo do contato.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidWorkAddress  <br/> |
|Conjunto de propriedades:  <br/> |PSETID_Address  <br/> |
|Long ID (LID):  <br/> |0x0000801B  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Contato  <br/> |
   
## <a name="remarks"></a>Comentários

Essa propriedade deve ser uma combinação de outras propriedades de endereço físico e se baseia na localidade do cliente.
