# <a name="pidlidotheraddress-canonical-property"></a>Propriedade canônica PidLidOtherAddress

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Especifica que o endereço completo do contato a outro endereço do.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidOtherAddress  <br/> |
|Propriedade definida:  <br/> |PSETID_Address  <br/> |
|ID de longo (LID):  <br/> |0x0000801C  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Contato  <br/> |
   
## <a name="remarks"></a>Comentários

Esta propriedade deve ser uma combinação das outras propriedades de endereço físico e baseia-se no idioma do cliente.
