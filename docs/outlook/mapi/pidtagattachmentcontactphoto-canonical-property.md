# <a name="pidtagattachmentcontactphoto-canonical-property"></a>Propriedade canônica PidTagAttachmentContactPhoto

  
  
**Aplica-se a**: Outlook 
  
Indica a existência de um anexo de foto de um contato.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_ATTACHMENT_CONTACTPHOTO  <br/> |
|Identificador:  <br/> |0x7FFF  <br/> |
|Tipo de dados:  <br/> |PT_BOOLEAN  <br/> |
|Área:  <br/> |Anexo de mensagem  <br/> |
   
## <a name="remarks"></a>Comentários

Este valor de propriedade deve ser definida como TRUE e deve haver apenas um anexo em um determinado objeto de contato.
