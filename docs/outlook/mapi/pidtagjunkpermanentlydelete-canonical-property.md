# <a name="pidtagjunkpermanentlydelete-canonical-property"></a>Propriedade canônica PidTagJunkPermanentlyDelete

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Significa que, se definido como 1, essas mensagens identificadas como spam podem ser excluídas permanentemente.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_JUNK_PERMANENTLY_DELETE  <br/> |
|Identificador:  <br/> |0x6102  <br/> |
|Tipo de dados:  <br/> |PT_LONG  <br/> |
|Área:  <br/> |Spam  <br/> |
