# <a name="pidtagformhidden-canonical-property"></a>Propriedade canônica PidTagFormHidden

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém TRUE se um formulário deve ser suprimida da exibição por redação menus e caixas de diálogo. 
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_FORM_HIDDEN  <br/> |
|Identificador:  <br/> |0x3307  <br/> |
|Tipo de dados:  <br/> |PT_BOOLEAN  <br/> |
|Área:  <br/> |MAPI comuns  <br/> |
   
## <a name="remarks"></a>Comentários

Propriedades relacionadas ao formulário são somente leitura.
