# <a name="pidtagstartdate-canonical-property"></a>Propriedade canônica PidTagStartDate

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém a data e a hora de início de um compromisso como gerenciados por um aplicativo de agendamento.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_START_DATE  <br/> |
|Identificador:  <br/> |0x0060  <br/> |
|Tipo de dados:  <br/> |PT_SYSTIME  <br/> |
|Área:  <br/> |Envelope MAPI  <br/> |
   
## <a name="remarks"></a>Comentários

Os aplicativos de agendamento devem definir essa propriedade **e PR_END_DATE** ([PidTagEndDate](pidtagenddate-canonical-property.md)) ao enviar solicitações de reunião.
