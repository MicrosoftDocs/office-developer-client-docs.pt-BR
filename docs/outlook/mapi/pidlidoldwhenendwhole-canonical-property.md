# <a name="pidlidoldwhenendwhole-canonical-property"></a>Propriedade canônica PidLidOldWhenEndWhole

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Indica o valor original da propriedade **dispidApptEndWhole** ([PidLidAppointmentEndWhole](pidlidappointmentendwhole-canonical-property.md)) antes de uma atualização de reunião.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidOldWhenEndWhole  <br/> |
|Conjunto de propriedades:  <br/> |PSETID_Meeting  <br/> |
|Long ID (LID):  <br/> |0x0000002A  <br/> |
|Tipo de dados:  <br/> |PT_SYSTIME  <br/> |
|Área:  <br/> |Meetings  <br/> |
   
## <a name="remarks"></a>Comentários

Esta propriedade não é obrigatória.
