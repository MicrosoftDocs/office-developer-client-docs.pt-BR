# <a name="pidtagmiddlename-canonical-property"></a>Propriedade canônica PidTagMiddleName

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém o nome do meio de um contato.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_MIDDLE_NAME, PR_MIDDLE_NAME_A, PR_MIDDLE_NAME_W  <br/> |
|Identificador:  <br/> |0x3A44  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Área:  <br/> |Endereço  <br/> |
   
## <a name="remarks"></a>Comentários

Essas propriedades são definidas pelo usuário.
