# <a name="pidnamewordcount-canonical-property"></a>Propriedade canônica PidNameWordCount

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Especifica a contagem de palavras do arquivo anexado ao documento.
  
|||
|:-----|:-----|
|Nomes amigáveis:  <br/> |Nenhum  <br/> |
|Conjunto de propriedades:  <br/> |PS_PUBLIC_STRINGS  <br/> |
|Nome da propriedade:  <br/> |WordCount  <br/> |
|Tipo de dados:  <br/> |PT_LONG  <br/> |
|Área:  <br/> |Comum  <br/> |
