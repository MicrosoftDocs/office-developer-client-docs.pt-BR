# <a name="pidtagmessagedeliveryid-canonical-property"></a>Propriedade canônica PidTagMessageDeliveryId

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém um identificador MTS (sistema de transferência de mensagens) para uma mensagem entregue a um aplicativo cliente.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_MESSAGE_DELIVERY_ID  <br/> |
|Identificador:  <br/> |0x001B  <br/> |
|Tipo de dados:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Servidor  <br/> |
