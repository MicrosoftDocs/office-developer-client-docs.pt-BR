# <a name="pidtagreportdispositiontoemailaddresses-canonical-property"></a>Propriedade canônica PidTagReportDispositionToEmailAddresses

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Indica os endereços de email dos destinatários para mensagens que solicitar confirmações de relatório.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_REPORT_DISPOSITION_TO_EMAIL_ADDRESSES, PR_REPORT_DISPOSITION_TO_EMAIL_ADDRESSES_A, PR_REPORT_DISPOSITION_TO_EMAIL_ADDRESSES_W  <br/> |
|Identificador:  <br/> |0x0084  <br/> |
|Tipo de dados:  <br/> |PT_MV_STRING8, PT_UNICODE  <br/> |
|Área:  <br/> |Envelope MAPI  <br/> |
