# <a name="pidlidlinkedtaskitems-canonical-property"></a>Propriedade canônica PidLidLinkedTaskItems

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Especifica uma lista da propriedade **PR_ENTRYID** ([PidTagEntryId](pidtagentryid-canonical-property.md)) das tarefas relacionadas ao calendário. 
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidLinkedTaskItems  <br/> |
|Conjunto de propriedades:  <br/> |PSETID_Appointment  <br/> |
|Long ID (LID):  <br/> |0x0000820C  <br/> |
|Tipo de dados:  <br/> |PT_MV_BINARY  <br/> |
|Área:  <br/> |Tarefa  <br/> |
   
## <a name="remarks"></a>Comentários

Esta propriedade não é obrigatória.
