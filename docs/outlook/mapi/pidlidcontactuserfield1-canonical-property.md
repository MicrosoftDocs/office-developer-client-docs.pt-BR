# <a name="pidlidcontactuserfield1-canonical-property"></a>Propriedade canônica PidLidContactUserField1

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Especifica o primeiro campo no contato destinado ao uso variado do contato.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidContactUserField1  <br/> |
|Conjunto de propriedades:  <br/> |PSETID_Address  <br/> |
|Long ID (LID):  <br/> |0x0000804F  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Contato  <br/> |
   
## <a name="remarks"></a>Comentários

Esta propriedade contém texto que não está relacionado a qualquer outra propriedade específica de contato.
