# <a name="pidlidclassificationdescription-canonical-property"></a>Propriedade canônica PidLidClassificationDescription

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Descreve o que a classificação de mensagens pretende alcançar.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidClassDesc  <br/> |
|Conjunto de propriedades:  <br/> |PSETID_Common  <br/> |
|Long ID (LID):  <br/> |0x000085B7  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Envio de mensagens gerais  <br/> |
   
## <a name="remarks"></a>Comentários
