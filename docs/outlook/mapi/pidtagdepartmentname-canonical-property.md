# <a name="pidtagdepartmentname-canonical-property"></a>Propriedade canônica PidTagDepartmentName

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém um nome para o departamento no qual o destinatário funciona. 
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_DEPARTMENT_NAME, PR_DEPARTMENT_NAME_A, PR_DEPARTMENT_NAME_W  <br/> |
|Identificador:  <br/> |0x3A18  <br/> |
|Tipo de dados:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Área:  <br/> |Usuário de email MAPI  <br/> |
   
## <a name="remarks"></a>Comentários

Essas propriedades fornecem informações de identificação e acesso sobre um destinatário. Eles são definidos pelo destinatário e sua organização.
