# <a name="pidlidpagedirstream-canonical-property"></a>Propriedade canônica PidLidPageDirStream

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Representa informações sobre páginas do formulário personalizado associado à mensagem.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidPageDirStream  <br/> |
|Propriedade definida:  <br/> |PSETID_Common  <br/> |
|ID de longo (LID):  <br/> |0x00008513  <br/> |
|Tipo de dados:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Configuração de tempo de execução  <br/> |
