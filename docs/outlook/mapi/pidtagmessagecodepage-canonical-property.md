# <a name="pidtagmessagecodepage-canonical-property"></a>Propriedade canônica PidTagMessageCodepage

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém a página de código que é usada para a mensagem.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_MESSAGE_CODEPAGE  <br/> |
|Identificador:  <br/> |0x3FFD  <br/> |
|Tipo de dados:  <br/> |PT_LONG  <br/> |
|Área:  <br/> |Comuns  <br/> |
   
## <a name="remarks"></a>Comentários

A página de código do objeto de pasta é usada se essa propriedade for definida como zero (0).
