# <a name="pidtagdiscardreason-canonical-property"></a>Propriedade canônica PidTagDiscardReason

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém um motivo pelo qual um MTA (agente de transferência de mensagens) descartou uma mensagem. 
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_DISCARD_REASON  <br/> |
|Identificador:  <br/> |0x0011  <br/> |
|Tipo de dados:  <br/> |PT_LONG  <br/> |
|Área:  <br/> |Envelope MAPI  <br/> |
   
## <a name="remarks"></a>Comentários

O motivo contido nessa propriedade é usado em um relatório de não entrega da mensagem.
