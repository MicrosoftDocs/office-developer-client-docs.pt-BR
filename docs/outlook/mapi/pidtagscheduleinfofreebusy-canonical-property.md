# <a name="pidtagscheduleinfofreebusy-canonical-property"></a>Propriedade canônica PidTagScheduleInfoFreeBusy

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém informações obsoletas.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_SCHDINFO_FREEBUSY  <br/> |
|Identificador:  <br/> |0x686C  <br/> |
|Tipo de dados:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Informações de disponibilidade  <br/> |
   
## <a name="remarks"></a>Comentários

Essa propriedade não deve ser definida e deve ser ignorada.
