# <a name="pidlidfax1emailtype-canonical-property"></a>Propriedade canônica PidLidFax1EmailType

  
  
**Aplica-se a**: Outlook 
  
Especifica o tipo de email de endereço de fax comercial do contato.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidFax1EmailType  <br/> |
|Propriedade definida:  <br/> |PSETID_Address  <br/> |
|ID de longo (LID):  <br/> |0x000080B7  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Contato  <br/> |
