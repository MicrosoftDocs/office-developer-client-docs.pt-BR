# <a name="pidtaginitials-canonical-property"></a>Propriedade canônica PidTagInitials Canonical Property

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém as iniciais para partes do nome completo do destinatário. 
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_INITIALS, PR_INITIALS_A, PR_INITIALS_W  <br/> |
|Identificador:  <br/> |0x3A0A  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Área:  <br/> |Endereço  <br/> |
   
## <a name="remarks"></a>Comentários

Essas propriedades fornecem identificação e acessar informações para um destinatário. Eles são definidos por destinatário e suas respectivas organizações.
