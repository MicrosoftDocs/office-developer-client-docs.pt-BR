# <a name="pidlidlogtype-canonical-property"></a>Propriedade canônica PidLidLogType

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém o nome de exibição do aplicativo de registro no diário (por exemplo, "MSWord") e normalmente é um atributo de forma livre de uma mensagem de diário, geralmente uma cadeia de caracteres.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidLogType  <br/> |
|Conjunto de propriedades:  <br/> |PSETID_Log  <br/> |
|Long ID (LID):  <br/> |0x00008700  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Diário  <br/> |
