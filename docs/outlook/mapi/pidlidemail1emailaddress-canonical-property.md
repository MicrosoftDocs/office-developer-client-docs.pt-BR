# <a name="pidlidemail1emailaddress-canonical-property"></a>Propriedade canônica PidLidEmail1EmailAddress

  
  
**Aplica-se a**: Outlook 
  
Especifica o primeiro endereço de email do contato.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidEmail1EmailAddress  <br/> |
|Propriedade definida:  <br/> |PSETID_Address  <br/> |
|ID de longo (LID):  <br/> |0x00008083  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Contato  <br/> |
   
## <a name="remarks"></a>Comentários

O valor dessa propriedade deve ser apropriado para o tipo de endereço especificado para este endereço de email.
