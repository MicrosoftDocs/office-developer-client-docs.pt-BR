# <a name="pidlidhomeaddress-canonical-property"></a>Propriedade canônica PidLidHomeAddress

  
  
**Aplica-se a**: Outlook 
  
Especifica o endereço completo do endereço residencial do contato.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidHomeAddress  <br/> |
|Propriedade definida:  <br/> |PSETID_Address  <br/> |
|ID de longo (LID):  <br/> |0x0000801A  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Contato  <br/> |
   
## <a name="remarks"></a>Comentários

Esta propriedade deve ser uma combinação das outras propriedades de endereço físico e baseia-se no idioma do cliente.
