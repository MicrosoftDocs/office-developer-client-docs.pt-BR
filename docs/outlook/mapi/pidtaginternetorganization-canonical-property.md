# <a name="pidtaginternetorganization-canonical-property"></a>Propriedade canônica PidTagInternetOrganization

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Mapeia para o cabeçalho de organização MIME.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_INTERNET_ORGANIZATION, PR_INTERNET_ORGANIZATION_A, PR_INTERNET_ORGANIZATION_W  <br/> |
|Identificador:  <br/> |0x1037  <br/> |
|Tipo de dados:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Área:  <br/> |MIME  <br/> |
