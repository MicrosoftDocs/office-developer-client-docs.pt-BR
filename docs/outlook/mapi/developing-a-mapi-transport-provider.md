# <a name="developing-a-mapi-transport-provider"></a>Desenvolver um provedor de transporte MAPI

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Você deve estar familiarizado com a arquitetura de subsistema MAPI, a linguagem C++ e com a gravação de bibliotecas de vínculos dinâmicos (DLLs) para as diversas plataformas do Windows antes de desenvolver um provedor de transporte.
