# <a name="pidlidfexceptionalbody-canonical-property"></a>Propriedade canônica PidLidFExceptionalBody

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Indica que a exceção mensagem incorporada do objeto tem um corpo que difere do objeto de calendário recorrente.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidFExceptionalBody  <br/> |
|Propriedade definida:  <br/> |PSETID_Appointment  <br/> |
|ID de longo (LID):  <br/> |0x00008206  <br/> |
|Tipo de dados:  <br/> |PT_BOOLEAN  <br/> |
|Área:  <br/> |Reuniões  <br/> |
   
## <a name="remarks"></a>Comentários

Se o valor dessa propriedade for TRUE, a exceção incorporada objeto deve ter um corpo de mensagem. Se o valor dessa propriedade é FALSE, ou se a propriedade não existir, um cliente ou servidor deve obter o corpo do objeto de calendário recorrente.
