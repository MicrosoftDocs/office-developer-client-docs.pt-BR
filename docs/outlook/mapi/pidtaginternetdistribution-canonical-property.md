# <a name="pidtaginternetdistribution-canonical-property"></a>Propriedade canônica PidTagInternetDistribution

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Mapeia para o cabeçalho de distribuição MIME.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_INTERNET_DISTRIBUTION, PR_INTERNET_DISTRIBUTION_A, PR_INTERNET_DISTRIBUTION_W  <br/> |
|Identificador:  <br/> |0x1032  <br/> |
|Tipo de dados:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Área:  <br/> |MIME  <br/> |
