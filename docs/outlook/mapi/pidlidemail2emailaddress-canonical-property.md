# <a name="pidlidemail2emailaddress-canonical-property"></a>Propriedade canônica PidLidEmail2EmailAddress

  
  
**Aplica-se a**: Outlook 
  
Especifica o segundo endereço de email do contato.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidEmail2EmailAddress  <br/> |
|Propriedade definida:  <br/> |PSETID_Address  <br/> |
|ID de longo (LID):  <br/> |0x00008093  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Contato  <br/> |
   
## <a name="remarks"></a>Comentários

O valor dessa propriedade deve ser apropriado para o tipo de endereço especificado para este endereço de email.
