# <a name="cbnewssortorderset"></a>CbNewSSortOrderSet

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Calcula o número de bytes a ser alocada para uma nova estrutura de [SizedSSortOrderSet](sizedssortorderset.md) que contém um número especificado de ordens de classificação representado por estruturas de [SSortOrder](ssortorder.md) . 
  
|||
|:-----|:-----|
|Arquivo de cabeçalho:  <br/> |Mapidefs.h  <br/> |
|Estrutura relacionada:  <br/> |**SSortOrderSet** <br/> |
   
```cpp
CbNewSSortOrderSet (_csort)
```

## <a name="parameters"></a>Parâmetros

 __csort_
  
> Contagem de estruturas de **SSortOrder** a serem incluídos na estrutura de **SSortOrderSet** .
