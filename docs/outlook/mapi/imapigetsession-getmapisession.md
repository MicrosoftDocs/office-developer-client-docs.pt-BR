# <a name="imapigetsessiongetmapisession"></a>IMAPIGetSession::GetMAPISession

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Retorna um ponteiro para a sessão MAPI associada ao objeto de suporte MAPI.
  
```cpp
HRESULT GetMAPISession(
  LPUNKNOWN *  lppSession
);
```

## <a name="parameters"></a>Parâmetros

 _lppSession_
  
> bota Um ponteiro para a sessão MAPI atual.
