# <a name="pidlidcontactuserfield4-canonical-property"></a>Propriedade canônica PidLidContactUserField4

  
  
**Aplica-se a**: Outlook 
  
Especifica o quarto campo no contato que destina-se ao uso de diversos para o contato.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidContactUserField4  <br/> |
|Propriedade definida:  <br/> |PSETID_Address  <br/> |
|ID de longo (LID):  <br/> |0x00008052  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Contato  <br/> |
   
## <a name="remarks"></a>Comentários

Essa propriedade contém o texto que não está relacionado a qualquer outra propriedade específicos para contatos.
