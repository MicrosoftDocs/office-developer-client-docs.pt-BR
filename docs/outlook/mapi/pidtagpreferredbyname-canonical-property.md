# <a name="pidtagpreferredbyname-canonical-property"></a>Propriedade canônica PidTagPreferredByName

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém o nome que o destinatário prefere que seja chamado.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_PREFERRED_BY_NAME, PR_PREFERRED_BY_NAME_A, PR_PREFERRED_BY_NAME_W  <br/> |
|Identificador:  <br/> |0x3A47  <br/> |
|Tipo de dados:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Área:  <br/> |Usuário de email MAPI  <br/> |
