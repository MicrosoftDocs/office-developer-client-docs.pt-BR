# <a name="pidlidyomicompanyname-canonical-property"></a>Propriedade canônica PidLidYomiCompanyName

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Especifica a pronúncia fonética do nome da empresa do contato.
