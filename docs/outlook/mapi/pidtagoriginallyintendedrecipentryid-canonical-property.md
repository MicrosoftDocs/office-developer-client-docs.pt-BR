# <a name="pidtagoriginallyintendedrecipentryid-canonical-property"></a>Propriedade canônica PidTagOriginallyIntendedRecipEntryId

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém o identificador de entrada do destinatário originalmente pretendido de uma mensagem encaminhada automaticamente.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_ORIGINALLY_INTENDED_RECIP_ENTRYID  <br/> |
|Identificador:  <br/> |0x1012  <br/> |
|Tipo de dados:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Servidor  <br/> |
   
## <a name="remarks"></a>Comentários

Essa propriedade é uma das propriedades de endereço do destinatário da mensagem originalmente pretendido. Ela deve ser definida pelo agente automático que encaminhou a mensagem.
  
Essa propriedade corresponde ao atributo de relatório X.400 por destinatário.
