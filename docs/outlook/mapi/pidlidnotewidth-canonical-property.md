# <a name="pidlidnotewidth-canonical-property"></a>Propriedade canônica PidLidNoteWidth

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Especifica a largura da janela de mensagem visível em pixels. 
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidNoteWidth  <br/> |
|Conjunto de propriedades:  <br/> |PSETID_Note  <br/> |
|Long ID (LID):  <br/> |0x00008B02  <br/> |
|Tipo de dados:  <br/> |PT_LONG  <br/> |
|Área:  <br/> |Observação Sobre o uso de sticky  <br/> |
   
## <a name="remarks"></a>Comentários

Esse valor deve ser maior do que zero.
