# <a name="pidlidnotewidth-canonical-property"></a>Propriedade canônica PidLidNoteWidth

  
  
**Aplica-se a**: Outlook 
  
Especifica a largura da janela de mensagem visíveis em pixels. 
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidNoteWidth  <br/> |
|Propriedade definida:  <br/> |PSETID_Note  <br/> |
|ID de longo (LID):  <br/> |0x00008B02  <br/> |
|Tipo de dados:  <br/> |PT_LONG  <br/> |
|Área:  <br/> |Nota  <br/> |
   
## <a name="remarks"></a>Comentários

Este valor deve ser maior do que zero.
