# <a name="pidnamekeywords-canonical-property"></a>Propriedade canônica PidNameKeywords

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém as palavras-chave ou categorias para o objeto de mensagem.
  
|||
|:-----|:-----|
|Nomes amigáveis:  <br/> |Nenhum  <br/> |
|Propriedade definida:  <br/> |PS_PUBLIC_STRINGS  <br/> |
|Nome da propriedade:  <br/> |Keywords  <br/> |
|Tipo de dados:  <br/> |PT_MV_UNICODE  <br/> |
|Área:  <br/> |Soluções gerais de mensagens  <br/> |
   
## <a name="remarks"></a>Comentários

Um valor de cadeia de caracteres múltipla que especifica as categorias para um objeto de mensagem, o tamanho de cada string dentro essa cadeia de caracteres de valores múltiplos de propriedade deve ser menor que 256.
