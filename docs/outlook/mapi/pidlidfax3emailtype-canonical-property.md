# <a name="pidlidfax3emailtype-canonical-property"></a>Propriedade canônica PidLidFax3EmailType

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Especifica o tipo de email do outro endereço de fax do contato.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidFax3EmailType  <br/> |
|Conjunto de propriedades:  <br/> |PSETID_Address  <br/> |
|Long ID (LID):  <br/> |0x000080D7  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Contato  <br/> |
