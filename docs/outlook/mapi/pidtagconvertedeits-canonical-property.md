# <a name="pidtagconvertedeits-canonical-property"></a>Propriedade canônica PidTagConvertedEits

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém um identificador para os tipos de texto em uma mensagem após a conversão.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_CONVERTED_EITS  <br/> |
|Identificador:  <br/> |0x000E  <br/> |
|Tipo de dados:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Exchange  <br/> |
   
## <a name="remarks"></a>Comentários

Essa propriedade indica quais tipos de informações codificadas (EITs) foram usados para converter os componentes de texto da mensagem.
