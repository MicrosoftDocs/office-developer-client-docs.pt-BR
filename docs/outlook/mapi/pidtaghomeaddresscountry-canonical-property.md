# <a name="pidtaghomeaddresscountry-canonical-property"></a>Propriedade canônica PidTagHomeAddressCountry

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém o distrito no endereço de um contato.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_HOME_ADDRESS_COUNTRY, PR_HOME_ADDRESS_COUNTRY_A, PR_HOME_ADDRESS_COUNTRY_W  <br/> |
|Identificador:  <br/> |0x3A5A  <br/> |
|Tipo de dados:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Área:  <br/> |Usuário de email MAPI  <br/> |
   
## <a name="remarks"></a>Comentários

Essas propriedades são definidas pelo usuário ou pela organização do usuário.
