# <a name="pidlidreminderplaysound-canonical-property"></a>Propriedade canônica PidLidReminderPlaySound

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Especifica se o cliente deve tocar um som quando o lembrete se tornar vencido.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidReminderPlaySound  <br/> |
|Conjunto de propriedades:  <br/> |PSETID_Common  <br/> |
|Long ID (LID):  <br/> |0x0000851E  <br/> |
|Tipo de dados:  <br/> |PT_BOOLEAN  <br/> |
|Área:  <br/> |Reminder  <br/> |
