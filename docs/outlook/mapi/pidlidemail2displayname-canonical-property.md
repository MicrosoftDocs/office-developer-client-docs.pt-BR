# <a name="pidlidemail2displayname-canonical-property"></a>Propriedade canônica PidLidEmail2DisplayName

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Especifica o nome de exibição legíveis de usuário para o segundo endereço de email.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidEmail2DisplayName  <br/> |
|Propriedade definida:  <br/> |PSETID_Address  <br/> |
|ID de longo (LID):  <br/> |0x00008090  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Contato  <br/> |
