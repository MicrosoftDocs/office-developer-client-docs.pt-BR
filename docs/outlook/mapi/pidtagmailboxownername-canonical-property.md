# <a name="pidtagmailboxownername-canonical-property"></a>Propriedade canônica PidTagMailboxOwnerName

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Exibe o nome do proprietário da caixa de correio.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_MAILBOX_OWNER_NAME, PR_MAILBOX_OWNER_NAME_A, PR_MAILBOX_OWNER_NAME_W  <br/> |
|Identificador:  <br/> |0x001F  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE, PT_UNICODE  <br/> |
|Área:  <br/> |Armazenamento de Mensagens  <br/> |
