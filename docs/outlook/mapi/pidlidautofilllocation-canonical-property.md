# <a name="pidlidautofilllocation-canonical-property"></a>Propriedade canônica PidLidAutoFillLocation

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Indica que o valor da propriedade **dispidLocation** ([PidLidLocation](pidlidlocation-canonical-property.md)) é definido para a propriedade **PR_DISPLAY_NAME** ([PidTagDisplayName](pidtagdisplayname-canonical-property.md)) do RecipientRow que representa um recurso.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidAutoFillLocation  <br/> |
|Propriedade definida:  <br/> |PSETID_Appointment  <br/> |
|ID de longo (LID):  <br/> |0x0000823A  <br/> |
|Tipo de dados:  <br/> |PT_BOOLEAN  <br/> |
|Área:  <br/> |Reuniões  <br/> |
   
## <a name="remarks"></a>Comentários

Para obter mais detalhes sobre RecipientRow, consulte o protocolo de mensagem e o objeto Attachment conforme especificado na [[MS-OXCMSG]](http://msdn.microsoft.com/library/7fd7ec40-deec-4c06-9493-1bc06b349682%28Office.15%29.aspx).
