# <a name="pidtagnonreceiptreason-canonical-property"></a>Propriedade canônica PidTagNonReceiptReason

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém os motivos pelos quais uma mensagem não foi recebida e faz parte de um relatório de não entrega.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_NON_RECEIPT_REASON  <br/> |
|Identificador:  <br/> |0x003E  <br/> |
|Tipo de dados:  <br/> |PT_LONG  <br/> |
|Área:  <br/> |Exchange  <br/> |
   
## <a name="remarks"></a>Comentários

Os motivos podem incluir, por exemplo, que a mensagem foi descartada.
