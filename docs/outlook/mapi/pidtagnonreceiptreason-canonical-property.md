# <a name="pidtagnonreceiptreason-canonical-property"></a>Propriedade canônica PidTagNonReceiptReason

  
  
**Aplica-se a**: Outlook 
  
Contém os motivos por que não foi recebida uma mensagem que faz parte de um relatório de falha na entrega.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_NON_RECEIPT_REASON  <br/> |
|Identificador:  <br/> |0x003E  <br/> |
|Tipo de dados:  <br/> |PT_LONG  <br/> |
|Área:  <br/> |Exchange  <br/> |
   
## <a name="remarks"></a>Comentários

Motivos podem incluir, por exemplo, a que a mensagem foi descartada.
