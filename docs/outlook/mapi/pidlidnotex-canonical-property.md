# <a name="pidlidnotex-canonical-property"></a>Propriedade canônica PidLidNoteX

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Especifica a distância, em pixels, da borda esquerda da tela onde uma interface do usuário exibe uma nota.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidNoteX  <br/> |
|Propriedade definida:  <br/> |PSETID_Note  <br/> |
|ID de longo (LID):  <br/> |0x00008B04  <br/> |
|Tipo de dados:  <br/> |PT_LONG  <br/> |
|Área:  <br/> |Nota  <br/> |
