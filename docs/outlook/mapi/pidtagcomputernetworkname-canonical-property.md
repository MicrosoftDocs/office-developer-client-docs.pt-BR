# <a name="pidtagcomputernetworkname-canonical-property"></a>Propriedade canônica PidTagComputerNetworkName

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém o nome da rede usado para transmitir a mensagem.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_COMPUTER_NETWORK_NAME, PR_COMPUTER_NETWORK_NAME_A, PR_COMPUTER_NETWORK_NAME_W  <br/> |
|Identificador:  <br/> |0x3A49  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Área:  <br/> |Contato  <br/> |
