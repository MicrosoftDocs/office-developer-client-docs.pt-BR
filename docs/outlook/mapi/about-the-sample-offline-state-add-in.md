# <a name="about-the-sample-offline-state-add-in"></a>Sobre o amostra Offline State suplemento

  
  
**Aplica-se a**: Outlook 
  
A API de estado Offline oferece suporte a chamadas de retorno indicando alterações no estado de conexão de um usuário no Outlook — por exemplo, de sendo online no Outlook para sendo offline. O suplemento de amostra Offline estado é um suplemento de COM escritos em C++ que demonstra como receber notificações de alterações de estado de conexão e como modificar o estado atual usando o API de estado Offline. Para obter mais informações sobre a API de estado Offline, consulte [Sobre o Offline estado API](about-the-offline-state-api.md).
