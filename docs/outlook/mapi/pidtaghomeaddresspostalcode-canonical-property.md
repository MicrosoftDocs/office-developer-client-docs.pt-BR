# <a name="pidtaghomeaddresspostalcode-canonical-property"></a>Propriedade canônica PidTagHomeAddressPostalCode

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém o código postal do endereço residencial do usuário.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_HOME_ADDRESS_POSTAL_CODE, PR_HOME_ADDRESS_POSTAL_CODE_A, PR_HOME_ADDRESS_POSTAL_CODE_W  <br/> |
|Identificador:  <br/> |0x3A5B  <br/> |
|Tipo de dados:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Área:  <br/> |Usuário de email MAPI  <br/> |
   
## <a name="remarks"></a>Comentários

Essas propriedades são definidas pelo usuário ou a organização do usuário.
