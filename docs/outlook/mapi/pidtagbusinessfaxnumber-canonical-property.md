# <a name="pidtagbusinessfaxnumber-canonical-property"></a>Propriedade canônica PidTagBusinessFaxNumber

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém o número de telefone do aparelho de fax comercial do destinatário. 
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_BUSINESS_FAX_NUMBER, PR_BUSINESS_FAX_NUMBER_A, PR_BUSINESS_FAX_NUMBER_W  <br/> |
|Identificador:  <br/> |0x3A24  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Área:  <br/> |Contato  <br/> |
   
## <a name="remarks"></a>Comentários

Essas propriedades são exemplos das propriedades que fornece informações de identificação e acesso sobre um destinatário. Essas propriedades são definidas pelo destinatário e pela organização do destinatário.
