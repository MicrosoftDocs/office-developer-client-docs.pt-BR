# <a name="pidtagexceptionreplacetime-canonical-property"></a>Propriedade canônica PidTagExceptionReplaceTime

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Indica a data e a hora originais em que a instância no padrão de recorrência teria ocorrido se não fosse uma exceção.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_EXCEPTION_REPLACETIME  <br/> |
|Identificador:  <br/> |0x7FF9  <br/> |
|Tipo de dados:  <br/> |PT_SYSTIME  <br/> |
|Área:  <br/> |Mensagem não transmitível definida pela classe  <br/> |
   
## <a name="remarks"></a>Comentários

Esse valor deve ser especificado em UTC (Tempo Universal Coordenado).
