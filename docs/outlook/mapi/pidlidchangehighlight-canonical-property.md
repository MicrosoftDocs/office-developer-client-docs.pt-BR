# <a name="pidlidchangehighlight-canonical-property"></a>Propriedade canônica PidLidChangeHighlight

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Especifica um campo de bits que indica como o objeto de reunião foi alterado.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidChangeHighlight  <br/> |
|Conjunto de propriedades:  <br/> |PSETID_Appointment  <br/> |
|Long ID (LID):  <br/> |0x00008204  <br/> |
|Tipo de dados:  <br/> |PT_LONG  <br/> |
|Área:  <br/> |Reuniões  <br/> |
   
## <a name="remarks"></a>Comentários

Essa propriedade não é necessária. Os sinalizadores individuais que podem ser definidos são detalhados [em [MS-OXOCAL]](https://msdn.microsoft.com/library/09861fde-c8e4-4028-9346-e7c214cfdba1%28Office.15%29.aspx).
