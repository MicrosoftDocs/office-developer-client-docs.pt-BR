# <a name="pidlidchangehighlight-canonical-property"></a>Propriedade canônica PidLidChangeHighlight

  
  
**Aplica-se a**: Outlook 
  
Especifica um campo de bit que indica como o objeto de reunião é alterada.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidChangeHighlight  <br/> |
|Propriedade definida:  <br/> |PSETID_Appointment  <br/> |
|ID de longo (LID):  <br/> |0x00008204  <br/> |
|Tipo de dados:  <br/> |PT_LONG  <br/> |
|Área:  <br/> |Reuniões  <br/> |
   
## <a name="remarks"></a>Comentários

Essa propriedade não é necessária. Os sinalizadores individuais que podem ser definidos são detalhados na [[MS-OXOCAL]](http://msdn.microsoft.com/library/09861fde-c8e4-4028-9346-e7c214cfdba1%28Office.15%29.aspx).
