# <a name="pidtagrecipientdisplayname-canonical-property"></a>Propriedade canônica PidTagRecipientDisplayName

  
  
**Aplica-se a**: Outlook 
  
Contém o valor da propriedade **PR_DISPLAY_NAME** ([PidTagDisplayName](pidtagdisplayname-canonical-property.md)).
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_RECIPIENT_DISPLAY_NAME, PR_RECIPIENT_DISPLAY_NAME_A, PR_RECIPIENT_DISPLAY_NAME_W  <br/> |
|Identificador:  <br/> |0x5FF6  <br/> |
|Tipo de dados:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Área:  <br/> |Destinatário de transporte  <br/> |
