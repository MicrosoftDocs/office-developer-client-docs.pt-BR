# <a name="pidtagrpcoverhttpproxyprincipalname-canonical-property"></a>Propriedade canônica PidTagRpcOverHttpProxyPrincipalName

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém o nome principal do servidor proxy usado pelo Microsoft Outlook para se conectar ao Microsoft Exchange Server usando uma chamada de procedimento remoto (RPC) sobre Protocolo de Transferência de Hipertexto (HTTP).
  
## 

|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_ROH_PROXY_PRINCIPAL_NAME  <br/> |
|Identificador:  <br/> |0x6625  <br/> |
|Tipo de propriedade:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Diversos  <br/> |
