<span data-ttu-id="9b19b-110">Os provedores de serviços geralmente são escritos para sistemas de mensagens específicos por desenvolvedores de software que têm conhecimento especializado ou experiência com um sistema específico.</span><span class="sxs-lookup"><span data-stu-id="9b19b-110">Service providers are typically written for specific messaging systems by software developers who have specialized knowledge or experience with a particular system.</span></span> <span data-ttu-id="9b19b-111">For example, the Microsoft Outlook 2013 and Microsoft Outlook 2010 Mobile Services use an address book provider to expose a mobile address book in Outlook.</span><span class="sxs-lookup"><span data-stu-id="9b19b-111">For example, the Microsoft Outlook 2013 and Microsoft Outlook 2010 Mobile Services use an address book provider to expose a mobile address book in Outlook.</span></span>
