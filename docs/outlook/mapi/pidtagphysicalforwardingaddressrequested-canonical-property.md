# <a name="pidtagphysicalforwardingaddressrequested-canonical-property"></a>Propriedade canônica PidTagPhysicalForwardingAddressRequested

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém TRUE se um remetente de mensagem solicitar que o agente de transferência de mensagem anexe um endereço de encaminhamento físico para um destinatário de mensagem.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_PHYSICAL_FORWARDING_ADDRESS_REQUESTED  <br/> |
|Identificador:  <br/> |0x0C0E  <br/> |
|Tipo de dados:  <br/> |PT_BOOLEAN  <br/> |
|Área:  <br/> |Destinatário MAPI  <br/> |
