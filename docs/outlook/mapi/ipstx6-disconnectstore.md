# <a name="ipstx6disconnectstore"></a>IPSTX6::DisconnectStore

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Desconecta o armazenamento local.
  
```cpp
HRESULT DisconnectStore( 
    ULONG ulFlags 
);
```

## <a name="parameters"></a>Parâmetros

 _ulFlags_
  
>  [in] Sinalizadores para modificar o comportamento. Este parâmetro deve sempre ser definido como 0.
