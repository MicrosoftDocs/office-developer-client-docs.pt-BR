# <a name="operating-environment-issues"></a>Problemas operacionais no ambiente

  
  
**Aplica-se a**: Outlook 
  
Os problemas do sistema operacional e o ambiente operacional são importantes quando os desenvolvedores MAPI escolher um ambiente de desenvolvimento e um ambiente para a instalação do SDK do Windows.
  
Os desenvolvedores MAPI devem estar cientes de que essas opções afetam o que seus componentes serão executados no sistema operacional e o intervalo dos componentes que serão capazes de destino.
