# <a name="pidlidlogdocumentprinted-canonical-property"></a>Propriedade canônica PidLidLogDocumentPrinted

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Indica se o documento foi impresso durante o registro no diário.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidLogDocPrinted  <br/> |
|Conjunto de propriedades:  <br/> |PSETID_Log  <br/> |
|Long ID (LID):  <br/> |0x0000870E  <br/> |
|Tipo de dados:  <br/> |PT_BOOLEAN  <br/> |
|Área:  <br/> |Diário  <br/> |
