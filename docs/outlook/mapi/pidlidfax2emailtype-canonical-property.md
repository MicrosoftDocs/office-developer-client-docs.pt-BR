# <a name="pidlidfax2emailtype-canonical-property"></a>Propriedade canônica PidLidFax2EmailType

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Especifica o tipo de email de endereço de fax residencial do contato.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidFax2EmailType  <br/> |
|Propriedade definida:  <br/> |PSETID_Address  <br/> |
|ID de longo (LID):  <br/> |0x000080C7  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Contato  <br/> |
