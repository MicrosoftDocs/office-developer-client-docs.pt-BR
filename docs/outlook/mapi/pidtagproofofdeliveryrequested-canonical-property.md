# <a name="pidtagproofofdeliveryrequested-canonical-property"></a>Propriedade canônica PidTagProofOfDeliveryRequested

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Esta propriedade conterá TRUE se um remetente de mensagem solicitar prova de entrega para um destinatário específico.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_PROOF_OF_DELIVERY_REQUESTED  <br/> |
|Identificador:  <br/> |0x0C12  <br/> |
|Tipo de dados:  <br/> |PT_BOOLEAN  <br/> |
|Área:  <br/> |Destinatário MAPI  <br/> |
