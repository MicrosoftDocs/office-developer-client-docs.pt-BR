# <a name="pidtagviewdescriptorversion-canonical-property"></a>Propriedade canônica PidTagViewDescriptorVersion

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Indica a versão de um descritor de modo de exibição.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_VD_VERSION  <br/> |
|Identificador:  <br/> |0x7007  <br/> |
|Tipo de dados:  <br/> |PT_LONG  <br/> |
|Área:  <br/> |Diversos  <br/> |
   
## <a name="remarks"></a>Comentários

O valor desta propriedade deve ser 8.
