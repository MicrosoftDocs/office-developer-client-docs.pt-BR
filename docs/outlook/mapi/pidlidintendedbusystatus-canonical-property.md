# <a name="pidlidintendedbusystatus-canonical-property"></a>Propriedade canônica PidLidIntendedBusyStatus

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Especifica o valor da propriedade **dispidBusyStatus** ([PidLidBusyStatus](pidlidbusystatus-canonical-property.md)) na reunião no calendário do organizador quando a solicitação de reunião ou a atualização da reunião foi enviada.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidIntendedBusyStatus  <br/> |
|Conjunto de propriedades:  <br/> |PSETID_Appointment  <br/> |
|Long ID (LID):  <br/> |0x00008224  <br/> |
|Tipo de dados:  <br/> |PT_LONG  <br/> |
|Área:  <br/> |Meetings  <br/> |
   
## <a name="remarks"></a>Comentários

Os valores permitidos dessa propriedade são os mesmos da propriedade **dispidBusyStatus**.
