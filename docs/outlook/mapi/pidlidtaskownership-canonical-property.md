# <a name="pidlidtaskownership-canonical-property"></a>Propriedade canônica PidLidTaskOwnership

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Indica a função do usuário atual em relação a tarefa.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidTaskOwnership  <br/> |
|Propriedade definida:  <br/> |PSETID_Task  <br/> |
|ID de longo (LID):  <br/> |0x00008129  <br/> |
|Tipo de dados:  <br/> |PT_LONG  <br/> |
|Área:  <br/> |Task  <br/> |
   
## <a name="remarks"></a>Comentários

Esta propriedade deve ser um dos seguintes valores.
  
|**Valor**|**Descrição**|
|:-----|:-----|
|0x00000000  <br/> |A tarefa não é atribuída.  <br/> |
|0x00000001  <br/> |A tarefa é uma cópia do cedente a tarefa da tarefa.  <br/> |
|0x00000002  <br/> |A tarefa é uma cópia do destinatário da tarefa da tarefa.  <br/> |
