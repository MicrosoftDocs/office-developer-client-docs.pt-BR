# <a name="pidtagotheraddresscountry-canonical-property"></a>Propriedade canônica PidTagOtherAddressCountry

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém o email outro país/região do usuário.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_OTHER_ADDRESS_COUNTRY, PR_OTHER_ADDRESS_COUNTRY_A, PR_OTHER_ADDRESS_COUNTRY_W  <br/> |
|Identificador:  <br/> |0x3A60  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Área:  <br/> |Catálogo de endereços  <br/> |
   
## <a name="remarks"></a>Comentários

Essas propriedades são definidas pelo usuário ou a organização do usuário.
