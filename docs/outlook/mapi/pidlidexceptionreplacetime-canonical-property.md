# <a name="pidlidexceptionreplacetime-canonical-property"></a>Propriedade canônica PidLidExceptionReplaceTime

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Especifica a data e a hora dentro do padrão de recorrência que a exceção substituirá.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidExceptionReplaceTime  <br/> |
|Conjunto de propriedades:  <br/> |PSETID_Appointment  <br/> |
|Long ID (LID):  <br/> |0x00008228  <br/> |
|Tipo de dados:  <br/> |PT_SYSTIME  <br/> |
|Área:  <br/> |Calendário  <br/> |
   
## <a name="remarks"></a>Comentários

O valor deve ser especificado em UTC (Tempo Universal Coordenado). Essa propriedade permite que o objeto de anexo de exceção seja encontrado para uma instância específica.
