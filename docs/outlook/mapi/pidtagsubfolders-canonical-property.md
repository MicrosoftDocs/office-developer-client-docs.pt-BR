# <a name="pidtagsubfolders-canonical-property"></a>Propriedade canônica PidTagSubfolders

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Conterá TRUE se uma pasta contiver subpastas.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_SUBFOLDERS  <br/> |
|Identificador:  <br/> |0x360A  <br/> |
|Tipo de dados:  <br/> |PT_BOOLEAN  <br/> |
|Área:  <br/> |Contêiner MAPI  <br/> |
   
## <a name="remarks"></a>Comentários

Repositórios de mensagem deverá fornecer essa propriedade para todas as pastas.
