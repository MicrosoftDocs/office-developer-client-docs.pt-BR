# <a name="pidtagcompanyname-canonical-property"></a>Propriedade canônica PidTagCompanyName

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém o nome da empresa do destinatário. 
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_COMPANY_NAME, PR_COMPANY_NAME_A, PR_COMPANY_NAME_W  <br/> |
|Identificador:  <br/> |0x3A16  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Área:  <br/> |Contato  <br/> |
   
## <a name="remarks"></a>Comentários

Essas propriedades fornecem informações de identificação e acesso sobre um destinatário. Eles são definidos pelo destinatário e pela organização do destinatário.
