# <a name="pidtagreportdispositionmode-canonical-property"></a>Propriedade canônica PidTagReportDispositionMode

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Indica a disposição do recibo para mensagens que solicitam recibos. 
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_REPORT_DISPOSITION_MODE, PR_REPORT_DISPOSITION_MODE_A, PR_REPORT_DISPOSITION_MODE_W  <br/> |
|Identificador:  <br/> |0x0081  <br/> |
|Tipo de dados:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Área:  <br/> |Envelope MAPI  <br/> |
   
## <a name="remarks"></a>Comentários

Os valores possíveis para essa propriedade são "manual-action/MDN-sent-automatically" e "manual-action/MDN-sent-manually".
