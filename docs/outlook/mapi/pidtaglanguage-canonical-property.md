# <a name="pidtaglanguage-canonical-property"></a>Propriedade canônica PidTagLanguage

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém um valor que indica o idioma no qual o usuário de mensagens está escrevendo mensagens.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_LANGUAGE, PR_LANGUAGE_A, PR_LANGUAGE_W  <br/> |
|Identificador:  <br/> |0x3A0C  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Área:  <br/> |Endereço  <br/> |
   
## <a name="remarks"></a>Comentários

A cadeia de caracteres contém um único código de país/região de dois caracteres.
