# <a name="pidtagotheraddressstateorprovince-canonical-property"></a>Propriedade canônica PidTagOtherAddressStateOrProvince

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém o nome do estado ou província usado em outro endereço.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_OTHER_ADDRESS_STATE_OR_PROVINCE, PR_OTHER_ADDRESS_STATE_OR_PROVINCE_A, PR_OTHER_ADDRESS_STATE_OR_PROVINCE_W  <br/> |
|Identificador:  <br/> |0x3A62  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Área:  <br/> |Endereço  <br/> |
   
## <a name="remarks"></a>Comentários

Essas propriedades são definidas pelo usuário ou a organização do usuário.
