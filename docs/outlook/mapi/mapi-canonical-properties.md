[PidLidContacts](pidlidcontacts-canonical-property.md) represents the **dispidContacts** property, a named property that has a numerical identifier and that specifies the name of contacts associated with a message; and [PidNamePhishingStamp](pidnamephishingstamp-canonical-property.md) represents "https://schemas.microsoft.com/outlook/phishingstamp," a named property that has a string name, and that specifies the string marking messages that are likely to be phishing.</span></span>

### <a name="identifying-properties-in-mapi"></a><span data-ttu-id="092bd-113">Identificar propriedades MAPI</span><span class="sxs-lookup"><span data-stu-id="092bd-113">Identifying Properties in MAPI</span></span>

<span data-ttu-id="092bd-114">Todas as propriedades na MAPI são identificadas por um identificador de propriedade que é um valor de 16 bits não assinado.</span><span class="sxs-lookup"><span data-stu-id="092bd-114">All properties in MAPI are identified by a property identifier that is an unsigned 16-bit value.</span></span> <span data-ttu-id="092bd-115">O identificador de propriedade e o tipo de propriedade (outro valor de 16 bits não assinado) constituem uma marca de propriedade para a propriedade.</span><span class="sxs-lookup"><span data-stu-id="092bd-115">The property identifier and the property type (another unsigned 16-bit value) constitute a property tag for the property.</span></span> 
  
<span data-ttu-id="092bd-116">MAPI usa uma marca de propriedade para definir propriedades de exclusivamente.</span><span class="sxs-lookup"><span data-stu-id="092bd-116">MAPI uses a property tag to uniquely define properties.</span></span> <span data-ttu-id="092bd-117">Propriedades que têm a mesma propriedade marca, como **PR_BUSINESS2_TELEPHONE_NUMBER** ([PidTagBusiness2TelephoneNumber](pidtagbusiness2telephonenumber-canonical-property.md)) e **PR_OFFICE2_TELEPHONE_NUMBER** ([PidTagBusiness2TelephoneNumber](pidtagbusiness2telephonenumber-canonical-property.md)), são consideradas idênticas Propriedades no MAPI.</span><span class="sxs-lookup"><span data-stu-id="092bd-117">Properties that have the same property tag, like **PR_BUSINESS2_TELEPHONE_NUMBER** ([PidTagBusiness2TelephoneNumber](pidtagbusiness2telephonenumber-canonical-property.md)) and **PR_OFFICE2_TELEPHONE_NUMBER** ([PidTagBusiness2TelephoneNumber](pidtagbusiness2telephonenumber-canonical-property.md)), are considered identical properties in MAPI.</span></span> <span data-ttu-id="092bd-118">Para obter uma lista das marcas de propriedade MAPI tenha definido para suas próprias propriedades, consulte o arquivo de cabeçalho MAPI, Mapitags.h.</span><span class="sxs-lookup"><span data-stu-id="092bd-118">For a list of property tags that MAPI has defined for its own properties, see the MAPI header file, Mapitags.h.</span></span>
  
<span data-ttu-id="092bd-119">Observe que o MAPI divide identificadores de propriedade em intervalos.</span><span class="sxs-lookup"><span data-stu-id="092bd-119">Note that MAPI divides property identifiers into ranges.</span></span> <span data-ttu-id="092bd-120">Onde um identificador cai no intervalo indica seu uso e a propriedade.</span><span class="sxs-lookup"><span data-stu-id="092bd-120">Where an identifier falls in the range indicates its use and ownership.</span></span> <span data-ttu-id="092bd-121">Os identificadores de propriedade das propriedades marcados se encaixam no intervalo de 0x0001 para 0x7FFF.</span><span class="sxs-lookup"><span data-stu-id="092bd-121">The property identifiers of tagged properties fall in the range of 0x0001 to 0x7FFF.</span></span> <span data-ttu-id="092bd-122">Dentro desse intervalo são os identificadores de propriedade das propriedades definidas pelo MAPI, que se encaixam no intervalo de 0x0001 para 0x3FFF.</span><span class="sxs-lookup"><span data-stu-id="092bd-122">Within this range are the property identifiers of MAPI-defined properties, which fall in the range of 0x0001 to 0x3FFF.</span></span> <span data-ttu-id="092bd-123">Os identificadores de propriedade de queda de propriedades nomeadas no intervalo de 0x8000 para 0x8FFF.</span><span class="sxs-lookup"><span data-stu-id="092bd-123">The property identifiers of named properties fall in the range from 0x8000 to 0x8FFF.</span></span> 
  
<span data-ttu-id="092bd-124">Além disso, propriedades nomeadas são atribuídas por um conjunto de propriedades e um longo ID (tampa), que é um valor de 32 bits não assinado, ou um string.</span><span class="sxs-lookup"><span data-stu-id="092bd-124">Named properties are additionally attributed by a property set, and either a long ID (LID), which is an unsigned 32-bit value, or a string.</span></span> <span data-ttu-id="092bd-125">Um conjunto de propriedades é um GUID que identifica um grupo de propriedades nomeadas com uma finalidade similar.</span><span class="sxs-lookup"><span data-stu-id="092bd-125">A property set is a GUID that identifies a group of named properties with a similar purpose.</span></span> <span data-ttu-id="092bd-126">O conjunto de propriedades e o nome LID ou cadeia de caracteres são usados para obter ou definir a propriedade nomeada.</span><span class="sxs-lookup"><span data-stu-id="092bd-126">The property set and LID or string name are used to get or set the named property.</span></span>
