# <a name="pidnametemplate-canonical-property"></a>Propriedade canônica PidNameTemplate

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Especifica o modelo do arquivo anexado ao documento.
  
|||
|:-----|:-----|
|Nomes amigáveis:  <br/> |Nenhum  <br/> |
|Conjunto de propriedades:  <br/> |PS_PUBLIC_STRINGS  <br/> |
|Nome da propriedade:  <br/> |Modelo  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Comum  <br/> |
