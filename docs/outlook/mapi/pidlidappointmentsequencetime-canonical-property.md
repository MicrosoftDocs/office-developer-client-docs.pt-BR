# <a name="pidlidappointmentsequencetime-canonical-property"></a>Propriedade canônica PidLidAppointmentSequenceTime

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Especifica a data e hora em que essa propriedade da última modificação.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidApptSeqTime  <br/> |
|Propriedade definida:  <br/> |PSETID_Appointment  <br/> |
|ID de longo (LID):  <br/> |0x00008202  <br/> |
|Tipo de dados:  <br/> |PT_SYSTIME  <br/> |
|Área:  <br/> |Reuniões  <br/> |
   
## <a name="remarks"></a>Comentários

O valor deve ser especificado no tempo Universal Coordenado (UTC).
