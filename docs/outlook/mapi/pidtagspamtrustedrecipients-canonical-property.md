# <a name="pidtagspamtrustedrecipients-canonical-property"></a>Propriedade canônica PidTagSpamTrustedRecipients
 
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém uma lista delimitada por ponto-e-vírgula de endereços de email e domínios que representam os destinatários confiáveis.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_SPAM_TRUSTED_RECIPIENTS_W  <br/> |
|Long ID (LID):  <br/> |0x0419  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Spam  <br/> |
