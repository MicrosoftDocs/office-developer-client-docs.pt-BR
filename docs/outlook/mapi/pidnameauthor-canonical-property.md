# <a name="pidnameauthor-canonical-property"></a>Propriedade canônica PidNameAuthor

  
  
**Aplica-se a**: Outlook 
  
Representa o autor do arquivo que está anexado ao documento.
  
|||
|:-----|:-----|
|Nomes amigáveis:  <br/> |Nenhum  <br/> |
|Propriedade definida:  <br/> |PS_PUBLIC_STRINGS  <br/> |
|Nome da propriedade:  <br/> |Autor  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Comuns  <br/> |
