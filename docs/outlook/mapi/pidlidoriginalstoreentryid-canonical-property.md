# <a name="pidlidoriginalstoreentryid-canonical-property"></a>Propriedade canônica PidLidOriginalStoreEntryId

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Especifica a identificação de entrada do repositório do representante.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidOrigStoreEid  <br/> |
|Propriedade definida:  <br/> |PSETID_Appointment  <br/> |
|ID de longo (LID):  <br/> |0x00008237  <br/> |
|Tipo de dados:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Reuniões  <br/> |
   
## <a name="remarks"></a>Comentários

Essa propriedade deve ser definida em objetos de reunião que foram criados ou atualizados por um representante.
