# <a name="pidtagreportoriginalsender-canonical-property"></a>Propriedade canônica PidTagReportOriginalSender

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Indica o remetente original da mensagem para mensagens que solicitam recibos.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_REPORT_ORIGINAL_SENDER, PR_REPORT_ORIGINAL_SENDER_A, PR_REPORT_ORIGINAL_SENDER_W  <br/> |
|Identificador:  <br/> |0x0082  <br/> |
|Tipo de dados:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Área:  <br/> |Envelope MAPI  <br/> |
