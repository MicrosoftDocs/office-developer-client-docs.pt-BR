# <a name="pidtagassistanttelephonenumber-canonical-property"></a>Propriedade canônica PidTagAssistantTelephoneNumber

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém o número de telefone do assistente administrativo do destinatário. 
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_ASSISTANT_TELEPHONE_NUMBER, PR_ASSISTANT_TELEPHONE_NUMBER_A, PR_ASSISTANT_TELEPHONE_NUMBER_W  <br/> |
|Identificador:  <br/> |0x3A2E  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Área:  <br/> |Endereço  <br/> |
   
## <a name="remarks"></a>Comentários

Essas propriedades fornecem informações de identificação e acesso para um destinatário. Eles são definidos pelo destinatário e sua organização. 
  
O número de telefone é para o assistente especificado na **propriedade PR_ASSISTANT** ([PidTagAssistant](pidtagassistant-canonical-property.md)).
