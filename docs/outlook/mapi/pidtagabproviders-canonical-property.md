# <a name="pidtagabproviders-canonical-property"></a>Propriedade canônica PidTagAbProviders

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém uma lista de identificadores para provedores de listas de endereços no perfil atual. 
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_AB_PROVIDERS  <br/> |
|Identificador:  <br/> |0x3D01  <br/> |
|Tipo de dados:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Catálogo de endereços  <br/> |
   
## <a name="remarks"></a>Comentários

Não use essa propriedade. Ele é reservado para uso pelo MAPI.
