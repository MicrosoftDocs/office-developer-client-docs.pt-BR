# <a name="pidlidemail1originalentryid-canonical-property"></a>Propriedade canônica PidLidEmail1OriginalEntryId

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Especifica a **EntryId** do objeto que corresponde ao primeiro endereço de email. 
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidEmail1OriginalEntryID  <br/> |
|Propriedade definida:  <br/> |PSETID_Address  <br/> |
|ID de longo (LID):  <br/> |0x00008085  <br/> |
|Tipo de dados:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Contato  <br/> |
   
## <a name="remarks"></a>Comentários

O valor dessa propriedade deve ser em um únicos **EntryId** para esse endereço eletrônico ou um objeto válido do catálogo de endereços **EntryId**.
