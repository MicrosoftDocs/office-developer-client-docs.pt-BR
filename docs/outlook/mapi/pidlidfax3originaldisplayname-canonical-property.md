# <a name="pidlidfax3originaldisplayname-canonical-property"></a>Propriedade canônica PidLidFax3OriginalDisplayName

  
  
**Aplica-se a**: Outlook 
  
Especifica o que outro endereço de fax do nome original para exibição do contato.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidFax3OriginalDisplayName  <br/> |
|Propriedade definida:  <br/> |PSETID_Address  <br/> |
|ID de longo (LID):  <br/> |0x000080D4  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Contato  <br/> |
   
## <a name="remarks"></a>Comentários

Essa propriedade, se houver, deve ser definida para o mesmo valor que a propriedade **PR_NORMALIZED_SUBJECT** ([PidTagNormalizedSubject](pidtagnormalizedsubject-canonical-property.md)).
