# <a name="pidlidfax3originaldisplayname-canonical-property"></a>Propriedade canônica PidLidFax3OriginalDisplayName

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Especifica o nome de exibição original do outro endereço de fax do contato.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidFax3OriginalDisplayName  <br/> |
|Conjunto de propriedades:  <br/> |PSETID_Address  <br/> |
|Long ID (LID):  <br/> |0x000080D4  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Contato  <br/> |
   
## <a name="remarks"></a>Comentários

Essa propriedade, se presente, deve ser definida com o mesmo valor que a propriedade **PR_NORMALIZED_SUBJECT** ([PidTagNormalizedSubject](pidtagnormalizedsubject-canonical-property.md)).
