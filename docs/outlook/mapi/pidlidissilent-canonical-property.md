# <a name="pidlidissilent-canonical-property"></a>Propriedade canônica PidLidIsSilent

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Indica que o usuário não incluiu nenhum texto no corpo da resposta da reunião.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |LID_IS_SILENT  <br/> |
|Conjunto de propriedades:  <br/> |PSETID_Meeting  <br/> |
|Long ID (LID):  <br/> |0x00000004  <br/> |
|Tipo de dados:  <br/> |PT_BOOLEAN  <br/> |
|Área:  <br/> |Reuniões  <br/> |
