# <a name="mapi-forms"></a>Formulários MAPI

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Após ler esta visão geral da arquitetura do formulário MAPI, você terá uma compreensão de quais são os formulários MAPI e como eles interagem com outros componentes do subsistema de MAPI. A finalidade desta seção é dar a você o conhecimento conceitual que você precisa implementar seus próprios servidores de formulário MAPI.
  
Antes de ler esta seção, familiarize-se com o material no tópico [Visão geral dos formulários de MAPI](mapi-forms-overview.md) . 
  
> [!NOTE]
> Como a arquitetura do MAPI formulário se baseia no modelo COM (Component Object), desenvolvimento de aplicativos de servidor do formulário requer conhecimento de programação COM. Para obter mais informações sobre o COM, consulte a seção de serviços de objetos ActiveX e COM no SDK do Windows.
