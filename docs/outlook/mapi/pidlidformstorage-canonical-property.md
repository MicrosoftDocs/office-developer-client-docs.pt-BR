# <a name="pidlidformstorage-canonical-property"></a>Propriedade canônica PidLidFormStorage

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Representa as propriedades do formulário que está associado com a mensagem.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidFormStorage  <br/> |
|Propriedade definida:  <br/> |PSETID_Common  <br/> |
|ID de longo (LID):  <br/> |0x0000850F  <br/> |
|Tipo de dados:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Configuração de tempo de execução  <br/> |
