# <a name="pidlidtaskstatusoncomplete-canonical-property"></a>Propriedade canônica PidLidTaskStatusOnComplete

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Indica se o destinatário da tarefa foi solicitado a enviar uma atualização de mensagem de email ao concluir a tarefa atribuída.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidTaskSOC  <br/> |
|Conjunto de propriedades:  <br/> |PSETID_Task  <br/> |
|Long ID (LID):  <br/> |0x00008119  <br/> |
|Tipo de dados:  <br/> |PT_BOOLEAN  <br/> |
|Área:  <br/> |Tarefa  <br/> |
