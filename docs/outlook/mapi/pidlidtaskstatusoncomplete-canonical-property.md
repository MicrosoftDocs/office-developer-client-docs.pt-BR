# <a name="pidlidtaskstatusoncomplete-canonical-property"></a>Propriedade canônica PidLidTaskStatusOnComplete

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Indica se o destinatário da tarefa tiver sido solicitado para enviar uma atualização de mensagem de email quando eles concluam a tarefa atribuída.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidTaskSOC  <br/> |
|Propriedade definida:  <br/> |PSETID_Task  <br/> |
|ID de longo (LID):  <br/> |0x00008119  <br/> |
|Tipo de dados:  <br/> |PT_BOOLEAN  <br/> |
|Área:  <br/> |Task  <br/> |
