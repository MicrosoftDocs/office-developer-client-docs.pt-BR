# <a name="pidtagcompanymaintelephonenumber-canonical-property"></a>Propriedade canônica PidTagCompanyMainTelephoneNumber

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém o número de telefone principal de uma empresa
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_COMPANY_MAIN_PHONE_NUMBER, PR_COMPANY_MAIN_PHONE_NUMBER_A, PR_COMPANY_MAIN_PHONE_NUMBER_W  <br/> |
|Identificador:  <br/> |0x3A57  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Área:  <br/> |Contato  <br/> |
   
## <a name="remarks"></a>Comentários

Essas propriedades são definidas pelo usuário ou pela organização do usuário.
