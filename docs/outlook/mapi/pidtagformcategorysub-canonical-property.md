# <a name="pidtagformcategorysub-canonical-property"></a>Propriedade canônica PidTagFormCategorySub

  
  
**Aplica-se a**: Outlook 
  
Contém a subcategoria de um formulário, conforme definido por um aplicativo cliente. 
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_FORM_CATEGORY_SUB, PR_FORM_CATEGORY_SUB_A, PR_FORM_CATEGORY_SUB_W  <br/> |
|Identificador:  <br/> |0x3305  <br/> |
|Tipo de dados:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Área:  <br/> |MAPI comuns  <br/> |
   
## <a name="remarks"></a>Comentários

Essas propriedades são subordinadas à categoria formulário principal que é fornecida na propriedade **PR_FORM_CATEGORY** ([PidTagFormCategory](pidtagformcategory-canonical-property.md)).
