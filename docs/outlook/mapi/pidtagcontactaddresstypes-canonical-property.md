# <a name="pidtagcontactaddresstypes-canonical-property"></a>Propriedade canônica PidTagContactAddressTypes

  
  
**Aplica-se a**: Outlook 
  
Contém uma lista de tipos de endereço de um contato.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_CONTACT_ADDRTYPES, PR_CONTACT_ADDRTYPES_A, PR_CONTACT_ADDRTYPES_W  <br/> |
|Identificador:  <br/> |0x3A54  <br/> |
|Tipo de dados:  <br/> |PT_MV_UNICODE, PT_MV_STRING8  <br/> |
|Área:  <br/> |Contato  <br/> |
