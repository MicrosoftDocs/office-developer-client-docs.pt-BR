# <a name="pidtagsearchfolderid-canonical-property"></a>Propriedade canônica PidTagSearchFolderId

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém um identificador global exclusivo (GUID) que identifica a pasta de pesquisa.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_WB_SF_ID  <br/> |
|Identificador:  <br/> |0x6842  <br/> |
|Tipo de dados:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Pesquisa  <br/> |
