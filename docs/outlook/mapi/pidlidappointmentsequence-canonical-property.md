# <a name="pidlidappointmentsequence-canonical-property"></a>Propriedade canônica PidLidAppointmentSequence

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Especifica o número de sequência de um objeto Meeting .
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidApptSequence  <br/> |
|Conjunto de propriedades:  <br/> |PSETID_Appointment  <br/> |
|Long ID (LID):  <br/> |0x00008201  <br/> |
|Tipo de dados:  <br/> |PT_LONG  <br/> |
|Área:  <br/> |Reuniões  <br/> |
