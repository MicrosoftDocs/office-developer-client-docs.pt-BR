# <a name="pidlidappointmentsequence-canonical-property"></a>Propriedade canônica PidLidAppointmentSequence

  
  
**Aplica-se a**: Outlook 
  
Especifica o número de sequência de um objeto de reunião.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidApptSequence  <br/> |
|Propriedade definida:  <br/> |PSETID_Appointment  <br/> |
|ID de longo (LID):  <br/> |0x00008201  <br/> |
|Tipo de dados:  <br/> |PT_LONG  <br/> |
|Área:  <br/> |Reuniões  <br/> |
