# <a name="pidtagbusinesshomepage-canonical-property"></a>Propriedade canônica PidTagBusinessHomePage

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém a URL da home page do negócio.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_BUSINESS_HOME_PAGE, PR_BUSINESS_HOME_PAGE_A, PR_BUSINESS_HOME_PAGE_W  <br/> |
|Identificador:  <br/> |0x3A51  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Área:  <br/> |Contato  <br/> |
   
## <a name="remarks"></a>Comentários

Essas propriedades são usadas como um link para a página de home comercial do contato.
