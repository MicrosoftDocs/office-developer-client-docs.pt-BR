# <a name="pidnameedittime-canonical-property"></a>Propriedade canônica PidNameEditTime

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Especifica a hora em que o arquivo foi editado pela última vez.
  
|||
|:-----|:-----|
|Nomes amigáveis:  <br/> |Nenhum  <br/> |
|Conjunto de propriedades:  <br/> |PS_PUBLIC_STRINGS  <br/> |
|Nome da propriedade:  <br/> |EditTime  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Comum  <br/> |
