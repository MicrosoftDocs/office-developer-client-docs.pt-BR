# <a name="pidtagpersonalhomepage-canonical-property"></a>Propriedade canônica PidTagPersonalHomePage

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém a URL da home page pessoal do usuário.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_PERSONAL_HOME_PAGE, PR_PERSONAL_HOME_PAGE_A, PR_PERSONAL_HOME_PAGE_W  <br/> |
|Identificador:  <br/> |0x3A50  <br/> |
|Tipo de dados:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Área:  <br/> |Usuário de email MAPI  <br/> |
   
## <a name="remarks"></a>Comentários

Essas propriedades são definidas pelo usuário ou a organização do usuário.
