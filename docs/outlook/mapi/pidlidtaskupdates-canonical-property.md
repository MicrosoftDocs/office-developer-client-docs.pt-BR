# <a name="pidlidtaskupdates-canonical-property"></a>Propriedade canônica PidLidTaskUpdates

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Indica se o destinatário da tarefa foi solicitado a enviar uma atualização de tarefa quando a tarefa atribuída for alterada.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidTaskUpdates  <br/> |
|Conjunto de propriedades:  <br/> |PSETID_Task  <br/> |
|Long ID (LID):  <br/> |0x0000811B  <br/> |
|Tipo de dados:  <br/> |PT_BOOLEAN  <br/> |
|Área:  <br/> |Tarefa  <br/> |
