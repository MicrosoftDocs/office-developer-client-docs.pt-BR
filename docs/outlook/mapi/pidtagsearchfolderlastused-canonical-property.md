# <a name="pidtagsearchfolderlastused-canonical-property"></a>Propriedade canônica PidTagSearchFolderLastUsed

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Representa a última vez em que a pasta foi acessada.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_WB_SF_LAST_USED  <br/> |
|Identificador:  <br/> |0x6834  <br/> |
|Tipo de dados:  <br/> |PT_LONG  <br/> |
|Área:  <br/> |Pesquisar  <br/> |
   
## <a name="remarks"></a>Comentários

Esta propriedade deve ser formatada como o número de minutos desde a meia-noite Tempo Universal Coordenado (UTC) em 1 de janeiro de 1601.
