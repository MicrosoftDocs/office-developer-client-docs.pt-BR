# <a name="pidtagviewdescriptorname-canonical-property"></a>Propriedade canônica PidTagViewDescriptorName

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém o nome de um descritor de modo de exibição.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_VD_NAME, PR_VD_NAME_A, PR_VD_NAME_W  <br/> |
|Identificador:  <br/> |0x7006  <br/> |
|Tipo de dados:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Área:  <br/> |Mensagem definida de classe transmittable  <br/> |
   
## <a name="remarks"></a>Comentários

Essas propriedades devem ser definidas para uma cadeia de caracteres não vazio para uma mensagem de informações de associar pasta (FAI) que contém definições de exibição.
