# <a name="pidlidglobalobjectid-canonical-property"></a>Propriedade canônica PidLidGlobalObjectId

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Especifica o identificador exclusivo do objeto calendar.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |LID_GLOBAL_OBJID  <br/> |
|Propriedade definida:  <br/> |PSETID_Meeting  <br/> |
|ID de longo (LID):  <br/> |0x00000003  <br/> |
|Tipo de dados:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Reuniões  <br/> |
   
## <a name="remarks"></a>Comentários

Uma vez definida para um objeto de calendário, o valor dessa propriedade não deve alterar. Uma descrição detalhada do formato pode ser encontrada no [[MS-OXOCAL]](http://msdn.microsoft.com/library/9b636532-9150-4836-9635-9c9b756c9ccf%28Office.15%29.aspx).
