# <a name="pidlidfax1emailaddress-canonical-property"></a>Propriedade canônica PidLidFax1EmailAddress

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Especifica o endereço de email de fax de negócios do contato.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidFax1EmailAddress  <br/> |
|Propriedade definida:  <br/> |PSETID_Address  <br/> |
|ID de longo (LID):  <br/> |0x000080B3  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Contato  <br/> |
   
## <a name="remarks"></a>Comentários

Essa propriedade, se houver, deve conter um nome de exibição legíveis de usuário, seguido de "@" caractere, seguido por um número de fax.
