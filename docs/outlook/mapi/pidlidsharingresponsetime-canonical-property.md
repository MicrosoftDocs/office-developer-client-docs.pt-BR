# <a name="pidlidsharingresponsetime-canonical-property"></a>Propriedade canônica PidLidSharingResponseTime

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Especifica a hora em que o destinatário da solicitação de compartilhamento enviou uma resposta de compartilhamento. Esta é uma propriedade de uma mensagem de compartilhamento.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidSharingResponseTime  <br/> |
|Conjunto de propriedades:  <br/> |PSETID_Sharing  <br/> |
|Long ID (LID):  <br/> |0x00008A28  <br/> |
|Tipo de dados:  <br/> |PT_SYSTIME  <br/> |
|Área:  <br/> |Compartilhamento  <br/> |
