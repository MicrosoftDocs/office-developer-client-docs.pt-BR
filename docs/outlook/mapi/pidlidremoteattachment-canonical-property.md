# <a name="pidlidremoteattachment-canonical-property"></a>Propriedade canônica PidLidRemoteAttachment

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Indica se um item remoto tem um anexo associado a ele.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidRemoteAttachment  <br/> |
|Propriedade definida:  <br/> |PSETID_Remote  <br/> |
|ID de longo (LID):  <br/> |0x00008F07  <br/> |
|Tipo de dados:  <br/> |PT_BOOLEAN  <br/> |
|Área:  <br/> |Mensagem remota  <br/> |
