# <a name="pidlidworkaddresscountry-canonical-property"></a>Propriedade canônica PidLidWorkAddressCountry

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Especifica a parte do país ou região do endereço de trabalho do contato.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidWorkAddressCountry  <br/> |
|Conjunto de propriedades:  <br/> |PSETID_Address  <br/> |
|Long ID (LID):  <br/> |0x00008049  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Contato  <br/> |
