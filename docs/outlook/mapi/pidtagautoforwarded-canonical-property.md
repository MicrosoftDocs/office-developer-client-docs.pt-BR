# <a name="pidtagautoforwarded-canonical-property"></a>Propriedade canônica PidTagAutoForwarded

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém TRUE se o cliente solicitar um campo de cabeçalho X-MS-Exchange-Organization-autoForwarded.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_AUTO_FORWARDED  <br/> |
|Identificador:  <br/> |0x0005  <br/> |
|Tipo de dados:  <br/> |PT_BOOLEAN  <br/> |
|Área:  <br/> |Relatórios gerais  <br/> |
   
## <a name="remarks"></a>Comentários

Se essa propriedade for definida como FALSE ou não usada, nenhum campo de cabeçalho X-MS-Exchange-Organization-autoForwarded será criado.
