# <a name="pidlidpostrsschannel-canonical-property"></a>Propriedade canônica PidLidPostRssChannel

  
  
**Aplica-se a**: Outlook 
  
Contém o conteúdo da \<título\> campo do XML a Atom \<feed\> ou RSS \<canal\>.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidPostRssChannel  <br/> |
|Propriedade definida:  <br/> |PSETID_PostRss  <br/> |
|ID de longo (LID):  <br/> |0x00008904  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |RSS  <br/> |
