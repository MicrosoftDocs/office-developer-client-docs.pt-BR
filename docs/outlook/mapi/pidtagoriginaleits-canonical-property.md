# <a name="pidtagoriginaleits-canonical-property"></a>Propriedade canônica PidTagOriginalEits

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém uma cópia dos tipos de informações codificadas originais (EITs) para o texto da mensagem.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_ORIGINAL_EITS  <br/> |
|Identificador:  <br/> |0x0021  <br/> |
|Tipo de dados:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Servidor  <br/> |
