# <a name="pidlidfax3addresstype-canonical-property"></a>Propriedade canônica PidLidFax3AddressType

  
  
**Aplica-se a**: Outlook 
  
Especifica o tipo de endereço para o outro endereço do contato fax.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidFax3AddrType  <br/> |
|Propriedade definida:  <br/> |PSETID_Address  <br/> |
|ID de longo (LID):  <br/> |0x000080D2  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Contato  <br/> |
   
## <a name="remarks"></a>Comentários

Essa propriedade, se houver, deve ser definida como "FAX".
