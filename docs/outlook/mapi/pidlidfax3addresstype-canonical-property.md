# <a name="pidlidfax3addresstype-canonical-property"></a>Propriedade canônica PidLidFax3AddressType

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Especifica o tipo de endereço para o endereço de fax do outro contato.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidFax3AddrType  <br/> |
|Conjunto de propriedades:  <br/> |PSETID_Address  <br/> |
|Long ID (LID):  <br/> |0x000080D2  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Contato  <br/> |
   
## <a name="remarks"></a>Comentários

Essa propriedade, se presente, deve ser definida como "FAX".
