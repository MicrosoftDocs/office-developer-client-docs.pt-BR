# <a name="pidlidfax3emailaddress-canonical-property"></a>Propriedade canônica PidLidFax3EmailAddress

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Especifica o endereço de email do outro endereço de fax do contato.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidFax3EmailAddress  <br/> |
|Conjunto de propriedades:  <br/> |PSETID_Address  <br/> |
|Long ID (LID):  <br/> |0x000080D3  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Contato  <br/> |
   
## <a name="remarks"></a>Comentários

Essa propriedade, se presente, deve conter um nome de exibição legível pelo usuário, seguido pelo caractere "@", seguido por um número de fax.
