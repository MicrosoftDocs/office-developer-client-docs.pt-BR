# <a name="pidlidfax3emailaddress-canonical-property"></a>Propriedade canônica PidLidFax3EmailAddress

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Especifica o que outro endereço de fax do endereço de email do contato.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidFax3EmailAddress  <br/> |
|Propriedade definida:  <br/> |PSETID_Address  <br/> |
|ID de longo (LID):  <br/> |0x000080D3  <br/> |
|Tipo de dados:  <br/> |PT_UNICODE  <br/> |
|Área:  <br/> |Contato  <br/> |
   
## <a name="remarks"></a>Comentários

Essa propriedade, se houver, deve conter um nome de exibição legíveis de usuário, seguido de "@" caractere, seguido por um número de fax.
