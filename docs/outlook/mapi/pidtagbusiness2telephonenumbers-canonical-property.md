# <a name="pidtagbusiness2telephonenumbers-canonical-property"></a>Propriedade canônica PidTagBusiness2TelephoneNumbers

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém uma lista de números de telefone para uma empresa.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_BUSINESS2_TELEPHONE_NUMBER_A_MV  <br/> |
|Identificador:  <br/> |0x3A1B  <br/> |
|Tipo de dados:  <br/> |PT_MV_STRING8  <br/> |
|Área:  <br/> |Contato  <br/> |
   
## <a name="remarks"></a>Comentários

Esta propriedade pode ser usada para listar os números de telefone alternativos, ou uma lista de extensões para uma empresa.
