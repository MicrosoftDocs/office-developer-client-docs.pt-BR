# <a name="pidtaginternetmessageid-canonical-property"></a>Propriedade canônica PidTagInternetMessageId

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Corresponde ao campo de ID de mensagem conforme especificado em [RFC2822].
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_INTERNET_MESSAGE_ID, PR_INTERNET_MESSAGE_ID_A, PR_INTERNET_MESSAGE_ID_W  <br/> |
|Identificador:  <br/> |0x1035  <br/> |
|Tipo de dados:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Área:  <br/> |MIME  <br/> |
   
## <a name="remarks"></a>Comentários

Essas propriedades devem estar presentes em todas as mensagens de email.
