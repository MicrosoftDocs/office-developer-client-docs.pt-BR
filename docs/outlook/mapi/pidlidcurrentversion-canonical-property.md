# <a name="pidlidcurrentversion-canonical-property"></a>Propriedade canônica PidLidCurrentVersion

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém um long que representa o número de complicação do aplicativo Outlook para um item.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidCurrentVersion  <br/> |
|Conjunto de propriedades:  <br/> |PSETID_Common  <br/> |
|Long ID (LID):  <br/> |0x00008552  <br/> |
|Tipo de dados:  <br/> |PT_LONG  <br/> |
|Área:  <br/> |Envio de mensagens gerais  <br/> |
