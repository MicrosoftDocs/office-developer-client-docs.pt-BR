# <a name="pidtagdisclosureofrecipients-canonical-property"></a>Propriedade canônica PidTagDisclosureOfRecipients

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém TRUE se a divulgação de destinatários é permitida.
