# <a name="pidtagoriginatoranddistributionlistexpansionhistory-canonical-property"></a>Propriedade canônica PidTagOriginatorAndDistributionListExpansionHistory

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Contém informações sobre um originador de mensagens e um histórico de expansão da lista de distribuição.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |PR_ORIGINATOR_AND_DL_EXPANSION_HISTORY  <br/> |
|Identificador:  <br/> |0x1002  <br/> |
|Tipo de dados:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |MIME  <br/> |
   
## <a name="remarks"></a>Comentários

Essa propriedade é usada na geração de relatório.
