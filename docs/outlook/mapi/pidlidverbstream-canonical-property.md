# <a name="pidlidverbstream-canonical-property"></a>Propriedade canônica PidLidVerbStream

  
  
**Aplica-se a**: Outlook 2013 | Outlook 2016 
  
Especifica quais respostas de votação o usuário pode fazer em resposta à mensagem.
  
|||
|:-----|:-----|
|Propriedades associadas:  <br/> |dispidVerbStream  <br/> |
|Conjunto de propriedades:  <br/> |PSETID_Common  <br/> |
|Long ID (LID):  <br/> |0x00008520  <br/> |
|Tipo de dados:  <br/> |PT_BINARY  <br/> |
|Área:  <br/> |Configuração de tempo de execução  <br/> |
