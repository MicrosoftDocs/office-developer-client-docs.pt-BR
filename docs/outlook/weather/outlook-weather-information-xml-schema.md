# <a name="outlook-weather-information-xml-schema"></a>Esquema XML de informações sobre o clima do Outlook

Esta seção contém a referência de esquema do Outlook 2013 para o esquema XML do Outlook clima informações. Este esquema permite que os provedores de serviço de meteorologia enviar informações de previsão do tempo para um local especificado como uma resposta do serviço web para o Outlook, assim que o Outlook pode exibir as informações na barra de meteorologia.
