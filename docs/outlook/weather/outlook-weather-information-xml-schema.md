# <a name="outlook-weather-information-xml-schema"></a>Esquema XML de informações meteorológicas do Outlook

Esta seção contém a referência de esquema do Outlook 2013 para o esquema XML de informações sobre o clima do Outlook. Esse esquema permite que os provedores de serviços de previsão do tempo enviem informações de previsão do tempo para um local especificado como uma resposta de serviço Web para o Outlook, para que o Outlook possa exibir as informações na Barra do Tempo.
