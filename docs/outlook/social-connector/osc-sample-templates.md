# <a name="osc-sample-templates"></a>Modelos de exemplo do OSC

Vários modelos de provedores em C++, C# e Visual Basic acompanham a Referência do Provedor do Outlook Social Connector 2013. É possível baixar os modelos em [Outlook Social Connector 2013: modelos de provedor](https://code.msdn.microsoft.com/Outlook-Social-Connector-73fd8d2c).
  
Os modelos do provedor estão disponíveis para uso quando você cria um provedor na linguagem de sua escolha. Para diminuir o tamanho do download e o tempo de carregamento do provedor, crie um provedor de produção usando C++. Se você estiver desenvolvendo um provedor para usuários que têm o .NET Framework 3.5 SP1 ou uma versão posterior do .NET Framework instalada no computador, escreva também um provedor gerenciado usando C# ou Visual Basic.
