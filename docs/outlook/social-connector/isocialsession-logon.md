# <a name="isocialsessionlogon"></a>ISocialSession::Logon

Logs para o site de rede social usando o nome de usuário especificado e a senha.
  
```cpp
HRESULT _stdcall Logon([in] BSTR username, [in] BSTR password);
```

## <a name="parameters"></a>Parâmetros

_nome de usuário_
  
> [in] Uma cadeia de caracteres que contém o nome de usuário para fazer logon.
    
_senha_
  
> [in] Uma cadeia de caracteres que contém a senha para fazer logon.
