# <a name="osc-provider-xml-examples"></a>Exemplos XML de provedor do OSC

Esta seção contém exemplos XML que usam os elementos XML **feed de atividades** , **amigos**e **recursos**, em conformidade com o esquema XML de provedor do Outlook Social Connector (OSC). 
  
- [Exemplo de XML de recursos](capabilities-xml-example.md) mostra os **recursos de** XML de um provedor OSC, especificando seus recursos e requisitos para que o OSC. 
    
- [Exemplo de XML de amigos](friends-xml-example.md) mostra **amigos** XML para as duas pessoas na rede social. 
    
- [Exemplo de XML de Feed de atividade](activity-feed-xml-example.md) mostra o **feed de atividades** XML de quatro atividades: dois perfis atualizações de imagem, uma atualização de status e uma postagem de blog. O XML também especifica os três modelos de exibição de atividades para exibir as atividades correspondentes.
