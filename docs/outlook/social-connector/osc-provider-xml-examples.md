# <a name="osc-provider-xml-examples"></a>Exemplos de XML do provedor OSC

Esta seção contém exemplos de XML que usam os recursos **,** amigos e elementos XML **activityFeed,** em conformidade com o esquema XML do provedor Outlook Social Connector (OSC). 
  
- [O exemplo xml de](capabilities-xml-example.md) recursos **mostra os** recursos XML de um provedor osC, especificando seus recursos e requisitos para o OSC. 
    
- [O exemplo XML de](friends-xml-example.md) amigos mostra **o** XML de amigos de duas pessoas na rede social. 
    
- [O Exemplo XML](activity-feed-xml-example.md) do Feed de Atividades mostra o XML **activityFeed** de quatro atividades: duas atualizações de imagem de perfil, uma atualização de status e uma postagem de blog. O XML também especifica três modelos de exibição de atividade para exibir as atividades correspondentes.
