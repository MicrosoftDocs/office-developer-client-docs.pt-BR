# <a name="recipients"></a>Destinatários

Esta seção fornece tarefas de exemplo que envolvem destinatários. Os destinatários são usuários que recebem mensagens de email de saída, solicitações de compartilhamento ou solicitações de reunião.

|[Usar a caixa de diálogo Selecionar Nomes para receber e atribuir destinatários a um compromisso](how-to-use-the-select-names-dialog-box-to-obtain-and-assign-recipients-to-an-appointment.md)  |Usa a caixa de diálogo **Selecionar Nomes** para receber e atribuir destinatários a um item de compromisso.|
