# <a name="sessions"></a>Sessões

Esta seção inclui uma tarefa de amostra que pertence às sessões do Microsoft Outlook. Uma sessão é um período de tempo durante o qual um usuário faz logon no Outlook. Uma sessão é representada no Outlook pela classe [NameSpace](https://msdn.microsoft.com/library/bb645857\(v=office.15\)). Essa classe oferece suporte para fazer logon e logoff, acessar objetos de armazenamento diretamente pela identificação, acessar determinadas pastas padrão especiais diretamente e acessar origens de dados pertencentes a outros usuários. A única fonte de dados com suporte é MAPI, que permite acesso a todos os dados do Outlook armazenados no email do usuário.
