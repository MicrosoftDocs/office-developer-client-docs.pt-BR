# <a name="conversations"></a>Conversas

Esta seção fornece exemplos de tarefas que envolvem conversas. As conversas no Outlook são representadas pelo objeto [Conversation](https://msdn.microsoft.com/library/ff184711\(v=office.15\)) e incluem um ou mais itens em uma pasta que compartilham o mesmo título do assunto ou thread.
