# <a name="address-book"></a>Catálogo de endereços

Esta seção fornece exemplos de tarefas que envolvem catálogos de endereços. Os catálogos de endereços são usados para fornecer um sistema de endereçamento para os usuários se comunicarem eletronicamente. Os exemplos de provedores de catálogos de endereços incluem a Lista de Endereços Global (GAL) do Exchange, o Catálogo de Endereços do Outlook, contêineres do Exchange e o catálogo de endereços offline (OAB).
