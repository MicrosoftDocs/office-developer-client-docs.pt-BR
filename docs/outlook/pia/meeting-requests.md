# <a name="meeting-requests"></a>Solicitações de reunião

Esta seção fornece tarefas de amostra que envolvem solicitações de reunião, que são solicitações para adicionar compromissos ao calendário de um destinatário.

## <a name="in-this-section"></a>Nesta seção

|Tópico|Descrição|
|:----|:----------|
|[Criar uma solicitação de reunião, adicionar destinatários e especificar um local](how-to-create-a-meeting-request-add-recipients-and-specify-a-location.md)  |Cria um item de compromisso como uma solicitação de reunião, especifica a hora, os destinatários e local da reunião e exibe o compromisso em um inspetor.|
|[Obter o organizador de uma reunião](how-to-get-the-organizer-of-a-meeting.md)  |Programaticamente retorna o organizador de uma reunião.|
|[Verificar todas as respostas à solicitação de reunião](how-to-check-all-responses-to-a-meeting-request.md)  |Verifica o status da resposta de cada destinatário para uma solicitação de reunião.|
|[Localizar o item de compromisso associado a uma solicitação de reunião](how-to-find-the-appointment-item-associated-with-a-meeting-request.md)  |Usa o método [GetAssociatedAppointment(Boolean)](https://msdn.microsoft.com/library/bb652725\(v=office.15\)) para localizar o compromisso associado a uma solicitação de reunião.|
|[Aceitar automaticamente uma solicitação de reunião](how-to-automatically-accept-a-meeting-request.md)  |Usa o método [Respond(OlMeetingResponse, Object, Object)](https://msdn.microsoft.com/library/bb647086\(v=office.15\)) para automaticamente aceitar uma solicitação de reunião.|
|[Solicitar que um usuário responda a uma solicitação de reunião](how-to-prompt-a-user-to-respond-to-a-meeting-request.md)  |Solicita ao usuário uma resposta à solicitação de reunião e permite que ele edite a resposta antes de enviá-la.|
