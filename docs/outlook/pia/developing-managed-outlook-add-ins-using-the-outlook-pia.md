# <a name="developing-managed-outlook-add-ins-using-the-outlook-pia"></a>Desenvolvimento de suplementos gerenciados do Outlook usando o Outlook PIA

Para ver informações mais básicas sobre como gravar aplicativos gerenciados para o Office usando o PIA, confira [Desenvolvimento do Office no Visual Studio](https://docs.microsoft.com/visualstudio/vsto/office-and-sharepoint-development-in-visual-studio?view=vs-2017).
