# <a name="contacts"></a>Contatos

Esta seção fornece exemplos de tópicos que envolvem contatos. Contatos representam pessoas e empresas com as quais você deseja se comunicar. A pasta Contatos é seu catálogo de endereços de email e armazenamento de informações para os contatos. Um objeto [ContactItem](https://msdn.microsoft.com/library/bb644956\(v=office.15\)) é um objeto interno do Outlook que representa contatos exclusivos em uma pasta Contatos. O objeto **ContactItem** tem mais de 100 propriedades, como [FirstName](https://msdn.microsoft.com/library/bb652965\(v=office.15\)), [CompanyName](https://msdn.microsoft.com/library/bb610212\(v=office.15\)) e [OfficeLocation](https://msdn.microsoft.com/library/bb647145\(v=office.15\)), bem como propriedades personalizadas se as propriedades internas não forem suficientes, que você pode usar para armazenar informações sobre cada contato.
