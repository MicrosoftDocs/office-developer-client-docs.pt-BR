# <a name="interfaces-account-management-api"></a>Interfaces (API de gerenciamento de contas)

Esta seção descreve as interfaces na API de gerenciamento de conta.
  
|**Interface**|**Descrição**|
|:-----|:-----|
|[IOlkAccount](iolkaccount.md) <br/> |Oferece suporte para obter e definir propriedades e outras informações sobre uma conta.  <br/> |
|[IOlkAccountHelper](iolkaccounthelper.md) <br/> |Fornece funcionalidade de ajuda na sessão MAPI atual para gerenciar contas.  <br/> |
|[IOlkAccountManager](iolkaccountmanager.md) <br/> |Gerencia o acesso a contas e configura notificações sobre alterações de conta.  <br/> |
|[IOlkAccountNotify](iolkaccountnotify.md) <br/> |Fornece um retorno de chamada para o cliente para alterações em uma conta.  <br/> |
|[IOlkEnum](iolkenum.md) <br/> |Oferece suporte à enumeração de contas como objetos [IUnknown](https://docs.microsoft.com/windows/desktop/api/unknwn/nn-unknwn-iunknown) .  <br/> |
|[IOlkErrorUnknown](iolkerrorunknown.md) <br/> |Fornece informações extras sobre o último erro.  <br/> |
