# <a name="ifreebusydataplaceholder2"></a>IFreeBusyData::Placeholder2

Esse membro é um espaço reservado e não tem suporte.
  
## <a name="quick-info"></a>Informações rápidas

Confira [IFreeBusyData](ifreebusydata.md).
  
```cpp
HRESULT Placeholder2(void *);
```

## <a name="return-values"></a>Valor de retorno

Sempre E_NOTIMPL, pois esse membro é um espaço reservado no **IFreeBusyData** e não é suportado. 
  
## <a name="remarks"></a>Comentários

Este membro deve retornar E_NOTIMPL.
