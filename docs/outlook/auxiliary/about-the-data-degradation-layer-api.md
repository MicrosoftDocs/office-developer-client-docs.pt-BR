# <a name="about-the-data-degradation-layer-api"></a>Sobre a API de camada de degradação de dados

A API da camada de degradação de dados fornece a capacidade de ocultar o formato de caractere nativo subjacente de um objeto do Outlook, permitindo que um cliente acesse o objeto em um formato de caractere preferido, e não no formato de caractere nativo do objeto. Ele fornece as definições de constantes mostradas em [constantes (API de camada de degradação de dados)](constants-data-degradation-layer-api.md)e a função [HrCreateNewWrappedObject](hrcreatenewwrappedobject.md) que retorna um objeto do Outlook disposto em um formato especificado.
