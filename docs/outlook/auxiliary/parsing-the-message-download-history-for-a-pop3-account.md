<span data-ttu-id="551ca-107">O provedor de protocolo POP (Post Office) para o Outlook permite aos usuários recuperarem e baixarem novas mensagens de email em seu dispositivo local e posteriormente class="sxs-lookup"><span data-stu-id="551ca-109">The mail client does this by first using the UIDL (Unique ID Listing) command to obtain a map of each message that has ever been delivered to that
