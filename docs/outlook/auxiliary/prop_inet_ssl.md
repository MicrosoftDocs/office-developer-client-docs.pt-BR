# <a name="propinetssl"></a>PROP_INET_SSL

Especifica se deve usar a camada de soquete seguro (SSL) para uma caixa de correio de Internet geral.
  
## <a name="quick-info"></a>Informações rápidas

|||
|:-----|:-----|
|Identificador:  <br/> |0x0105  <br/> |
|Tipo de propriedade:  <br/> |PT_DWORD  <br/> |
|Marca de propriedade:  <br/> |0x01050003  <br/> |
|Access:  <br/> |Somente leitura  <br/> |
   
## <a name="remarks"></a>Coment�rios

Um meio de valor zero não usar SSL, caso contrário, use SSL.
