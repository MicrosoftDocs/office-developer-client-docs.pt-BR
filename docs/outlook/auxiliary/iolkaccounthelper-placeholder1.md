# <a name="iolkaccounthelperplaceholder1"></a>IOlkAccountHelper::Placeholder1

Esse membro é um espaço reservado e não tem suporte. Ele sempre deve retornar E_NOTIMPL.
  
## <a name="quick-info"></a>Informações rápidas

```cpp
HRESULT IOlkAccountHelper::Placeholder1 (  
    void* 
);
```

## <a name="return-values"></a>Valor de retorno

Sempre E_NOTIMPL, pois esse membro é um espaço reservado em [IOlkAccountHelper](iolkaccounthelper.md) e não é suportado. 
  
## <a name="remarks"></a>Comentários

Este membro deve sempre retornar E_NOTIMPL.
