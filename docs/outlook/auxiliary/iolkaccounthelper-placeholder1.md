# <a name="iolkaccounthelperplaceholder1"></a>IOlkAccountHelper::Placeholder1

Este membro é um espaço reservado e não é suportado. Ela sempre deve retornar E_NOTIMPL.
  
## <a name="quick-info"></a>Informações rápidas

```cpp
HRESULT IOlkAccountHelper::Placeholder1 (  
    void* 
);
```

## <a name="return-values"></a>Valores de retorno

Sempre E_NOTIMPL, pois este membro é um espaço reservado no [IOlkAccountHelper](iolkaccounthelper.md) e não é suportado. 
  
## <a name="remarks"></a>Comentários

Este membro sempre deve retornar E_NOTIMPL.
