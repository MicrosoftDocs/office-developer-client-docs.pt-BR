Representa o número da porta para uma caixa de correio geral na Internet.

|Identificador:  <br/> |0x0104  <br/> |
|Tipo de propriedade:  <br/> |PT_DWORD  <br/> |
|Marca de propriedade:  <br/> |0x01040003  <br/> |
|Access:  <br/> |Somente leitura  <br/> |
