# <a name="data-degradation-layer-api-reference"></a>Referência da API da camada de degradação de dados

Esta seção fornece conteúdo de referência para a API da camada de degradação de dados, que oferece a capacidade de ocultar o formato de caractere nativo subjacente de um objeto do Outlook, permitindo que um cliente acesse o objeto em um formato de caractere preferido, e não no formato de caractere nativo do objeto.
