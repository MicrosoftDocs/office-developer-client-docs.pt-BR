# <a name="text-block-format-section"></a>Seção Text Block Format

Contém as células que controlam o alinhamento, as margens e a posição de texto de parada de tabulação padrão em um bloco de texto da forma.
  
## <a name="remarks"></a>Comentários

Você também pode definir esses atributos de texto usando a caixa de diálogo **texto** (na guia **página inicial** , clique o iniciador de caixa de diálogo do grupo **fonte** ).
