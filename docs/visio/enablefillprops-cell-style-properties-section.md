# <a name="enablefillprops-cell-style-properties-section"></a>Célula EnableFillProps (Seção Style Properties)

Determina se um estilo inclui propriedades de preenchimento.
  
|**Valor**|**Descrição**|
|:-----|:-----|
|VERDADEIRO  <br/> |Incluir propriedades de preenchimento.  <br/> |
|FALSO  <br/> |Excluir propriedades de preenchimento.  <br/> |
   
## <a name="remarks"></a>Comentários

Para obter uma referência para a célula EnableFillProps pelo nome, a partir de outra fórmula ou programa que use a propriedade **CellsU**, utilize: 
  
|||
|:-----|:-----|
|Nome da célula:  <br/> |EnableFillProps  <br/> |
   
Para obter uma referência para a célula EnableFillPropspelo índice a partir de um programa, use a propriedade **CellsSRC** com os seguintes argumentos: 
  
|||
|:-----|:-----|
|Índice da seção:  <br/> |**visSectionObject** <br/> |
|Índice da linha:  <br/> |**visRowStyle** <br/> |
|Índice da célula:  <br/> |**visStyleIncludesFill** <br/> |
