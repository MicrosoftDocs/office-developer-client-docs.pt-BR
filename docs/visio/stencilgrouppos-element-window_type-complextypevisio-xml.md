# <a name="stencilgrouppos-element-windowtype-complextype-visio-xml"></a>Elemento StencilGroupPos (Window_Type complexType) ('Visio XML')

Contém um número inteiro que especifica a posição relativa de um estêncil dentro de um grupo em uma janela.

|**Arquivo de esquema** <br/> |VisioSchema15.xsd  <br/> |
|**Partes do documento** <br/> |Windows.XML  <br/> |

<xs:element name="StencilGroupPos" type="StencilGroupPos_Type" minOccurs="0" maxOccurs="1" >
</xs:element>
