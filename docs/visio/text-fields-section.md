# <a name="text-fields-section"></a>Seção Text Fields

Exibe as funções e as fórmulas inseridas no texto da forma usando a caixa de diálogo **Campo**. 
  
## <a name="remarks"></a>Comentários

A única forma de adicionar essa seção é inserir um campo em uma forma usando a caixa de diálogo **Campo** (guia **Inserir**). 
  
Cada campo de texto em um bloco de texto tem uma linha correspondente na janela ShapeSheet. O nome da linha representa o número de caracteres que o campo contém.
