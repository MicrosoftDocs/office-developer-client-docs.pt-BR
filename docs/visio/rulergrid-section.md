# <a name="ruler-amp-grid-section"></a>Seção Ruler &amp; Grid

Mostra as configurações atuais da grade e das réguas da página.
  
## <a name="remarks"></a>Comentários

Essa seção é uma propriedade de página que pode ser visualizada somente na janela ShapeSheet de uma página. As células que ele contém correspondem às configurações da  caixa de diálogo Grade de Régua (na guia Exibir, no grupo **Mostrar/Ocultar,** clique no iniciador da caixa de diálogo). **&amp;**
