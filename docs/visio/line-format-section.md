# <a name="line-format-section"></a>Seção Line Format

Contém as células que controlam os atributos de linha de uma forma, como padrão, espessura e cor. Elas determinam se as extremidades de linha são formatadas (por exemplo, com uma ponta de seta), o tamanho dos formatos das extremidades de linha, o raio de um círculo arredondado para uma linha e o estilo de fim de linha (arredondado ou quadrado).
  
## <a name="remarks"></a>Comentários

Você pode definir formatos de linha usando o painel **Formatar forma** (na guia **página inicial** , no grupo **estilos de forma** , clique em **linha**, **Opções de linha**), aplicando um estilo de linha ou inserindo uma fórmula em uma célula de formato de linha.
