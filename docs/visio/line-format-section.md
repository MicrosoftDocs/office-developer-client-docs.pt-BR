Elas determinam se as extremidades de linha são formatadas (por exemplo, com uma ponta de seta), o tamanho dos formatos das extremidades de linha, o raio de um círculo arredondado para uma linha e o estilo de fim de linha (arredondado ou quadrado).</span><span class="sxs-lookup"><span data-stu-id="e5185-p102">Contains cells that control line attributes for a shape, such as pattern, weight, and color.
