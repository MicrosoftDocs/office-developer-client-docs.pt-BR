# <a name="noalignbox-cell-miscellaneous-section"></a>Célula NoAlignBox (Seção Miscellaneous)

Alterna entre exibir ou não o retângulo de seleção da forma selecionada.
  
|**Valor**|**Descrição**|
|:-----|:-----|
| VERDADEIRO  <br/> | O retângulo de seleção não é exibido ao selecionar uma forma.  <br/> |
| FALSO  <br/> | O retângulo de seleção é exibido ao selecionar uma forma.  <br/> |
   
## <a name="remarks"></a>Comentários

Para fazer referência à célula NoAlignBox pelo nome, a partir de outra fórmula ou programa que usa a propriedade **CellsU**, utilize: 
  
|||
|:-----|:-----|
| Nome da célula:  <br/> | NoAlignBox  <br/> |
   
Para fazer referência à célula NoAlignBox pelo índice a partir de um programa, use a propriedade **CellsSRC** com os seguintes argumentos: 
  
|||
|:-----|:-----|
| Índice da seção:  <br/> |**visSectionObject** <br/> |
| Índice da linha:  <br/> |**visRowMisc** <br/> |
| Índice da célula:  <br/> |**visNoAlignBox** <br/> |
