# <a name="listmembercount-function"></a>Função LISTMEMBERCOUNT

Retorna o número de formas de membro na forma de contêiner da lista.
  
## <a name="version-information"></a>Informações da versão

Version Added: Visio 2010
 
  
## <a name="syntax"></a>Sintaxe

LISTMEMBERCOUNT()
  
### <a name="return-value"></a>Valor retornado

Inteiro
  
## <a name="remarks"></a>Comentários

Se a forma não for um contêiner de lista, a função LISTMEMBERCOUNT retornará -1.
