# <a name="headercenter-element-headerfootertype-complextype-visio-xml"></a>Elemento HeaderCenter (HeaderFooter_Type complexType) (' Visio XML ')

Obtém ou define a sequência de caracteres de texto que é exibida na parte central do cabeçalho de um documento.
  
## <a name="element-information"></a>Informações de elemento

|||
|:-----|:-----|
|**Tipo de elemento** <br/> |[HeaderCenter_Type](headercenter_type-complextypevisio-xml.md) <br/> |
|**Namespace** <br/> |https://schemas.microsoft.com/office/visio/2012/main  <br/> |
|**Arquivo de esquema** <br/> |VisioSchema15.xsd  <br/> |
|**Partes do documento** <br/> |document.xml  <br/> |
   
## <a name="definition"></a>Definição

```XML
< xs:element name="HeaderCenter" type="HeaderCenter_Type" minOccurs="0" maxOccurs="1" >
</xs:element >
```

Se o esquema definir requisitos específicos, como **sequence**, **minOccurs**,**maxOccurs** e **choice**, confira a seção de definição.
