# <a name="headercenter-element-headerfootertype-complextype-visio-xml"></a>Elemento HeaderCenter (HeaderFooter_Type complexType) ('Visio XML')

Contém a cadeia de caracteres de texto que aparece na parte central do cabeçalho de um documento.

|:-----|:-----|

|**Arquivo de esquema** <br/> |VisioSchema15.xsd  <br/> |
|**Partes do documento** <br/> |Document  <br/> |

```XML
< xs:element name="HeaderCenter" type="HeaderCenter_Type" minOccurs="0" maxOccurs="1" >
</xs:element >
```
