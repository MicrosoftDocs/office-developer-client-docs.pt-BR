# <a name="enabletextprops-cell-style-properties-section"></a>Célula EnableTextProps (Seção Style Properties)

Determina se um estilo inclui propriedades de texto.
  
|**Valor**|**Descrição**|
|:-----|:-----|
|TRUE  <br/> |Incluir propriedades de texto.  <br/> |
|FALSE  <br/> |Excluir propriedades de texto.  <br/> |
   
## <a name="remarks"></a>Comentários

Para obter uma referência para a célula EnableTextProps pelo nome, a partir de outra fórmula ou programa que use a propriedade **CellsU**, utilize: 
  
|||
|:-----|:-----|
|Nome da célula:  <br/> |EnableTextProps  <br/> |
   
Para obter uma referência para a célula EnableTextProps pelo índice a partir de um programa, use a propriedade **CellsSRC** com os seguintes argumentos: 
  
|||
|:-----|:-----|
|Índice da seção:  <br/> |**visSectionObject** <br/> |
|Índice da linha:  <br/> |**visRowStyle** <br/> |
|Índice da célula:  <br/> |**visStyleIncludesText** <br/> |
