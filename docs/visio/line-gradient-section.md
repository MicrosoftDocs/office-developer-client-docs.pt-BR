# <a name="line-gradient-section"></a>Seção Line Gradient

Mostra os valores de cor do gradiente de linha aplicados à forma. 
  
## <a name="remarks"></a>Comentários

Cada [linha da linha Gradient Stop (Seção Line Gradient)](gradient-stop-row-line-gradient-section.md) da seção representa uma parada de gradiente na linha. Esta seção pode estar vazia. 
  
Somente as primeiras 10 linhas contidas na seção são usadas pelas configurações de gradiente. Todas **as linhas Gradient Stop** além de 10 são ignoradas. 
  
Você pode definir as propriedades  de gradiente de linha na  seção Linha  do painel Formatar Forma (na guia Página Base, no grupo Estilos de Forma, clique em Linha e em Opções de **Linha).**   
  
Você pode habilitar ou desabilitar uma cor de linha de gradiente para a forma alterando o valor da célula [LineGradientEnabled (Seção Gradient Properties).](linegradientenabled-cell-gradient-properties-section.md)
