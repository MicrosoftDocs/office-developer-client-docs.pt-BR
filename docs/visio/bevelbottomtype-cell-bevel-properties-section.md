# <a name="bevelbottomtype-cell-bevel-properties-section"></a>Célula BevelBottomType (seção chanfrar Propriedades)

Especifica o tipo de bisel inferior do bisel de uma forma.
  
|**Valor**|**Descrição**|
|:-----|:-----|
|,0  <br/> |Sem bisel  <br/> |
|1  <br/> |Bisel circular  <br/> |
|duas  <br/> |Bisel de baixo reLevo  <br/> |
|3D  <br/> |Bisel cruzado  <br/> |
|quatro  <br/> |Bisel oblíquo frio  <br/> |
|0,5  <br/> |Bisel angular  <br/> |
|6  <br/> |Bisel redondo macio  <br/> |
|178  <br/> |Bisel convexo  <br/> |
|8  <br/> |Bisel angular  <br/> |
|241  <br/> |Bisel malha  <br/> |
|254  <br/> |Bisel talho  <br/> |
|11  <br/> |Bisel de borda sólida  <br/> |
|3,6  <br/> |Bisel-Art Deco  <br/> |
   
## <a name="remarks"></a>Comentários

Para obter uma referência para a célula **BevelBottomType** pelo nome, a partir de outra fórmula, por valor do atributo **N** de um elemento **Cell** no formato de arquivo. vsdx ou de um programa que usa a propriedade **Cells** , utilize: 
  
|||
|:-----|:-----|
| Nome da célula:  <br/> | BevelBottomType  <br/> |
   
Para obter uma referência para a célula **BevelBottomType** pelo índice a partir de um programa, use a propriedade **CellsSRC** com os seguintes argumentos: 
  
|||
|:-----|:-----|
| Índice da seção:  <br/> |**visSectionObject** <br/> |
| Índice da linha:  <br/> |**visRowBevelProperties** <br/> |
| Índice da célula:  <br/> |**visBevelBottomType** <br/> |
