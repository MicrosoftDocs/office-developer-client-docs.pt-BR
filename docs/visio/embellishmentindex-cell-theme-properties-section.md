# <a name="embellishmentindex-cell-theme-properties-section"></a>Célula EmbellishmentIndex (Seção Theme Properties)

Altera a aparência (embelezamento) de callouts, contêineres, linhas do tempo e formas de organograma.
  
## <a name="remarks"></a>Comentários

Para obter uma referência para a célula **EmbelishmentIndex** pelo nome, a partir de outra fórmula, pelo valor do atributo **N** de um elemento **Cell** ou por um programa que usa a propriedade **CellsU,** utilize: 
  
|||
|:-----|:-----|
| Nome da célula:  <br/> | EmbelishmentIndex  <br/> |
   
Para fazer referência à célula **EmbelishmentIndex** pelo índice a partir de um programa, use a propriedade **CellsSRC** com os seguintes argumentos: 
  
|||
|:-----|:-----|
| Índice da seção:  <br/> |**visSectionObject** <br/> |
| Índice de linha:  <br/> |**visRowThemeProperties** <br/> |
| Índice de célula:  <br/> |**visEmbellishmentIndex** <br/> |
