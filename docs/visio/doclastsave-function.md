# <a name="doclastsave-function"></a>Função DOCLASTSAVE

Retorna a data e a hora em que um documento foi salvo pela última vez, como um valor serial.
  
## <a name="syntax"></a>Sintaxe

DOCLASTSAVE ()
  
## <a name="remarks"></a>Comentários

Utilize a função FORMAT para formatar o valor.
