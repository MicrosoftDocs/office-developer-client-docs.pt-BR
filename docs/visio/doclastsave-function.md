# <a name="doclastsave-function"></a>Função DOCLASTSAVE

Retorna a data e hora de que um documento foi salvo pela última vez, de como um valor de série.
  
## <a name="syntax"></a>Sintaxe

DOCLASTSAVE ()
  
## <a name="remarks"></a>Comentários

Utilize a função FORMAT para formatar o valor.
