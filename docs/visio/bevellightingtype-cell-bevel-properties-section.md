# <a name="bevellightingtype-cell-bevel-properties-section"></a>Célula BevelLightingType (Seção Bevel Properties)

Determina o tipo de iluminação usada pelo efeito de bevel.
  
|**Valor**|**Descrição**|
|:-----|:-----|
|0  <br/> |Sem iluminação  <br/> |
|1   <br/> |Three Point  <br/> |
|2   <br/> |Balancear  <br/> |
|3   <br/> |Suaves  <br/> |
|4   <br/> |Insuperá  <br/> |
|5   <br/> |Inundação  <br/> |
|6   <br/> |Contrastando  <br/> |
|7   <br/> |Manhã  <br/> |
|8   <br/> |Nascer do sol  <br/> |
|9   <br/> |Obsoleto  <br/> |
|10   <br/> |Mário  <br/> |
|11  <br/> |Congelamento  <br/> |
|12   <br/> |Plano  <br/> |
|13   <br/> |Two Point  <br/> |
|14   <br/> |Brilhar  <br/> |
|15   <br/> |Sala Brilhante  <br/> |
   
## <a name="remarks"></a>Comentários

Para fazer referência à célula **BevelLightingType** pelo nome, a partir de outra fórmula, pelo valor do atributo **N** de um elemento **Cell** ou por um programa que usa a propriedade **CellsU,** utilize: 
  
|||
|:-----|:-----|
|Nome da célula:  <br/> |BevelLightingType  <br/> |
   
Para fazer referência à célula **BevelLightingType** pelo índice a partir de um programa, use a propriedade **CellsSRC** com os seguintes argumentos: 
  
|||
|:-----|:-----|
|Índice da seção:  <br/> |**visSectionObject** <br/> |
|Índice de linha:  <br/> |**visRowBevelProperties** <br/> |
|Índice de célula:  <br/> |**visBevelLightingType** <br/> |
