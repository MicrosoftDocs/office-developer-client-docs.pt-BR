# <a name="cosh-function"></a>Função COSH

Retorna o cosseno hiperbólico de um ângulo.
  
## <a name="syntax"></a>Sintaxe

COSH ( *ângulo*) 
  
### <a name="parameters"></a>Parâmetros

|**Name**|**Obrigatório/opcional**|**Tipo de dados**|**Descrição**|
|:-----|:-----|:-----|:-----|
| _ângulo_ <br/> |Obrigatório  <br/> |**Numérica** <br/> |O ângulo do qual obter o hiperbólico.  <br/> |
   
## <a name="remarks"></a>Comentários

A função COSH usa radianos, a menos que você especifique uma unidade de medida diferente em *Angle* . 
  
## <a name="example"></a>Exemplo

COSH(45 graus) 
  
Retornará 1,3246.
