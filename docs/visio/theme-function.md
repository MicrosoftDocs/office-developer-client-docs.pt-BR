# <a name="theme-function"></a>Função THEME

Obtém as configurações de formato do tema atual.
  
## <a name="syntax"></a>Sintaxe

THEME()
  
## <a name="example"></a>Exemplo

```vb
Shape.FillForegnd = THEME("FillColor")
```

Atribui a cor de preenchimento do tema atual à forma.
