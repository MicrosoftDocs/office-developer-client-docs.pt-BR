# <a name="foreign-image-info-section"></a>Seção Foreign Image Info

Contém a largura e a altura de um objeto de um outro programa usado em um desenho do Microsoft Visio e aparece na janela ShapeSheet somente para esse objetos. Também indica a distância do deslocamento das imagens do objeto em relação às bordas.
  
## <a name="remarks"></a>Comentários

Nas versões do Visio anteriores à versão 2000, esta seção recebeu o nome **informações da imagem**.
