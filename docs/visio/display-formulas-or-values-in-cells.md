# <a name="display-formulas-or-values-in-cells"></a>Exibir fórmulas ou valores nas células

1. Clique na janela ShapeSheet para torná-la ativa.
    
2. Na guia **Ferramentas do ShapeSheet**, do grupo **Exibir**, clique em **Fórmulas** para exibir fórmulas e em **Valores** para exibir valores. 
    
    > [!NOTE]
    > Também é possível clicar com o botão direito do mouse na janela ShapeSheet e clicar em **Fórmulas** ou **Valores** no menu de atalho.
