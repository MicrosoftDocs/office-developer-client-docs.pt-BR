# <a name="linegradientangle-cell-gradient-properties-section"></a>Célula LineGradientAngle (Seção Gradient Properties)

Determina o ângulo do gradiente de linha para um gradiente linear, de 0 a 359,9 graus.
  
## <a name="remarks"></a>Comentários

Para fazer referência à célula **LineGradientAngle** pelo nome, a partir de outra fórmula, pelo valor do atributo **N** de um elemento **Cell** ou por um programa que usa a propriedade **CellsU,** utilize: 
  
|||
|:-----|:-----|
| Nome da célula:  <br/> | LineGradientAngle  <br/> |
   
Para fazer referência à célula **LineGradientAngle** pelo índice a partir de um programa, use a propriedade **CellsSRC** com os seguintes argumentos: 
  
|||
|:-----|:-----|
| Índice da seção:  <br/> |**visSectionObject** <br/> |
| Índice de linha:  <br/> |**visRowGradientProperties** <br/> |
| Índice de célula:  <br/> |**visLineGradientAngle** <br/> |
