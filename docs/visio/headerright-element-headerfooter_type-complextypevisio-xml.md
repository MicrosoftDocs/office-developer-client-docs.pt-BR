# <a name="headerright-element-headerfootertype-complextype-visio-xml"></a>Elemento HeaderRight (HeaderFooter_Type complexType) ('Visio XML')

|**Arquivo de esquema** <br/> |VisioSchema15.xsd  <br/> |

```XML

</xs:element >
```
