# <a name="sinh-function"></a>Função SINH

Retorna o seno hiperbólico de um ângulo. 
  
## <a name="syntax"></a>Sintaxe

SINH(** *angle* ** ) 
  
### <a name="parameters"></a>Parâmetros

|**Name**|**Obrigatório/opcional**|**Tipo de dados**|**Descrição**|
|:-----|:-----|:-----|:-----|
| _ângulo_ <br/> |Obrigatório  <br/> |**Numérica** <br/> |O ângulo do qual obter o seno hiperbólico.  <br/> |
   
### <a name="return-value"></a>Valor de retorno

Número
  
## <a name="remarks"></a>Comentários

A função SINH usa radianos, a menos que você especifique uma unidade de medida diferente em _ângulo._
  
## <a name="example"></a>Exemplo

SINH(45 graus) 
  
Retornará 0,8687.
