# <a name="ln-function"></a>Função LN

Retorna o logaritmo natural de um número. O número deve ser positivo, ou LN retorna o valor de erro #NUM.
  
## <a name="syntax"></a>Sintaxe

LN (* * *número* * *) 
  
### <a name="parameters"></a>Parâmetros

|**Name**|**Obrigatório/opcional**|**Tipo de dados**|**Descrição**|
|:-----|:-----|:-----|:-----|
| _number_ <br/> |Obrigatório  <br/> |**Numérica** <br/> | O número cujo logaritmo natural você deseja localizar.  <br/> |
   
### <a name="return-value"></a>Valor de retorno

Numeric
  
## <a name="example"></a>Exemplo

LN (42) 
  
Retornará 3,7377.
