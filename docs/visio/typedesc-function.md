# <a name="typedesc-function"></a>Função TYPEDESC

Retorna uma cadeia de caracteres que descreve o tipo interno de um objeto. 
  
## <a name="syntax"></a>Sintaxe

TYPEDESC ()
  
### <a name="return-value"></a>Valor de retorno

String
