# <a name="typedesc-function"></a>Função TYPEDESC

Retorna uma cadeia de caracteres que descreve o tipo de um objeto interno. 
  
## <a name="syntax"></a>Sintaxe

TYPEDESC ()
  
### <a name="return-value"></a>Valor retornado

Cadeia de caracteres
