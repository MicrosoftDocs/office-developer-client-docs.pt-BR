# <a name="tan-function-visioshapesheet"></a>Função TAN (VisioShapeSheet)

Retorna a tangente de um ângulo.
  
## <a name="syntax"></a>Sintaxe

TAN (* * *Angle* * *) 
  
### <a name="parameters"></a>Parâmetros

|**Name**|**Obrigatório/opcional**|**Tipo de dados**|**Descrição**|
|:-----|:-----|:-----|:-----|
| _ângulo_ <br/> |Obrigatório  <br/> |**Numérica** <br/> |O ângulo do qual obter a tangente.  <br/> |
   
### <a name="return-value"></a>Valor de retorno

Numeric
  
## <a name="remarks"></a>Comentários

A função TAN usa radianos, a menos que você especifique uma unidade de medida diferente em *Angle* . 
  
## <a name="example"></a>Exemplo

TAN(45 graus) 
  
Retornará 1.
