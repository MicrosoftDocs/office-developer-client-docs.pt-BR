# <a name="protectshapes-element-documentsettingstype-complextype-visio-xml"></a>Elemento ProtectShapes (DocumentSettings_Type complexType) (' Visio XML ')

Especifica se o usuário é impedido de selecionar formas que têm seu elemento **LockSelect** definido como 1.

|**Arquivo de esquema** <br/> |VisioSchema15.xsd  <br/> |
|**Partes do documento** <br/> |document.xml  <br/> |

```XML
< xs:element name="ProtectShapes" type="ProtectShapes_Type" minOccurs="0" maxOccurs="1" >
</xs:element >
```
