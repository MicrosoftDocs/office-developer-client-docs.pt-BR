# <a name="theme-properties-section"></a>Seção Theme Properties

Mostra os valores de índice do tema atualmente aplicado ao documento, incluindo cor, fonte, conector e valores de efeitos. 
  
## <a name="remarks"></a>Comentários

Você pode definir valores de tema na guia **design** ou fazer alterações nas células da seção de **Propriedades de tema** do ShapeSheet.
