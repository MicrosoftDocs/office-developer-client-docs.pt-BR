# <a name="atan-function"></a>Função ATAN

Retorna o arco tangente de um número, por exemplo, o ângulo cuja tangente é _número_. O ângulo resultante está no intervalo -PI/2 < = ângulo < = PI/2 radianos (-90 < = ângulo < = 90 graus). 
  
## <a name="syntax"></a>Sintaxe

ATAN (* * *número* * *) 
  
### <a name="parameters"></a>Parâmetros

|**Name**|**Obrigatório/Opcional**|**Tipo de dados**|**Descrição**|
|:-----|:-----|:-----|:-----|
| _number_ <br/> |Obrigatório  <br/> |**Numérico** <br/> |A tangente do ângulo.  <br/> |
   
## <a name="example"></a>Exemplo

ATAN(1) 
  
Retorna 45 graus.
