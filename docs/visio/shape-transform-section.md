# <a name="shape-transform-section"></a>Seção Shape Transform

Contém informações gerais de posicionamento de uma forma: por exemplo, a largura, a altura, o ângulo e o centro de rotação (pino); se a forma foi virada e como a forma deve se comportar quando redimensionada em um grupo.
