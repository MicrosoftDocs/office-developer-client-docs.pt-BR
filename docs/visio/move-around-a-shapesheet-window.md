# <a name="move-around-a-shapesheet-window"></a>Mover-se pela janela do ShapeSheet

Para mover de uma célula para outra em uma janela do ShapeSheet, utilize as teclas de seta e a tecla TAB. Clique em uma célula para ativá-la ou role pelas células da janela do ShapeSheet sem alterar a célula ativa.
  
Você pode também utilizar as técnicas da tabela a seguir.
  
|**To**|**Faça isto**|
|:-----|:-----|
| Ativar uma célula  <br/> | Clique na célula com o ponteiro.  <br/> |
| Selecionar uma linha de células numerada  <br/> | Clique no número da linha.  <br/> |
| Mover uma célula à esquerda  <br/> | Pressione a tecla SETA À ESQUERDA ou SHIFT+TAB.  <br/> |
| Mover uma célula à direita  <br/> | Pressione a tecla SETA À DIREITA ou TAB.  <br/> |
| Mover uma célula para cima  <br/> | Pressione a tecla SETA PARA CIMA.  <br/> |
| Mover uma célula para baixo  <br/> | Pressione a tecla SETA PARA BAIXO.  <br/> |
| Mover para a última célula na linha  <br/> | Pressione END.  <br/> |
| Mover para a primeira célula na linha  <br/> | Pressione HOME.  <br/> |
| Rolar para baixo uma tela  <br/> | Pressione PAGEDOWN.  <br/> |
| Rolar para cima uma tela  <br/> | Pressione PAGEUP.  <br/> |
| Rolar para a direita uma tela  <br/> | Pressione CTRL+PAGEDOWN.  <br/> |
| Rolar para a esquerda uma tela  <br/> | Pressione CTRL+PAGEUP.  <br/> |
| Rolar para o canto inferior direito da janela  <br/> | Pressione CTRL+END.  <br/> |
| Rolar para o canto superior direito da janela  <br/> | Pressione CTRL+HOME.  <br/> |
