# <a name="page-properties-section"></a>Seção Page Properties

Contém as células que controlam os atributos de página, como a escala, a altura e a largura da página.
  
## <a name="remarks"></a>Comentários

Você pode definir os valores das propriedades de página fazendo uma entrada na célula seção **Propriedades da página** , ou usando a caixa de diálogo **Configurar página** (na guia **Design** , clique o iniciador de caixa de diálogo do grupo **Configurar página** ).
