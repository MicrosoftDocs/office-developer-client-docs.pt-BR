# <a name="fonttoid-function"></a>Função FONTTOID

Retorna o identificador (ID) da fonte especificada.
  
> [!NOTE]
> É recomendável usar a função de [fonte](font-function.md) , que é semelhante à função **FONTTOID** . No Microsoft Visio 2013, a função de **fonte** foi introduzida para substituir **FONTTOID**. 
  
## <a name="syntax"></a>Sintaxe

FONTTOID (* * *font_name* * *) 
  
### <a name="parameters"></a>Parâmetros

|**Name**|**Obrigatório/Opcional**|**Tipo de dados**|**Descrição**|
|:-----|:-----|:-----|:-----|
| _font_name_ <br/> |Obrigatório  <br/> |**String** <br/> |O nome da fonte.  <br/> |
