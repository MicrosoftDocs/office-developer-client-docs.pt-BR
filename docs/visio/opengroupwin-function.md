# <a name="opengroupwin-function"></a>Função OPENGROUPWIN

Abre o grupo na janela de grupo. Se o objeto não for um grupo, essa função não faz nada.
  
## <a name="syntax"></a>Sintaxe

OPENGROUPWIN( )
  
## <a name="remarks"></a>Comentários

Em versões anteriores do Visio, essa função aparece como _OPENGROUPWIN. As versões 4.0 e posteriores do Visio aceitam os dois estilos.
