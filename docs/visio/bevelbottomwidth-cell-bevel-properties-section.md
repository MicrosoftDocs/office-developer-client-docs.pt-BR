# <a name="bevelbottomwidth-cell-bevel-properties-section"></a>Célula BevelBottomWidth (seção Propriedades de bisel)

Determina a largura do bisel baixo em pontos. 
  
## <a name="remarks"></a>Coment�rios

Para fazer referência à célula **BevelBottomWidth** pelo nome a partir de outra fórmula, pelo valor do atributo **N** de um elemento de **célula** ou um programa que usa a propriedade **CellsU** , utilize: 
  
|||
|:-----|:-----|
| Nome da célula:  <br/> | BevelBottomWidth  <br/> |
   
Para obter uma referência à célula **BevelBottomWidth** pelo índice a partir de um programa, use a propriedade **CellsSRC** com os seguintes argumentos: 
  
|||
|:-----|:-----|
| Índice da seção:  <br/> |**visSectionObject** <br/> |
| Índice da linha:  <br/> |**visRowBevelProperties** <br/> |
| Índice da célula:  <br/> |**visBevelBottomWidth** <br/> |
