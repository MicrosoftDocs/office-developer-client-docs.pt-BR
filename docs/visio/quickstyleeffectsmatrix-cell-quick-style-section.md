# <a name="quickstyleeffectsmatrix-cell-quick-style-section"></a>Célula QuickStyleEffectsMatrix (Seção Quick Style)

Determina os efeitos de Estilo Rápido que a forma herda do tema ativo, como um inteiro de 0 a 6. 
  
## <a name="remarks"></a>Comentários

Para fazer referência à célula **QuickStyleEffectsMatrix** pelo nome, a partir de outra fórmula, pelo valor do atributo **N** de um elemento **Cell** ou por um programa que usa a propriedade **CellsU,** utilize: 
  
|||
|:-----|:-----|
| Nome da célula:  <br/> | QuickStyleEffectsMatrix  <br/> |
   
Para fazer referência à **célula QuickStyleEffectsMatrix** pelo índice a partir de um programa, use a propriedade **CellsSRC** com os seguintes argumentos: 
  
|||
|:-----|:-----|
| Índice da seção:  <br/> |**visSectionObject** <br/> |
| Índice de linha:  <br/> |**visRowQuickStyleProperties** <br/> |
| Índice de célula:  <br/> |**visQuickStyleEffectsMatrix** <br/> |
