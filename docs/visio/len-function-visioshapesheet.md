# <a name="len-function-visioshapesheet"></a>Função LEN (VisioShapeSheet)

Retorna o número de caracteres em uma cadeia de caracteres de texto.
  
## <a name="syntax"></a>Sintaxe

LEN (** *texto* ** ) 
  
### <a name="parameters"></a>Parâmetros

|**Name**|**Obrigatório/opcional**|**Tipo de dados**|**Descrição**|
|:-----|:-----|:-----|:-----|
| _text_ <br/> |Obrigatório  <br/> |**String** <br/> |A cadeia de caracteres cujo comprimento você quer localizar.  <br/> |
   
### <a name="return-value"></a>Valor de retorno

Numeric
  
## <a name="example"></a>Exemplo

LEN ("Seattle, WA") 
  
Retornará 11.
