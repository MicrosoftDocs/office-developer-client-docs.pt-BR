# <a name="showconnectionpoints-element-windowtype-complextype-visio-xml"></a>Elemento ShowConnectionPoints (Window_Type complexType) ('Visio XML')

|||
|:-----|:-----|
|**Tipo de elemento** <br/> |[ShowConnectionPoints_Type](showconnectionpoints_type-complextypevisio-xml.md) <br/> |
|**Namespace** <br/> |http://schemas.microsoft.com/office/visio/2012/main  <br/> |
|**Arquivo de esquema** <br/> |VisioSchema15.xsd  <br/> |
|**Partes do documento** <br/> |Windows.XML  <br/> |

```XML
< xs:element name="ShowConnectionPoints" type="ShowConnectionPoints_Type" minOccurs="0" maxOccurs="1" >
</xs:element >
```
