|:-----|:-----|
|**Tipo de elemento** <br/> |[ShowConnectionPoints_Type](showconnectionpoints_type-complextypevisio-xml.md) <br/> |

|**Arquivo de esquema** <br/> |VisioSchema15.xsd  <br/> |
