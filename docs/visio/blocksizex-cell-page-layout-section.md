# <a name="blocksizex-cell-page-layout-section"></a>Célula BlockSizeX (Seção Page Layout)

Determina o tamanho do bloco horizontal, a área na qual cada uma das formas deve caber na página de desenho ao dispor as formas usando a caixa de diálogo **Configurar layout** (na guia **design** , no grupo **layout** , clique em refazer o **layout da página **e, em seguida, clique em **mais opções de layout**.
  
## <a name="remarks"></a>Comentários

Também é possível definir esse valor na caixa de diálogo **Espaçamento de Layout e Direcionamento** (na guia **Design**, clique na seta do grupo **Configurar Página**, clique na guia **Layout e Direcionamento** e em **Espaçamento**).
  
Para obter uma referência para a célula BlockSizeX pelo nome, a partir de outra fórmula ou programa que use a propriedade **CellsU**, utilize: 
  
|||
|:-----|:-----|
|Nome da célula:  <br/> |BlockSizeX  <br/> |
   
Para obter uma referência para a célula BlockSizeX pelo índice a partir de um programa, use a propriedade **CellsSRC** com os seguintes argumentos: 
  
|||
|:-----|:-----|
| Índice da seção:  <br/> |**visSectionObject** <br/> |
| Índice da linha:  <br/> |**visRowPageLayout** <br/> |
| Índice da célula:  <br/> |**visPLOBlockSizeX** <br/> |
