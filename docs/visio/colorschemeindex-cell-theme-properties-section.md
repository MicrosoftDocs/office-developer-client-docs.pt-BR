# <a name="colorschemeindex-cell-theme-properties-section"></a>Célula ColorSchemeIndex (Seção Theme Properties)

Determina o índice do tema que o esquema de cores da forma assume depois, como um inteiro.
  
## <a name="remarks"></a>Comentários

Para fazer referência à célula **ColorSchemeIndex** pelo nome, a partir de outra fórmula, pelo valor do atributo **N** de um elemento **Cell** ou por um programa que usa a propriedade **CellsU,** utilize: 
  
|||
|:-----|:-----|
| Nome da célula:  <br/> | ColorSchemeIndex  <br/> |
   
Para fazer referência à célula **ColorSchemeIndex** pelo índice a partir de um programa, use a propriedade **CellsSRC** com os seguintes argumentos: 
  
|||
|:-----|:-----|
| Índice da seção:  <br/> |**visSectionObject** <br/> |
| Índice de linha:  <br/> |**visRowThemeProperties** <br/> |
| Índice de célula:  <br/> |**visColorSchemeIndex** <br/> |
