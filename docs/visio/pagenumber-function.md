# <a name="pagenumber-function"></a>Função PAGENUMBER

Retorna o número da página que contém a forma. 
  
## <a name="syntax"></a>Sintaxe

PAGENUMBER ()
  
### <a name="return-value"></a>Valor de retorno

Número
  
## <a name="remarks"></a>Comentários

Se a página atual for uma página de plano de fundo (sem número), ou se ocorrer um erro, a função PAGENUMBER retornará zero.
