# <a name="drawingresizetype-cell-page-properties-section"></a>Célula DrawingResizeType (Seção Page Properties)

Determina se a página de desenho é ressalvada automaticamente para se ajustar ao diagrama. 
  
## <a name="remarks"></a>Comentários

Para obter uma referência à célula DrawingResizeType pelo nome a partir de outra fórmula ou de um programa que use a propriedade **CellsU**, utilize: 
  
|||
|:-----|:-----|
|Nome da célula:  <br/> |DrawingResizeType  <br/> |
   
Para obter uma referência à célula DrawingResizeType pelo índice a partir de um programa, use a propriedade **CellsSRC** com os seguintes argumentos: 
  
|||
|:-----|:-----|
|Índice da seção:  <br/> |**visSectionObject** <br/> |
|Índice de linha:  <br/> |**visRowPage** <br/> |
|Índice de célula:  <br/> |**visPageDrawResizeType** <br/> |
