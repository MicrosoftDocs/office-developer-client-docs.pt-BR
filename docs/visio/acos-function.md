# <a name="acos-function"></a>Função ACOS

Retorna o arco cosseno de um número, por exemplo, o ângulo cujo cosseno é *número* . 
  
## <a name="syntax"></a>Sintaxe

ACOS ( *número*) 
  
## <a name="remarks"></a>Comentários

O valor de entrada deve estar no intervalo -1 \<= número \<= 1 ou um #NUM! erro será retornado. O ângulo resultante está no intervalo de 0 \<= ângulo \<= PI radianos (0 \<= ângulo \<= 180 graus).
  
## <a name="example"></a>Exemplo

ACOS(0) 
  
Retorna 90 graus.
