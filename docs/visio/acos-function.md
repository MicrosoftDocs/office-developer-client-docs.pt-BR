# <a name="acos-function"></a>Função ACOS

Retorna o arco cosseno do número, por exemplo, o ângulo cujo cosseno é o *número* . 
  
## <a name="syntax"></a>Sintaxe

ACOS ( *número*) 
  
## <a name="remarks"></a>Comentários

O valor de entrada deve estar no intervalo-1 \<= número \<= 1 ou um #NUM! será retornado. O ângulo \<resultante está no intervalo 0 = Angle \<= pi radianos (0 \<= Angle \<= 180 graus).
  
## <a name="example"></a>Exemplo

ACOS (0) 
  
Retorna 90 graus.
