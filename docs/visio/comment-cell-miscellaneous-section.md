# <a name="comment-cell-miscellaneous-section"></a>Célula Comment (Seção Miscellaneous)

Contém o texto de comentário em formato de sequência de caracteres de uma forma.
  
## <a name="remarks"></a>Comentários

Também é possível inserir um comentário clicando em **Novo Comentário** na guia **Revisão**). 
  
Para obter uma referência para a célula Comment pelo nome, a partir de outra fórmula ou programa que use a propriedade **CellsU**, utilize: 
  
|||
|:-----|:-----|
|Nome da célula:  <br/> |Comentário  <br/> |
   
Para fazer referência à célula Comment pelo índice a partir de um programa, use a propriedade **CellsSRC** com estes argumentos: 
  
|||
|:-----|:-----|
|Índice da seção:  <br/> |**visSectionObject** <br/> |
|Índice da linha:  <br/> |**visRowMisc** <br/> |
|Índice da célula:  <br/> |**visComment** <br/> |
