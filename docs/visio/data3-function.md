# <a name="data3-function"></a>Função DATA3

Retorna o texto da caixa **dados 3** na caixa de **diálogo nome da forma** para uma forma (na guia **desenvolvedor** , clique em **nome da forma** ), como uma cadeia de caracteres. 
  
## <a name="syntax"></a>Sintaxe

DATA3 ()
  
## <a name="remarks"></a>Comentários

Se a caixa estiver vazia, a função retornará uma cadeia de caracteres vazia.
