# <a name="data3-function"></a>Função DATA3

Retorna o texto da caixa de **Data 3** na caixa de diálogo **Nome da forma** para uma forma (na guia **desenvolvedor** , clique o **Nome da forma** ), como uma cadeia de caracteres. 
  
## <a name="syntax"></a>Sintaxe

DATA3 ()
  
## <a name="remarks"></a>Comentários

Se a caixa estiver vazia, a função retornará uma cadeia de caracteres vazia.
