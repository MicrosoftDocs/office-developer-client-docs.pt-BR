# <a name="layers-section"></a>Seção Layers

Mostra todas as camadas definidas para a página e as propriedades definidas para cada camada. 
  
## <a name="remarks"></a>Comentários

Essa seção é uma propriedade de página que pode ser visualizada somente na janela ShapeSheet de uma página. Suas células correspondem às opções na caixa de diálogo **Propriedades da camada** (na guia **Página Inicial** do grupo **Edição**, clique em **Camadas** e em **Propriedades da Camada**).
