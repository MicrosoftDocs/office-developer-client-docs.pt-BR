# <a name="icon-element-mastertype-complextype-visio-xml"></a>Elemento de ícone (Master_Type complexType) ('Visio XML')

Especifica uma MIME (Multipurpose Internet Mail Extensions) codificada ícone binário (no formato. ico) para um mestre elemento em um documento.
  
## <a name="element-information"></a>Elemento de informações

|||
|:-----|:-----|
|**Tipo de elemento** <br/> |[Icon_Type](icon_type-complextypevisio-xml.md) <br/> |
|**Namespace** <br/> |https://schemas.microsoft.com/office/visio/2012/main  <br/> |
|**Arquivo de esquema** <br/> |VisioSchema15.xsd  <br/> |
|**Partes do documento** <br/> |Masters.XML  <br/> |
   
## <a name="definition"></a>Definição

```XML
< xs:element name="Icon" type="Icon_Type" minOccurs="0" maxOccurs="1" >
</xs:element >
```

## <a name="elements-and-attributes"></a>Elementos e atributos

Se o esquema define os requisitos específicos, como a **sequência**, **minOccurs**, **maxOccurs**e **Escolha**, consulte a seção de definição.

|[Master](master-element-masters_type-complextypevisio-xml.md) <br/> |[Master_Type](master_type-complextypevisio-xml.md) <br/> |Especifica um mestre em um desenho.  <br/> |
   
### <a name="child-elements"></a>Elementos filho
