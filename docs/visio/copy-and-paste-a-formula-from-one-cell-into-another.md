# <a name="copy-and-paste-a-formula-from-one-cell-into-another"></a>Copiar e colar uma fórmula de uma célula para outra

1. Selecione a célula que contém a fórmula desejada.
    
2. Pressione CTRL + C.
    
    Para substituir a fórmula inteira, selecione a célula cuja fórmula você deseja substituir.
    
    Para colar a fórmula em uma fórmula existente, clique duas vezes na célula e, em seguida, clique em um ponto de inserção onde deseja colar a fórmula.
    
3. Pressione CTRL + V
    
4. Pressione ENTER para aceitar a fórmula.
