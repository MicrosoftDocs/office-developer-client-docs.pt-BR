## <a name="definition"></a>Definição

```XML
          <xs:complexType name="Comments_Type">
          
          <xs:sequence>
    <xs:element name="AuthorList"  type="AuthorList_Type"
     minOccurs="0"
     maxOccurs="1"
    >
    </xs:element>

    >

  type="xsd:boolean"
    />
      </xs:complexType>
      
```
