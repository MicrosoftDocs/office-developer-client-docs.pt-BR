# <a name="type-function"></a>Função TYPE

Retorna o tipo interno de um objeto como um número. 
  
## <a name="syntax"></a>Sintaxe

TYPE ()
  
### <a name="return-value"></a>Valor de retorno

Número
  
## <a name="remarks"></a>Comentários

A tabela a seguir relaciona tipos de objetos internos como números.

|Page  <br/> |70  <br/> |
|Grupo  <br/> |71  <br/> |
|Forma  <br/> |72  <br/> |
|Estilo  <br/> |74  <br/> |
|Escrita  <br/> |77  <br/> |
|Foreign  <br/> |78  <br/> |
|Documento  <br/> |79  <br/> |
