# <a name="type-function"></a>Função TYPE

Retorna o tipo interno de um objeto como um número. 
  
## <a name="syntax"></a>Sintaxe

TYPE ()
  
### <a name="return-value"></a>Valor de retorno

Número
  
## <a name="remarks"></a>Comentários

A tabela a seguir relaciona tipos de objetos internos como números.
  
|**Object**|**Número**|
|:-----|:-----|
|Page  <br/> |70  <br/> |
|Group  <br/> |71  <br/> |
|Shape  <br/> |72  <br/> |
|Style  <br/> |74  <br/> |
|Guia  <br/> |77  <br/> |
|Externo  <br/> |78  <br/> |
|Document  <br/> |79  <br/> |
