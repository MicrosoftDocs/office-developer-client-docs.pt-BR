# <a name="updatealignbox-cell-miscellaneous-section"></a>Célula UpdateAlignBox (Seção Miscellaneous)

Recalcula o retângulo de seleção sempre que uma alça de controle for movida.
  
## <a name="remarks"></a>Comentários

Para obter uma referência à célula UpdateAlignBox pelo nome, a partir de outra fórmula ou programa que usa a propriedade **CellsU** , utilize: 
  
|||
|:-----|:-----|
| Nome da célula:  <br/> | UpdateAlignBox  <br/> |
   
Para obter uma referência à célula UpdateAlignBox pelo índice a partir de um programa, use a propriedade **CellsSRC** com os seguintes argumentos: 
  
|||
|:-----|:-----|
| Índice da seção:  <br/> |**visSectionObject** <br/> |
| Índice da linha:  <br/> |**visRowMisc** <br/> |
| Índice da célula:  <br/> |**visUpdateAlignBox** <br/> |
