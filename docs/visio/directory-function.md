# <a name="directory-function"></a>Função DIRECTORY

Retorna o caminho completo do diretório (pasta) no qual um documento é salvo, como uma cadeia de caracteres. Retorna uma cadeia de caracteres vazia se o documento não tiver sido salvo.
  
## <a name="syntax"></a>Sintaxe

DIRECTORY ()
