# <a name="shapeshdwblur-cell-fill-format-section"></a>Célula ShapeShdwBlur (Seção Fill Format)

Determina o tamanho do desfoque da sombra de uma forma, em pontos (0,00 a 100,00). 
  
## <a name="remarks"></a>Comentários

Para fazer referência à célula **ShapeShdwBlur** pelo nome, a partir de outra fórmula, pelo valor do atributo **N** de um elemento **Cell** ou por um programa que usa a propriedade **CellsU,** utilize: 
  
|||
|:-----|:-----|
| Nome da célula:  <br/> | ShapeShdwBlur  <br/> |
   
Para fazer referência à **célula ShapeShdwBlur** pelo índice a partir de um programa, use a propriedade **CellsSRC** com os seguintes argumentos: 
  
|||
|:-----|:-----|
| Índice da seção:  <br/> |**visSectionObject** <br/> |
| Índice de linha:  <br/> |**visRowFill** <br/> |
| Índice de célula:  <br/> |**visFillShdwBlur** <br/> |
