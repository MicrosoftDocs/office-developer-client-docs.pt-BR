# <a name="shapeshdwblur-cell-fill-format-section"></a>Célula ShapeShdwBlur (seção Fill Format)

Determina o tamanho do desfoque para a sombra de uma forma, em pontos (0, 0 a 100, 0). 
  
## <a name="remarks"></a>Comentários

Para obter uma referência para a célula **ShapeShdwBlur** pelo nome, a partir de outra fórmula, por valor do atributo **N** de um elemento **Cell** ou de um programa que usa a propriedade **Cells** , utilize: 
  
|||
|:-----|:-----|
| Nome da célula:  <br/> | ShapeShdwBlur  <br/> |
   
Para obter uma referência para a célula **ShapeShdwBlur** pelo índice a partir de um programa, use a propriedade **CellsSRC** com os seguintes argumentos: 
  
|||
|:-----|:-----|
| Índice da seção:  <br/> |**visSectionObject** <br/> |
| Índice da linha:  <br/> |**visRowFill** <br/> |
| Índice da célula:  <br/> |**visFillShdwBlur** <br/> |
