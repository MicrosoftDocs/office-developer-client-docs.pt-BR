# <a name="connectorschemeindex-cell-theme-properties-section"></a>Célula ConnectorSchemeIndex (Seção Theme Properties)

Determina o esquema de conector de um tema que é aplicado à forma, como um inteiro. 
  
## <a name="remarks"></a>Comentários

Para fazer referência à célula **ConnectorSchemeIndex** pelo nome, a partir de outra fórmula, pelo valor do atributo **N** de um elemento **Cell** ou por um programa que usa a propriedade **CellsU,** utilize: 
  
|||
|:-----|:-----|
| Nome da célula:  <br/> | ConnectorSchemeIndex  <br/> |
   
Para fazer referência à **célula ConnectorSchemeIndex** pelo índice a partir de um programa, use a propriedade **CellsSRC** com os seguintes argumentos: 
  
|||
|:-----|:-----|
| Índice da seção:  <br/> |**visSectionObject** <br/> |
| Índice de linha:  <br/> |**visRowThemeProperties** <br/> |
| Índice de célula:  <br/> |**visConnectorSchemeIndex ** <br/> |
