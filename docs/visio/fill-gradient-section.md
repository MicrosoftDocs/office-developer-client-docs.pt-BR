# <a name="fill-gradient-section"></a>Seção Fill Gradient

Mostra os valores de cor do gradiente de preenchimento aplicados à forma. 
  
## <a name="remarks"></a>Comentários

Cada [linha da linha Gradient Stop (Seção Fill Gradient)](gradient-stop-row-fill-gradient-section.md) na seção representa uma parada de gradiente no preenchimento. Esta seção pode estar vazia. 
  
Somente as primeiras 10 linhas contidas na seção são usadas pelas configurações de gradiente. Todas **as linhas Gradient Stop** além de 10 são ignoradas. 
  
You can set the fill gradient properties in the **Format Shape** pane (on the **Home** tab, in the **Shape Styles** group, click **Fill**, and then click **Fill Options**). 
  
Você pode habilitar ou desabilitar um preenchimento de gradiente para a forma alterando o valor da célula [FillGradientEnabled (Seção Gradient Properties).](fillgradientenabled-cell-gradient-properties-section.md)
