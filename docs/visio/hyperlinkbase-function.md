# <a name="hyperlinkbase-function"></a>Função HYPERLINKBASE

Retorna o texto do campo **base hiperlink** das propriedades de um documento. 
  
## <a name="syntax"></a>Sintaxe

HYPERLINKBASE ()
  
### <a name="return-value"></a>Valor de retorno

Cadeia de caracteres
