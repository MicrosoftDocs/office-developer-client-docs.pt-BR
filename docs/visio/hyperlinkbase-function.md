# <a name="hyperlinkbase-function"></a>Função HYPERLINKBASE

Retorna o texto do campo **base do hiperlink** de propriedades de um documento. 
  
## <a name="syntax"></a>Sintaxe

HYPERLINKBASE ()
  
### <a name="return-value"></a>Valor retornado

String
