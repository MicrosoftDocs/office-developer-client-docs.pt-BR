# <a name="controls-section"></a>Seção Controls

Contém uma linha e células para cada alça de controle definida para a forma.
  
## <a name="remarks"></a>Comentários

Você pode definir as alças de controle, na seção **controles** na janela ShapeSheet. Se a forma não tiver nenhum seção **Controls** , você pode inserir um na janela ShapeSheet. Com o botão direito na janela ShapeSheet, clique em **Inserir seção**e, em seguida, marque a caixa de seleção de **controles** . Para adicionar as alças de controle a uma seção **Controls** existente, do mouse em uma linha e clique em **Inserir linha** no menu de atalho.
