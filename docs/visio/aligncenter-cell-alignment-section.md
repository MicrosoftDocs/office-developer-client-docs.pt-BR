# <a name="aligncenter-cell-alignment-section"></a>Célula AlignCenter (Seção Alignment)

Determina a posição horizontal, em relação à origem de seu pai, de uma guia vertical ou ponto de guia ao qual o centro horizontal da forma está alinhado.
  
## <a name="remarks"></a>Comentários

Para fazer referência à célula AlignCenter pelo nome, a partir de outra fórmula ou programa que usa a propriedade **CellsU**, utilize: 
  
|||
|:-----|:-----|
| Nome da célula:  <br/> | AlignCenter  <br/> |
   
Para fazer referência à célula AlignCenter pelo índice a partir de um programa, use a propriedade **CellsSRC** com os seguintes argumentos: 
  
|||
|:-----|:-----|
| Índice da seção:  <br/> |**visSectionObject** <br/> |
| Índice da linha:  <br/> |**visRowAlign** <br/> |
| Índice da célula:  <br/> |**visAlignCenter** <br/> |
