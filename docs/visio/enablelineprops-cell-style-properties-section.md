# <a name="enablelineprops-cell-style-properties-section"></a>Célula EnableFillProps (Seção Style Properties)

Determina se um estilo inclui propriedades de linha.
  
|**Valor**|**Descrição**|
|:-----|:-----|
|VERDADEIRO  <br/> |Incluir propriedades de linha.  <br/> |
|FALSE  <br/> |Excluir propriedades de linha.  <br/> |
   
## <a name="remarks"></a>Comentários

Para obter uma referência para a célula EnableFillProps pelo nome, a partir de outra fórmula ou programa que use a propriedade **CellsU**, utilize: 
  
|||
|:-----|:-----|
|Nome da célula:  <br/> |EnableFillProps  <br/> |
   
Para obter uma referência para a célula EnableFillProps pelo índice a partir de um programa, use a propriedade **CellsSRC** com os seguintes argumentos: 
  
|||
|:-----|:-----|
|Índice da seção:  <br/> |**visSectionObject** <br/> |
|Índice de linha:  <br/> |**visRowStyle** <br/> |
|Índice da célula:  <br/> |**visStyleIncludesLine** <br/> |
