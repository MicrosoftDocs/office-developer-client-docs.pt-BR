# <a name="reference-a-cell-in-a-formula"></a>Fazer referência de uma célula em uma fórmula

1. Clique duas vezes na célula em que deseja colocar a referência e coloque o ponto de inserção no local que deseja na fórmula.
    
2. Clique na célula à qual você deseja fazer referência.
    
3. Pressione ENTER para aceitar a fórmula e ESC para cancelar as alterações.
