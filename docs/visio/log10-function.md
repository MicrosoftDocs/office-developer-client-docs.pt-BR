# <a name="log10-function"></a>Função LOG10

Retorna o logaritmo de base 10 de um número. O número deve ser positivo ou LOG10 retorna o valor de erro #NUM.
  
## <a name="syntax"></a>Sintaxe

LOG10 (* * *número* * *) 
  
### <a name="parameters"></a>Parâmetros

|**Nome**|**Obrigatório/opcional**|**Tipo de dados**|**Descrição**|
|:-----|:-----|:-----|:-----|
| _number_ <br/> |Obrigatório  <br/> |**Numeric** <br/> | O número cujo logaritmo de base 10 você deseja localizar.  <br/> |
   
### <a name="return-value"></a>Valor de retorno

Numeric
  
## <a name="example"></a>Exemplo

LOG10 (42) 
  
Retornará 1,6232.
