# <a name="blob-function"></a>Função BLOB

Decodifica uma sequência de caracteres base64 e a armazena como dados binários. 
  
## <a name="syntax"></a>Sintaxe

BLOB (* * *texto binário* * *) 
  
### <a name="parameters"></a>Parâmetros

|**Name**|**Obrigatório/Opcional**|**Tipo de dados**|**Descrição**|
|:-----|:-----|:-----|:-----|
| _texto binário_ <br/> |Obrigatório  <br/> |**String** <br/> | Uma cadeia de caracteres codificada base64.  <br/> |
