# <a name="effectschemeindex-cell-theme-properties-section"></a>Célula EffectSchemeIndex (Seção Theme Properties)

Determina o esquema de efeitos do tema aplicado a uma forma, como um inteiro.
  
## <a name="remarks"></a>Comentários

Para fazer referência à célula **EffectSchemeIndex** pelo nome, a partir de outra fórmula, pelo valor do atributo **N** de um elemento **Cell** ou por um programa que usa a propriedade **CellsU,** utilize: 
  
|||
|:-----|:-----|
| Nome da célula:  <br/> | EffectSchemeIndex  <br/> |
   
Para fazer referência à célula **EffectSchemeIndex** pelo índice a partir de um programa, use a propriedade **CellsSRC** com os seguintes argumentos: 
  
|||
|:-----|:-----|
| Índice da seção:  <br/> |**visSectionObject** <br/> |
| Índice de linha:  <br/> |**visRowThemeProperties** <br/> |
| Índice de célula:  <br/> |**visEffectSchemeIndex ** <br/> |
