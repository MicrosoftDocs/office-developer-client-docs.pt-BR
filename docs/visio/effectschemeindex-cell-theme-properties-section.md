# <a name="effectschemeindex-cell-theme-properties-section"></a>Célula EffectSchemeIndex (seção Theme Properties)

Determina o esquema de efeito do tema aplicado a uma forma, como um inteiro.
  
## <a name="remarks"></a>Comentários

Para obter uma referência para a célula **EffectSchemeIndex** pelo nome, a partir de outra fórmula, por valor do atributo **N** de um elemento **Cell** ou de um programa que usa a propriedade **Cells** , utilize: 
  
|||
|:-----|:-----|
| Nome da célula:  <br/> | EffectSchemeIndex  <br/> |
   
Para obter uma referência para a célula **EffectSchemeIndex** pelo índice a partir de um programa, use a propriedade **CellsSRC** com os seguintes argumentos: 
  
|||
|:-----|:-----|
| Índice da seção:  <br/> |**visSectionObject** <br/> |
| Índice da linha:  <br/> |**visRowThemeProperties** <br/> |
| Índice da célula:  <br/> |* * visEffectSchemeIndex * * <br/> |
