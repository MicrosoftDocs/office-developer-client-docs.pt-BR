# <a name="page-layout-section"></a>Seção Page Layout

Contém as células que controlam as configurações de página para formas e conectores, como o espaçamento entre todas as formas da página, o espaçamento entre todos os conectores na página e o estilo de roteamento de todos os conectores da página.
  
## <a name="remarks"></a>Comentários

Para substituir os padrões da página para uma forma específica, use as células da seção **Shape Layout** . 
  
Você também pode definir os valores das células desta seção na guia **Layout e roteamento** na caixa de diálogo **Configurar página** (na guia **Design** , clique o iniciador de caixa de diálogo do grupo **Configurar página** ).
