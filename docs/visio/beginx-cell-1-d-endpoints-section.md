# <a name="beginx-cell-1-d-endpoints-section"></a>Célula InícioX (Seção 1-D Pontos de Extremidade)

Representa a coordenada  *x*  do ponto inicial da forma 1-D, em relação à origem de seu item pai. 
  
## <a name="remarks"></a>Comentários

Para fazer referência à célula BeginX pelo nome, a partir de outra fórmula ou programa que usa a propriedade **CellsU**, utilize: 
  
|||
|:-----|:-----|
| Nome da célula:  <br/> | InícioX  <br/> |
   
Para fazer referência à célula BeginX pelo índice, a partir de um programa, use a propriedade **CellsSRC** com os seguintes argumentos: 
  
|||
|:-----|:-----|
| Índice da seção:  <br/> |**visSectionObject** <br/> |
| Índice de linha:  <br/> |**visRowXForm1D** <br/> |
| Índice de célula:  <br/> |**vis1DBeginX** <br/> |
