# <a name="quickstylefontmatrix-cell-quick-style-section"></a>Célula QuickStyleFontMatrix (Seção Quick Style)

Determina o estilo da fonte para cada Estilo Rápido, como um inteiro de 1 a 6.
  
## <a name="remarks"></a>Comentários

Para fazer referência à célula **QuickStyleFontMatrix** pelo nome, a partir de outra fórmula, pelo valor do atributo **N** de um elemento **Cell** ou por um programa que usa a propriedade **CellsU,** utilize: 
  
|||
|:-----|:-----|
| Nome da célula:  <br/> | QuickStyleFontMatrix  <br/> |
   
Para fazer referência à **célula QuickStyleFontMatrix** pelo índice a partir de um programa, use a propriedade **CellsSRC** com os seguintes argumentos: 
  
|||
|:-----|:-----|
| Índice da seção:  <br/> |**visSectionObject** <br/> |
| Índice de linha:  <br/> |**visRowQuickStyleProperties** <br/> |
| Índice de célula:  <br/> |**visQuickStyleFontMatrix** <br/> |
