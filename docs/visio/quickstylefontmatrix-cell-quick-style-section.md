# <a name="quickstylefontmatrix-cell-quick-style-section"></a>Célula QuickStyleFontMatrix (Seção Quick Style)

Determina o estilo da fonte para cada estilo rápido, como um inteiro de 1 a 6.
  
## <a name="remarks"></a>Comentários

Para fazer referência à célula **QuickStyleFontMatrix** pelo nome a partir de outra fórmula, pelo valor do atributo **N** de um elemento de **célula** ou um programa que usa a propriedade **CellsU** , utilize: 
  
|||
|:-----|:-----|
| Nome da célula:  <br/> | QuickStyleFontMatrix  <br/> |
   
Para obter uma referência à célula **QuickStyleFontMatrix** pelo índice a partir de um programa, use a propriedade **CellsSRC** com os seguintes argumentos: 
  
|||
|:-----|:-----|
| Índice da seção:  <br/> |**visSectionObject** <br/> |
| Índice da linha:  <br/> |**visRowQuickStyleProperties** <br/> |
| Índice da célula:  <br/> |**visQuickStyleFontMatrix** <br/> |
