# <a name="cos-function-visioshapesheet"></a>Função COS (VisioShapeSheet)

Retorna o cosseno de um ângulo. 
  
## <a name="syntax"></a>Sintaxe

COS( *angle*) 
  
### <a name="parameters"></a>Parâmetros

|**Name**|**Obrigatório/opcional**|**Tipo de dados**|**Descrição**|
|:-----|:-----|:-----|:-----|
| _ângulo_ <br/> |Obrigatório  <br/> |**Numérica** <br/> |O ângulo do qual obter o cosseno.  <br/> |
   
## <a name="remarks"></a>Comentários

A função COS usa radianos, a menos que você especifique uma unidade de medida diferente em *ângulo.* 
  
## <a name="example"></a>Exemplo

COS(45 graus) 
  
Retorna 0,7071.
