# <a name="endx-cell-1-d-endpoints-section"></a>Célula EndX (Seção 1-D Endpoints)

Representa o *x* -coordenadas do ponto de extremidade da forma 1D em relação à origem de seu pai. 
  
## <a name="remarks"></a>Comentários

Para fazer referência à célula EndX pelo nome, a partir de outra fórmula ou programa que usa a propriedade **CellsU**, utilize: 
  
|||
|:-----|:-----|
| Nome da célula:  <br/> | EndX  <br/> |
   
Para fazer referência à célula EndX pelo índice, a partir de um programa, use a propriedade **CellsSRC** com os seguintes argumentos: 
  
|||
|:-----|:-----|
| Índice da seção:  <br/> |**visSectionObject** <br/> |
| Índice da linha:  <br/> |**visRowXForm1D** <br/> |
| Índice da célula:  <br/> |**vis1DEndX** <br/> |
