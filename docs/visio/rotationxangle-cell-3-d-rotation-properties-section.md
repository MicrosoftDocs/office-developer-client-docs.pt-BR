# <a name="rotationxangle-cell-3-d-rotation-properties-section"></a>Célula RotationXAngle (Seção 3-D Rotation Properties)

Determina o ângulo de rotação ao longo do eixo x, em graus (0,0 - 359.9). 
  
## <a name="remarks"></a>Comentários

Para fazer referência à célula **RotationXAngle** pelo nome a partir de outra fórmula, pelo valor do atributo **N** de um elemento de **célula** ou um programa que usa a propriedade **CellsU** , utilize: 
  
|||
|:-----|:-----|
|Nome da célula:  <br/> |RotationXAngle  <br/> |
   
Para obter uma referência à célula **RotationXAngle** pelo índice a partir de um programa, use a propriedade **CellsSRC** com os seguintes argumentos: 
  
|||
|:-----|:-----|
|Índice da seção:  <br/> |**visSectionObject** <br/> |
|Índice da linha:  <br/> |**visRow3DRotationProperties** <br/> |
|Índice da célula:  <br/> |**visRotationXAngle** <br/> |
