# <a name="opentextwin-function"></a>Função OPENTEXTWIN

Abre o bloco de texto da forma para que o texto possa ser editado.
  
## <a name="syntax"></a>Sintaxe

OPENTEXTWIN( )
  
## <a name="remarks"></a>Comentários

Em versões anteriores do Visio, essa função aparece como _OPENTEXTWIN. As versões 4.0 e posteriores do Visio aceitam os dois estilos.
