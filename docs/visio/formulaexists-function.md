# <a name="formulaexists-function"></a>Função FORMULAEXISTS

Indica se a célula referenciada contém uma fórmula. 
  
## <a name="syntax"></a>Sintaxe

FORMULAEXISTS (* * *cellref* * *) 
  
### <a name="parameters"></a>Parâmetros

|**Name**|**Obrigatório/Opcional**|**Tipo de dados**|**Descrição**|
|:-----|:-----|:-----|:-----|
| _CellRef_ <br/> |Obrigatório  <br/> |**String** <br/> |A célula cuja presença de fórmula você deseja verificar.  <br/> |
   
## <a name="remarks"></a>Comentários

A função retorna 1 se a célula contiver uma fórmula; da FORMULAEXISTS Se ele não contiver uma fórmula, retornará zero (0).
