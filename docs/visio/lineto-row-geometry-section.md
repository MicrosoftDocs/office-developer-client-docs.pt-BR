# <a name="lineto-row-geometry-section"></a>Linha LineTo (Seção Geometry)

Contém *x* - e *y* -coordenadas do vértice final de um segmento de linha reta. 
  
Uma linha LineTo contém as células a seguir.
  
|**Célula**|**Descrição**|
|:-----|:-----|
|[X](x-cell-geometry-section.md) <br/> |*X* -coordenadas do vértice final de um segmento de linha reta.  <br/> |
|[Y](y-cell-geometry-section.md) <br/> |*Y* -coordenadas do vértice final de um segmento de linha reta.  <br/> |
