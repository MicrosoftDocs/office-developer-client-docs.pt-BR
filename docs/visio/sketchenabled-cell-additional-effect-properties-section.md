# <a name="sketchenabled-cell-additional-effect-properties-section"></a>Célula SketchEnabled (Seção Additional Effect Properties)

Determina se um efeito de esboço é exibido na forma ou não, como um valor booleano. 
  
## <a name="remarks"></a>Comentários

Para fazer referência à célula **SketchEnabled** pelo nome a partir de outra fórmula, pelo valor do atributo **N** de um elemento de **célula** ou um programa que usa a propriedade **CellsU** , utilize: 
  
|||
|:-----|:-----|
| Nome da célula:  <br/> | SketchEnabled  <br/> |
   
Para obter uma referência à célula **SketchEnabled** pelo índice a partir de um programa, use a propriedade **CellsSRC** com os seguintes argumentos: 
  
|||
|:-----|:-----|
| Índice da seção:  <br/> |**visSectionObject** <br/> |
| Índice da linha:  <br/> |**visRowOtherEffectProperties** <br/> |
| Índice da célula:  <br/> |**visSketchEnabled** <br/> |
