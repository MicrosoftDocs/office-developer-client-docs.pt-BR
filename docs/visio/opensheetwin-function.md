# <a name="opensheetwin-function"></a>Função OPENSHEETWIN

Abre a janela ShapeSheet em uma nova janela.
  
## <a name="syntax"></a>Sintaxe

OPENSHEETWIN( )
  
## <a name="remarks"></a>Comentários

Em versões anteriores do Visio, essa função aparece como _OPENSHEETWIN. As versões 4.0 e posteriores do Visio aceitam os dois estilos.
