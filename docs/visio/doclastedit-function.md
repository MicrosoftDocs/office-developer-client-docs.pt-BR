# <a name="doclastedit-function"></a>Função DOCLASTEDIT

Retorna, como valor de série, a data e a hora em que um documento foi editado pela última vez.
  
## <a name="syntax"></a>Sintaxe

DOCLASTEDIT()
  
## <a name="remarks"></a>Comentários

Use a função [FORMAT](format-function.md) para formatar o valor.
