|**Extensão de base** <br/> |IndexedRow_Type  <br/> |
   
## <a name="definition"></a>Definição

```XML
          <xs:complexType name="FieldRow_Type">
          
          <xs:complexContent>
          <xs:extension base="IndexedRow_Type">
          <xs:sequence>
    <xs:element name="Cell"  type="Cell_Type"

     maxOccurs="unbounded"

    </xs:element>

      </xs:extension>
      </xs:complexContent>
      </xs:complexType>
      
```
