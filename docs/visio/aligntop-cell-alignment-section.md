# <a name="aligntop-cell-alignment-section"></a>Célula AlignTop (Seção Alignment)

Determina a posição vertical, em relação à origem de seu pai, de uma guia horizontal ou ponto de guia ao qual a borda superior da forma está alinhada.
  
## <a name="remarks"></a>Comentários

Para fazer referência à célula AlignTop pelo nome, a partir de outra fórmula ou programa que usa a propriedade **CellsU**, utilize: 
  
|||
|:-----|:-----|
| Nome da célula:  <br/> | AlignTop  <br/> |
   
Para fazer referência à célula AlignTop pelo índice a partir de um programa, use a propriedade **CellsSRC** com os seguintes argumentos: 
  
|||
|:-----|:-----|
| Índice da seção:  <br/> |**visSectionObject** <br/> |
| Índice da linha:  <br/> |**visRowAlign** <br/> |
| Índice da célula:  <br/> |**visAlignTop** <br/> |
