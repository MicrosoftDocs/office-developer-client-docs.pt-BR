# <a name="additional-effect-properties-section"></a>Seção Additional Effect Properties

Mostra os efeitos adicionais aplicados à forma, incluindo efeitos de reflexão, brilho e bordas suaves. Esta seção também incluiu os valores para um efeito de Esboço.
  
## <a name="remarks"></a>Comentários

Você pode definir os efeitos de reflexão, brilho e bordas suaves na caixa de diálogo Formatar Forma **(Efeitos** de Estilos de   >  **Forma**  >  **Home).** Um efeito de Esboço pode ser aplicado a uma forma usando um dos temas de esboço (Efeitos de Temas de **Design)** ou fazendo alterações nos valores nas células  >    >  ShapeSheet correspondentes.
