# <a name="doccreation-function"></a>Função DOCCREATION

Retorna a data e hora de que um documento foi criado, como um valor de série.
  
## <a name="syntax"></a>Sintaxe

DOCCREATION ()
  
## <a name="remarks"></a>Comentários

Utilize a função FORMAT para formatar o valor.
