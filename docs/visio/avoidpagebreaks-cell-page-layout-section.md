# <a name="avoidpagebreaks-cell-page-layout-section"></a>Célula AvoidPageBreaks (Seção Page Layout)

Determina se as formas podem ser posicionadas sobre quebras de página quando estão alinhadas ou espaçadas de forma incremental, ou ambos.
  
## <a name="remarks"></a>Comentários

Para obter uma referência à célula AvoidPageBreaks pelo nome a partir de outra fórmula ou de um programa que use a propriedade **CellsU**, utilize: 
  
|||
|:-----|:-----|
|Nome da célula:  <br/> |AvoidPageBreaks  <br/> |
   
Para obter uma referência à célula AvoidPageBreaks pelo índice a partir de um programa, use a propriedade **CellsSRC** com os seguintes argumentos: 
  
|||
|:-----|:-----|
|Índice da seção:  <br/> |**visSectionObject** <br/> |
|Índice da linha:  <br/> |**visRowPageLayout** <br/> |
|Índice da célula:  <br/> |**visPLOAvoidPageBreaks** <br/> |
