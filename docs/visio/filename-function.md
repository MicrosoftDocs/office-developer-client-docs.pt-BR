# <a name="filename-function"></a>Função FILENAME

Retorna o nome do arquivo de um documento como uma cadeia de caracteres.
  
## <a name="syntax"></a>Sintaxe

FILENAME ()
  
### <a name="return-value"></a>Valor retornado

String
