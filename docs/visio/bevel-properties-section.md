# <a name="bevel-properties-section"></a>Seção Bevel Properties

Mostra os atributos para quaisquer efeitos de bisel aplicados à forma. 
  
## <a name="remarks"></a>Comentários

Você pode aplicar um bisel a uma forma no painel de ferramentas **Formatar forma** (**Home** > **Estilos de forma** > **bisel** > **Opções 3D**, ou por meio de alterações às células na seção **Propriedades de bisel** do ShapeSheet.
