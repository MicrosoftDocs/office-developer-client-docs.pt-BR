# <a name="bevel-properties-section"></a>Seção Bevel Properties

Mostra os atributos para quaisquer efeitos de bevel aplicados à forma. 
  
## <a name="remarks"></a>Comentários

Você pode aplicar um bevel  a uma forma no painel Formatar Forma **(** Home  >  **Shape Styles**  >  **Bevel**  >  **3D Options**, ou fazendo alterações nas células na seção **Bevel Properties** do ShapeSheet.
