# <a name="data2-function"></a>Função DATA2

Retorna o texto da caixa de **Data 2** na caixa de diálogo **Nome da forma** para uma forma (na guia **desenvolvedor** , clique o **Nome da forma** ), como uma cadeia de caracteres. 
  
## <a name="syntax"></a>Sintaxe

DATA2 ()
  
## <a name="remarks"></a>Comentários

Se a caixa estiver vazia, a função retornará uma cadeia de caracteres vazia.
