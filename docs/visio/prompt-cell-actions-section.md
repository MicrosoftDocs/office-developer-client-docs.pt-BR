# <a name="prompt-cell-actions-section"></a>Célula Prompt (Seção Actions)

No Visio 2002, esta célula não é mais utilizada.
  
## <a name="remarks"></a>Comentários

Em versões anteriores do Visio, esta célula especificava um prompt descritivo, exibido na barra de status, quando o comando de atalho correspondente era selecionado.
