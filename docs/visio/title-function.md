# <a name="title-function"></a>Função TITLE

Retorna o texto do **campo Título** das propriedades de um documento. 
  
## <a name="syntax"></a>Sintaxe

TITLE ()
  
### <a name="return-value"></a>Valor de retorno

Cadeia de caracteres
