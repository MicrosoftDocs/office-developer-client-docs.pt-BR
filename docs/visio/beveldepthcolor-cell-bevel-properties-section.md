# <a name="beveldepthcolor-cell-bevel-properties-section"></a>Célula BevelDepthColor (Seção Bevel Properties)

Determina a cor de profundidade do bisel, como um valor RGB ou conforme determinado pelo tema ativo.
  
## <a name="remarks"></a>Comentários

Para fazer referência à célula **BevelDepthColor** pelo nome a partir de outra fórmula, pelo valor do atributo **N** de um elemento de **célula** ou um programa que usa a propriedade **CellsU** , utilize: 
  
|||
|:-----|:-----|
| Nome da célula:  <br/> | BevelDepthColor  <br/> |
   
Para obter uma referência à célula **BevelDepthColor** pelo índice a partir de um programa, use a propriedade **CellsSRC** com os seguintes argumentos: 
  
|||
|:-----|:-----|
| Índice da seção:  <br/> |**visSectionObject** <br/> |
| Índice da linha:  <br/> |**visRowBevelProperties** <br/> |
| Índice da célula:  <br/> |**visBevelDepthColor** <br/> |
