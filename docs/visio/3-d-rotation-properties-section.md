# <a name="3-d-rotation-properties-section"></a>Seção 3-D Rotation Properties

Mostra os valores atuais do ângulo de rotação de uma forma ao longo dos eixos x-, y-e z.
  
> [!NOTE]
> A seção 3-D Rotation Properties não inclui células que controlam os efeitos de bisel da forma. Para definir um efeito de bisel, altere os valores das células na seção [Propriedades de bisel](bevel-properties-section.md) . 
  
## <a name="remarks"></a>Comentários

Você pode alterar os valores de rotação 3D na caixa de **diálogo Formatar forma** (**estilos** > de forma**inicial** > **efeitos** > de**3D de rotação** ou fazendo uma alteração em uma das células nas propriedades de **rotação 3D **seção.
