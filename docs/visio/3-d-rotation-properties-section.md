# <a name="3-d-rotation-properties-section"></a>Seção 3-D Rotation Properties

Mostra os valores atuais do ângulo de rotação de uma forma ao longo dos eixos x, y e z.
  
> [!NOTE]
> A seção Propriedades de Rotação 3D não inclui células que controlam efeitos de bevel de forma. Para definir um efeito de bevel, altere os valores das células na seção [Bevel Properties.](bevel-properties-section.md) 
  
## <a name="remarks"></a>Comentários

Você pode alterar os valores de  rotação 3D na caixa de diálogo Formatar Forma **(** Rotação 3D dos Estilos de Forma Home ou fazendo uma alteração em uma das células na seção Propriedades de Rotação  >    >    >   **3D.**
