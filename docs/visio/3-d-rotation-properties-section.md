# <a name="3-d-rotation-properties-section"></a>Seção 3-D Rotation Properties

Mostra os valores atuais das ângulo de uma forma de rotação ao longo de x, y-e eixos z.
  
> [!NOTE]
> Seção de propriedades de rotação 3D não inclui as células que controlam os efeitos de bisel de forma. Para definir um efeito de bisel, altere os valores das células na seção [Propriedades de bisel](bevel-properties-section.md) . 
  
## <a name="remarks"></a>Comentários

Você pode alterar os valores de rotação 3D na caixa de diálogo **Formatar forma** (**Home** > **Estilos de forma** > **efeitos** > **Rotação 3D** ou por fazer uma alteração em uma das células no **Propriedades de rotação 3D **seção.
