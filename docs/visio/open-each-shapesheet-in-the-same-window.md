# <a name="open-each-shapesheet-in-the-same-window"></a>Abrir cada ShapeSheet na mesma janela

1. Clique na guia **arquivo** , em **Opções**e em **avançado**.
    
2. Em **Geral**, selecione **Abrir cada ShapeSheet na mesma janela**.
    
3. Clique em **OK**.
