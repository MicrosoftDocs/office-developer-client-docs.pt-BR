# <a name="project-conceptual-and-how-to-articles"></a>Artigos conceituais e de orientação do Project

Encontre links para informações nesta seção sobre as principais partes da plataforma do Project 2013, introdução ao desenvolvimento do CSOM usando o JavaScript e o .NET Framework, suplementos do painel de tarefas no Project Professional 2013, a interface do Project Server e o Serviço de relatórios OData.
  
O Project Server 2013 é baseado na estrutura de serviço Web da interface do Project Server (PSI) introduzida no Office Project Server 2007. A plataforma do Project Server 2013 é criada com o .NET Framework 4. O Project Web App requer o SharePoint Server 2013 como base para a funcionalidade de novos recursos, como o modelo de objeto do lado do cliente (CSOM) para acessar o Project Server na nuvem, a versão 4 do Windows Workflow Foundation (WF4) e o serviço OData para o tabelas de banco de dados de relatórios. O Project Professional 2013 pode usar os novos suplementos do Office no painel de tarefas.
