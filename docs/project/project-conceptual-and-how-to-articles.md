# <a name="project-conceptual-and-how-to-articles"></a>Artigos de conceituais e instruções do Project

Encontre links para informações nesta seção sobre as principais partes da plataforma do Project 2013, Introdução ao desenvolvimento de CSOM usando JavaScript e o .NET Framework, tarefa painel complementos no Project Professional 2013, a Interface do Project Server e o Serviço de relatório de OData.
  
Project Server 2013 baseia-se na estrutura do serviço web do Project Server Interface (PSI) introduzida no Office Project Server 2007. A plataforma do Project Server 2013 é criada com o .NET Framework 4. Project Web App requer o SharePoint Server 2013 como base para a funcionalidade dos novos recursos, como o modelo de objeto do cliente (CSOM) para acesso ao Project Server na nuvem, versão 4 do Windows Workflow Foundation (WF4) e o serviço OData para o tabelas de banco de dados de relatórios. Project Professional 2013 pode usar o painel de tarefas Novo suplementos do Office.
