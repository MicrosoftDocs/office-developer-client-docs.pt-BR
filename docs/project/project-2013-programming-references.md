# <a name="project-2013-programming-references"></a>Referências de programação do Project 2013

Veja uma visão geral da referência da Interface do Project Server (PSI), como usar a interface ASMX e interface WCF do PSI, informações sobre os códigos de erro do Project Server e uma referência para o serviço OData do **ProjectData**. 
  
> [!NOTE]
> Para as referências principais para PSI e o modelo de objeto do lado cliente (CSOM) no Project Server 2013, confira as seções a seguir: [Biblioteca de classe e referência do serviço Web](https://msdn.microsoft.com/library/ef1830e0-3c9a-4f98-aa0a-5556c298e7d1%28Office.15%29.aspx) e [Referência da API JavaScript do Project Server 2013](javascript-library-and-rest-reference-for-project-server-2013.md). > Para as referências de Suplementos do Office, que incluem a referência de JavaScript para aplicativos de painel de tarefas do Project 2013, confira [API JavaScript para Office](https://msdn.microsoft.com/library/fp142185.aspx). > Para a referência de VBA para o Project Standard 2013 e o Project Professional 2013, confira [Referência para desenvolvedores do VBA do Project 2013](https://msdn.microsoft.com/library/jj235035.aspx). > A referência para os modos de exibição e tabelas de relatório no banco de dados do Project Server 2013 local ainda não está disponível. 
  
## <a name="in-this-section"></a>Nesta seção

[Visão geral de referência do Project PSI](project-psi-reference-overview.md) Descreve os assemblies e namespaces na referência PSI do Project Server 2013. 
  
[Pré-requisitos para amostras de código baseado em ASMX no Project](prerequisites-for-asmx-based-code-samples-in-project.md) Descreve como usar amostras de código na referência PSI criada usando serviços Web ASMX nos seus aplicativos de teste. 
  
[Pré-requisitos para amostras de código baseadas em WCF no Project](prerequisites-for-wcf-based-code-samples-in-project.md) Descreve como usar na referência PSI amostras de código criadas usando os serviços do Windows Communication Foundation (WCF) em seus aplicativos de teste. 
  
[Códigos de erro do Project Server](project-server-error-codes.md) Lista os códigos e descrições de erros do Project Server por área funcional. 
  
[ProjectData – Referência do serviço OData do Project](https://msdn.microsoft.com/library/office/jj163015.aspx) Inclui uma visão geral do serviço OData para dados de relatório do Project Server, uma introdução ao uso de feeds OData com consultas LINQ e referências para os metadados XML no serviço **ProjectData**.
