Encontre conteúdo de instruções, artigos técnicos, referências e postagens de blogs para soluções de desenvolvimento e personalização dos aplicativos de cliente de área de trabalho do OneNote.
  
> [!NOTE]
> As APIs descritas nesta documentação destinam-se apenas para soluções de cliente de área de trabalho do OneNote Win32 em cenários não conectados. Para cenários conectados, use as APIs de serviço recomendadas do OneNote. Para saber mais, visite [dev.onenote.com](https://dev.onenote.com/).
