# <a name="getting-started-with-the-excel-xll-sdk"></a>Introdução ao Excel XLL SDK

**Aplica-se a**: Excel 2013 | Office 2013 | Visual Studio 
  
O Microsoft Excel 2013 XLL Software Development Kit (SDK) fornece informações para ajudá-lo a começar a criar complementos DLL para Excel 2013 que usam a API de C. Esta seção fornece informações sobre a API de C e também links para documentação de referência para ajudá-lo a começar a desenvolver o Excel 2013 XLL.
  
## <a name="in-this-section"></a>Nesta seção

- [Programação com a API de C](programming-with-the-c-api-in-excel.md)no Excel: introduz a API de C e discute como ela se compara a outras opções de programação no Excel 2013.
    
- [Novidades na API de C para Excel:](what-s-new-in-the-c-api-for-excel.md)descreve as atualizações da API de C para Excel 2013.
    
- [Problemas conhecidos no desenvolvimento de XLL do Excel:](known-issues-in-excel-xll-development.md)discute problemas encontrados na programação XLL do Excel.
    
## <a name="related-sections"></a>Seções relacionadas
