# <a name="welcome-to-the-excel-software-development-kit"></a>Bem-vindo ao Software Development Kit do Excel

 **Aplica-se a**: Excel 2013 | Office 2013 | Visual Studio 
  
Bem-vindo à documentação do Excel 2013 XLL Software Development Kit (SDK). Essa referência contém visões gerais conceituais, tarefas de programação e exemplos para ajudar você a desenvolver Microsoft Excel 2013 XLLs.
  
Revisado em: Novembro de 2012
  
Baixe o [Excel 2013 XLL SDK](https://go.microsoft.com/fwlink/?LinkID=251082&amp;clcid=0x409).
  
O Excel 2013 XLL SDK inclui:
  
- **Interface de programação de aplicativo (API) C** – inclui arquivos de origem e de cabeçalho que permitem que os DLLs acessem a funcionalidade do Excel 2013 e uma descrição da interface que uma DLL deve expor para trabalhar com o Gerenciador de Suplementos do Excel.
    
- **Projetos do Microsoft Visual Studio** – Inclui código fonte C/C++ e mostra como usar a API do C. Essas amostras de projetos servem como exemplos e como ponto de partida para o desenvolvimento do seu próprio suplemento.
    
A documentação SDK contém as seguintes seções:
  
- [Introdução ao Excel XLL SDK](getting-started-with-the-excel-xll-sdk.md)
    
- [Desenvolvimento de XLLs do Excel](developing-excel-xlls.md)
    
- [Developing Excel Cluster Connectors](developing-excel-cluster-connectors.md)
    
- [Excel XLL SDK API Function Reference](excel-xll-sdk-api-function-reference.md)
    
## <a name="functionality-not-covered"></a>Funcionalidade não coberta

Os seguintes assuntos não serão abordados:
  
- Desenvolvendo comandos e funções definidas pelo usuário em planilhas de macro do Excel (XLM).
    
- Criando funções definidas pelo usuário em DLLs que controlam o fluxo de execução de uma macro XLM.
    
    Essas funções retornam um tipo especial de dados de controle de fluxo, que não está descrito nesta documentação.
