# <a name="raiseerror-macro-action-access-custom-web-app"></a>Ação de macro Gerarerro (aplicativo Web personalizado do Access)

A ação **gerarerro** exibe uma janela pop-up que contém uma mensagem de erro especificada. 
  
> [!IMPORTANT]
> [!IMPORTANTE] A Microsoft não recomenda mais criar e usar aplicativos Web do Access no SharePoint. Como alternativa, use o [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) para criar soluções de negócios sem código para a Web e dispositivos móveis. 
  
> [!NOTE]
> A ação **GerarErro** está disponível somente em Macros de Dados. 
  
## <a name="setting"></a>Configuração

A ação **gerarerro** tem o argumento a seguir. 
  
|**Argumento**|**Descrição**|
|:-----|:-----|
| _Descrição do erro_ <br/> |Uma expressão de cadeia de caracteres que descreve o erro.  <br/> |
   
## <a name="remarks"></a>Comentários

Quando a ação **gerarerro** é chamada, todas as operações da transação atual são revertidas.
