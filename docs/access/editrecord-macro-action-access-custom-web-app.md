# <a name="editrecord-macro-action-access-custom-web-app"></a>Ação de macro Editarregistro (aplicativo Web personalizado do Access)

Coloca o registro atual em modo de edição.
  
> [!IMPORTANT]
> A Microsoft não recomenda mais criar e usar aplicativos Web do Access no SharePoint. Como alternativa, use o [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) para criar soluções de negócios sem código para a Web e dispositivos móveis. 
  
## <a name="syntax"></a>Sintaxe

 **EditRecord**
  
## <a name="remarks"></a>Comentários

Use a ação **SaveRecord** para salvar as alterações feitas no registro ou a ação **UndoRecord** para descartar as alterações. 
  
A ação **editarregistro** é equivalente a selecionar o botão **Editar** na barra de ações.
