# <a name="editrecord-macro-action-access-custom-web-app"></a>Ação de Macro EditarRegistro (aplicativo da web personalizado do Access)

Coloca o registro atual no modo de edição.
  
> [!IMPORTANT]
> [!IMPORTANTE] A Microsoft não recomenda mais criar e usar aplicativos Web do Access no SharePoint. Como alternativa, use o [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) para criar soluções de negócios sem código para a Web e dispositivos móveis. 
  
## <a name="syntax"></a>Syntax

 **EditarRegistro**
  
## <a name="remarks"></a>Coment�rios

Use a ação **SaveRecord** para salvar as alterações feitas no registro, ou a ação **UndoRecord** para descartar as alterações. 
  
A ação **EditarRegistro** é equivalente à seleção no botão **Editar** na barra de ação.
