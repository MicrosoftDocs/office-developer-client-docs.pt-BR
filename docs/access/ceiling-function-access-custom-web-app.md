# <a name="ceiling-function-access-custom-web-app"></a>Função teto (aplicativo Web personalizado do Access)

Retorna o menor inteiro maior ou igual à expressão numérica especificada.
  
> [!IMPORTANT]
> A Microsoft não recomenda mais criar e usar aplicativos Web do Access no SharePoint. Como alternativa, use o [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) para criar soluções de negócios sem código para a Web e dispositivos móveis. 
  
## <a name="syntax"></a>Sintaxe

**Teto** (*Numericé*) 
  
A função teto contém o argumento a seguir.
  
|**Nome do argumento**|**Descrição**|
|:-----|:-----|
| *Numericé*  <br/> |Uma expressão numérica  <br/> |
