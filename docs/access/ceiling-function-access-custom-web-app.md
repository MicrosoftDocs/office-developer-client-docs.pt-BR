# <a name="ceiling-function-access-custom-web-app"></a>Função Ceiling (aplicativo da web personalizado do Access)

Retorna o menor inteiro maior ou igual a, a expressão numérica especificada.
  
> [!IMPORTANT]
> [!IMPORTANTE] A Microsoft não recomenda mais criar e usar aplicativos Web do Access no SharePoint. Como alternativa, use o [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) para criar soluções de negócios sem código para a Web e dispositivos móveis. 
  
## <a name="syntax"></a>Syntax

**Teto** (*NumericExpression*) 
  
A função teto contém os seguintes argumentos.
  
|**Nome do argumento**|**Descrição**|
|:-----|:-----|
| *NumericExpression*  <br/> |Uma expressão numérica  <br/> |
