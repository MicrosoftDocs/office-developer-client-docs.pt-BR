# <a name="connectionupdatable-property-dao"></a>Propriedade Connection.Updatable (DAO)


**Aplica-se a**: Access 2013, o Office 2013

Retorna um valor que indica se você pode alterar o objeto DAO. **Boolean** somente leitura. Somente leitura.

## <a name="syntax"></a>Sintaxe

*expressão* . Atualizável

*expressão* Uma expressão que retorna um objeto de **Conexão** .
