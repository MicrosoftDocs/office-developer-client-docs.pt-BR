# <a name="procedures-collection-adox"></a>Coleção Procedures (ADOX)


**Aplica-se a**: Access 2013 | Office 2013

Contém todos os objetos [Procedure](procedure-object-adox.md) de um catálogo.

## <a name="remarks"></a>Comentários

O método [Append](append-method-adox-procedures.md) de uma coleção **Procedures** é exclusivo de ADOX. Você pode:

  - Adicionar um novo procedimento à coleção com o método **Append**.

As propriedades e os métodos restantes são padrão das coleções do ADO. Você pode:

  - Acessar um procedimento na coleção com a propriedade [Item](item-property-ado.md).

  - Retornar o número de procedimentos contidos na coleção com a propriedade [Count](count-property-ado.md).

  - Remover um procedimento da coleção com o método [Delete](delete-method-adox-collections.md).

  - Atualizar os objetos da coleção para refletir o atual esquema do banco de dados com o método [Refresh](refresh-method-ado.md).
