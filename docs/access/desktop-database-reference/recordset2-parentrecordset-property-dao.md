# <a name="recordset2parentrecordset-property-dao"></a>Propriedade Recordset2. ParentRecordset (DAO)


**Aplica-se ao:** Access 2013, Office 2013 

Retorna o **Recordset** pai do conjunto de registros especificado. Somente leitura.

## <a name="version-information"></a>Informação da versão

Versão adicionada: Access 2007

## <a name="syntax"></a>Sintaxe

*expressão* . ParentRecordset

*expressão* Uma variável que representa um objeto **Recordset2** .

## <a name="remarks"></a>Comentários

A propriedade **ParentRecordset** retornará **Null** se o conjunto de registros especificado não representar um campo com vários valores.
