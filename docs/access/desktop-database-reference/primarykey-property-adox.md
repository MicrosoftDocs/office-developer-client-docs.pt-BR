# <a name="primarykey-property-adox"></a>Propriedade PrimaryKey (ADOX)


**Aplica-se a**: Access 2013, o Office 2013

Indica se o índice representa a chama primária na tabela.

## <a name="settings-and-return-values"></a>Configurações e valores de retorno

Define e retorna um valor **Boolean**.

## <a name="remarks"></a>Comentários

O valor padrão é **False**.

Esta propriedade é somente leitura em objetos [Index](index-object-adox.md) já acrescentados a uma coleção.
