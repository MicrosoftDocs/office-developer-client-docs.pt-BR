# <a name="recordset2lastupdated-property-dao"></a>Propriedade Recordset2.LastUpdated (DAO)


**Aplica-se a**: Access 2013, o Office 2013

Retorna a data e a hora da alteração mais recente feita em uma tabela base. **Variant** somente leitura.

## <a name="syntax"></a>Sintaxe

*expressão* . LastUpdated

*expressão* Uma variável que representa um objeto **Recordset2** .

## <a name="remarks"></a>Comentários

As configurações de data e hora derivadas do computador no qual a tabela base foi criada ou atualizada pela última vez.
