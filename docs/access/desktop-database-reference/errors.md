# <a name="errors"></a>Erros

**Aplica-se a**: Access 2013, o Office 2013

Qualquer operação envolvendo objetos ADO pode gerar um ou mais erros do provedor. Na ocorrência de cada erro, um ou mais objetos **Error** são incluídos na coleção **Errors** do objeto **Connection**. Para obter detalhes sobre o tratamento de erros no seu aplicativo ADO e avisos, consulte [Capítulo 6: tratamento de erros](chapter-6-error-handling.md).

Um mecanismo individual pode gerar erros de aplicativo. Por exemplo, no Visual Basic, o objeto **Err** conterá os erros no nível do aplicativo.
