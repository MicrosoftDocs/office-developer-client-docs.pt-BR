# <a name="caption-property-ado-md"></a>Propriedade Caption (ADO MD)


**Aplica-se a**: Access 2013 | Office 2013

Indica a legenda de texto a ser usada para a exibição de um objeto [Level](level-object-ado-md.md) ou [Member](member-object-ado-md.md).

## <a name="return-values"></a>Valores de retorno

Retorna um objeto **String** e é somente leitura.
