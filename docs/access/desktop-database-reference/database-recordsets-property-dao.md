# <a name="databaserecordsets-property-dao"></a>Propriedade Database.Recordsets (DAO)


**Aplica-se a**: Access 2013, o Office 2013

Retorna uma coleção **Recordsets** que contém todos os recordsets abertos para o banco de dados especificado. Somente leitura.

## <a name="syntax"></a>Sintaxe

*expressão* . Conjuntos de registros

*expressão* Uma variável que representa um objeto de **banco de dados** .
