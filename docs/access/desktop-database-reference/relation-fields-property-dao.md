# <a name="relationfields-property-dao"></a>Propriedade Relation.Fields (DAO)


**Aplica-se ao:** Access 2013, Office 2013

Retorna uma coleção **Fields** que representa todos os objetos **Field** armazenados para o objeto especificado. Somente leitura.

## <a name="syntax"></a>Sintaxe

*expressão* . Campos

*expressão* Uma variável que representa um **objeto Relation** .
