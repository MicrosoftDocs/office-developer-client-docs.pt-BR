# <a name="databaseupdatable-property-dao"></a>Propriedade Database.Updatable (DAO)


**Aplica-se a**: Access 2013 | Office 2013

Retorna um valor que indica se você pode alterar um objeto DAO. **Boolean** somente leitura.

## <a name="syntax"></a>Sintaxe

*expressão* . Atualizável

*expressão* Uma variável que representa um objeto de **banco de dados** .
