# <a name="indexfields-property-dao"></a>Propriedade Index.Fields (DAO)


**Aplica-se a**: Access 2013, o Office 2013

Retorna uma coleção **Fields** que representa todos os objetos **Field** armazenados para o objeto especificado. Leitura/gravação.

## <a name="syntax"></a>Sintaxe

*expressão* . Campos

*expressão* Uma variável que representa um objeto **Index** .
