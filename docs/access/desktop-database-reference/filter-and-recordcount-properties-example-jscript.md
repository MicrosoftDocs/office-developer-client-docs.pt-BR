# <a name="filter-and-recordcount-properties-example-jscript"></a>Exemplo das propriedades Filter e RecordCount (JScript)


**Aplica-se a**: Access 2013 | Office 2013

Este exemplo abre um **Recordset** na tabela Companies do banco de dados Northwind e, em seguida, usa a propriedade [Filter](filter-property-ado.md) para limitar os registros visíveis àqueles em que o campo CompanyName começa com a letra D. Recorte e cole o código a seguir no Bloco de Notas ou em outro editor de texto e salve-o como **FilterJS.asp**.
