# <a name="using-ado-with-microsoft-visual-j"></a>Usando o ADO com o Microsoft Visual J++


**Aplica-se a**: Access 2013 | Office 2013

Você pode implementar o ADO usando o Visual J++ das maneiras a seguir:

  - Com o Visual J++ 6.0 (ou posterior), use o [ADO for Windows Foundation Classes](ado-wfc-programming.md).

  - Com o Visual J++ 1.x, use o [Java Type Library Wizard](using-the-java-type-library-wizard.md).

  - Use os utilitários do [Microsoft SDK for Java](using-the-microsoft-sdk-for-java.md).

Para obter mais informações sobre como usar o ADO com o Visual J++, consulte os tópicos a seguir:

  - [Invólucros de classe ADO Java](ado-java-class-wrappers.md)

  - [Índice da sintaxe do ADO/WFC](https://msdn.microsoft.com/library/jj250066\(v=office.15\))
