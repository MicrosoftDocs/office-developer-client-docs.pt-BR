# <a name="querydefsrefresh-method-dao"></a>Método QueryDefs.Refresh (DAO)


**Aplica-se a**: Access 2013, o Office 2013

Não aceito para este objeto.

## <a name="syntax"></a>Sintaxe

*expressão* . Atualizar

*expressão* Uma variável que representa um objeto **QueryDefs** .

## <a name="remarks"></a>Comentários

Você não pode usar o método **Refresh** com as coleções que não são persistentes, como as coleções **Connections**, **Databases**, **Recordsets**, **Workspaces** ou **QueryDefs** de um objeto **Connection**.
