# <a name="views-collection-adox"></a>Coleção Views (ADOX)


**Aplica-se a**: Access 2013, o Office 2013

Contém todos os objetos [View](view-object-adox.md) de um catálogo.

## <a name="remarks"></a>Comentários

O método [Append](append-method-adox-views.md) de uma coleção **Views** é exclusivo do ADOX. Você pode:

  - Adicionar um novo modo de exibição à coleção com o método **Append**.

As propriedades e os métodos restantes são padrão das coleções do ADO. Você pode:

  - Acessar um modo de exibição na coleção com a propriedade [Item](item-property-ado.md).

  - Retornar o número de modos de exibição contidos na coleção com a propriedade [Count](count-property-ado.md).

  - Remover um modo de exibição da coleção com o método [Delete](delete-method-adox-collections.md).

  - Atualizar os objetos da coleção para refletir o atual esquema do banco de dados com o método [Refresh](refresh-method-ado.md).
