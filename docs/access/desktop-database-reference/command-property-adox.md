# <a name="command-property-adox"></a>Propriedade Command (ADOX)


**Aplica-se a**: Access 2013 | Office 2013

Especifica um objeto [Command](command-object-ado.md) do ADO que pode ser usado para criar ou executar o procedimento.

## <a name="settings-and-return-values"></a>Configurações e valores de retorno

Define ou retorna um objeto **Command** do ADO.

## <a name="remarks"></a>Comentários

Ocorrerá um erro durante a obtenção e definição dessa propriedade se o provedor não oferecer suporte a comandos persistentes.
