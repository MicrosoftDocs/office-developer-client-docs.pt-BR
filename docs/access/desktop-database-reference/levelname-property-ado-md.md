# <a name="levelname-property-ado-md"></a>Propriedade LevelName (ADO MD)


**Aplica-se a**: Access 2013 | Office 2013

Indica o nome do nível de um membro.

## <a name="return-values"></a>Valores de retorno

Retorna um objeto **String** e é somente leitura.

## <a name="remarks"></a>Comentários

Use a propriedade **LevelName** para recuperar o nome do nível ao qual pertence um membro. Isso corresponde à propriedade [Name](name-property-ado-md.md) de um objeto [Level](level-object-ado-md.md).
