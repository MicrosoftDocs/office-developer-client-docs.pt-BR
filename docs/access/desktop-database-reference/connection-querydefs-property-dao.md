# <a name="connectionquerydefs-property-dao"></a>Propriedade Connection.QueryDefs (DAO)


**Aplica-se a**: Access 2013, o Office 2013

Retorna uma coleção **QueryDefs** que contém todos os objetos **QueryDef** da conexão especificada. Somente leitura.

## <a name="syntax"></a>Sintaxe

*expressão* . QueryDefs

*expressão* Uma variável que representa um objeto de **Conexão** .
