# <a name="understanding-recordset-structure"></a>Noções básicas sobre a estrutura do Recordset


**Aplica-se a**: Access 2013 | Office 2013

Todos **Recordset** têm uma coleção **Fields** composta por um ou mais objetos **Field**. Um objeto **Field** geralmente representa uma coluna de tabela. Os tópicos a seguir explicarão como navegar pela coleção **Fields** e obter informações sobre cada campo. Em seguida, eles abordarão os tipos de informações que estão disponíveis para você através do objeto **Field** e como usá-las.

  - [A coleção Fields](the-fields-collection.md)

  - [O objeto Field](the-field-object.md)

  - [Trabalhando com Recordsets](working-with-recordsets.md)
