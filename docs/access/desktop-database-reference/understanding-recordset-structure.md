# <a name="understanding-recordset-structure"></a>Noções básicas sobre a estrutura do Recordset

**Aplica-se a**: Access 2013, o Office 2013

Todos **Recordset** têm uma coleção **Fields** composta por um ou mais objetos **Field**. Um objeto **Field** geralmente representa uma coluna de tabela. 

Os tópicos a seguir explicam como navegar por meio da coleção de **campos** e obter informações sobre cada campo. Então, eles discutem que tipo de informação é disponibilizado para você pelo objeto **Field** e como usá-lo.

- [Coleção Fields](the-fields-collection.md)
- [Objeto Field](the-field-object.md)
- [Trabalhando com Recordsets](working-with-recordsets.md)
