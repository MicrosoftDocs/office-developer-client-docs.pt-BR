<span data-ttu-id="bb38e-p104">O Java Type Library Wizard converte os tipos enumerados, encontrados em qualquer biblioteca de tipos específica, no tipo INT (inteiro). ```vb

<span data-ttu-id="bb38e-p105">Também seria possível herdar do invólucro de tipo enumerado gerado pelo Java Type Library Wizard. Nesse caso, você poderia remover "msado15." da sintaxe.
