# <a name="view-an-activex-control-about-box"></a>Exibir a caixa sobre de um controle ActiveX

**Aplica-se a**: Access 2013, o Office 2013

Para exibir uma caixa **sobre** mostrando informações de versão e os direitos autorais para um controle ActiveX:

1. Escolha a propriedade **sobre** caixa na folha de propriedades do Microsoft Access.

2. Escolha o botão **Construir** , à direita da caixa da propriedade.
