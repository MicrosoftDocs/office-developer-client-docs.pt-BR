# <a name="view-an-activex-controls-about-box"></a>Exibir um controle ActiveX sobre caixa

**Aplica-se a**: Access 2013 | Office 2013

Para exibir uma caixa **Sobre** que mostra a versão e as informações sobre copyright de um controle ActiveX, clique na caixa de propriedades **Sobre** da folha de propriedades do Microsoft Access. Em seguida, clique no botão **Criar** à direita da caixa de propriedades.

# <a name="view-an-activex-control-about-box"></a>Exibir um controle ActiveX sobre caixa

**Aplica-se a**: Access 2013 | Office 2013

Para exibir uma caixa **sobre** mostrando informações de versão e os direitos autorais para um controle ActiveX:

1. Escolha a propriedade **sobre** caixa na folha de propriedades do Microsoft Access.

2. Escolha o botão **Construir** , à direita da caixa da propriedade.
