# <a name="connectionname-property-dao"></a>Propriedade Connection.Name (DAO)


**Aplica-se ao:** Access 2013, Office 2013

Retorna o nome de um **[Connection](connection-object-dao.md)**.

## <a name="syntax"></a>Sintaxe

*expressão* . Tdomínio

*expressão* Uma variável que representa um objeto **Connection** .

## <a name="remarks"></a>Comentários

Use uma propriedade **Name** do objeto com a instrução em código **Dim** do Visual Basic for Applications para criar outras instâncias do objeto.
