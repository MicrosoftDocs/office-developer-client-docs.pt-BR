# <a name="connectiontransactions-property-dao"></a>Propriedade Connection.Transactions (DAO)


**Aplica-se a**: Access 2013, o Office 2013

Retorna um valor que indica se um objeto aceita transactions. **Boolean** somente leitura.

## <a name="syntax"></a>Sintaxe

*expressão* . Transações

*expressão* Uma variável que representa um objeto de **Conexão** .
