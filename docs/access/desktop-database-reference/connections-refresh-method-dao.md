# <a name="connectionsrefresh-method-dao"></a>Método Connections.Refresh (DAO)


**Aplica-se a**: Access 2013, o Office 2013

Não aceito para a coleção **[Connections](connections-collection-dao.md)**.

Você não pode usar o método **Refresh** com as coleções que não são persistentes, como as coleções **Connections**, **Databases**, **Recordsets**, **Workspaces** ou **QueryDefs** de um objeto **Connection**.
