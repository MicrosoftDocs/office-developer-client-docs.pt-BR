# <a name="indexes-collection-adox"></a>Coleção Indexes (ADOX)


**Aplica-se a**: Access 2013, o Office 2013

Contém todos os objetos [Index](index-object-adox.md) de uma tabela.

## <a name="remarks"></a>Comentários

O método [Append](append-method-adox-indexes.md) de uma coleção **Indexes** é exclusivo de ADOX. Você pode:

  - Adicionar um novo índice à coleção com o método **Append**.

As propriedades e métodos restantes são padrão das coleções do ADO. Você pode:

  - Acessar um índice na coleção com a propriedade [Item](item-property-ado.md).

  - Retornar o número de índices contidos na coleção com a propriedade [Count](count-property-ado.md).

  - Remover um índice da coleção com o método [Delete](delete-method-adox-collections.md).

  - Atualizar os objetos da coleção para refletir o atual esquema do banco de dados com o método [Refresh](refresh-method-ado.md).
