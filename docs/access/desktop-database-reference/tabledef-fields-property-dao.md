# <a name="tabledeffields-property-dao"></a>Propriedade TableDef.Fields (DAO)


**Aplica-se a**: Access 2013, o Office 2013

Retorna uma coleção **Fields** que representa todos os objetos **Field** armazenados para o objeto especificado. Somente leitura.

## <a name="syntax"></a>Sintaxe

*expressão* . Campos

*expressão* Uma variável que representa um objeto **TableDef** .
