# <a name="definedsize-property-adox"></a>Propriedade DefinedSize (ADOX)


**Aplica-se a**: Access 2013, o Office 2013

Indica o tamanho máximo definido para a coluna.

## <a name="settings-and-return-values"></a>Configurações e valores de retorno

Define e retorna um valor **Long** que é o tamanho máximo em caracteres dos valores de dados.

## <a name="remarks"></a>Comentários

O valor padrão é zero (0).

Esta propriedade é somente leitura em objetos [Column](column-object-adox.md) já acrescentados a uma coleção.
