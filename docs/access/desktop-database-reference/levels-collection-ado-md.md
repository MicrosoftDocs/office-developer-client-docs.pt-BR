# <a name="levels-collection-ado-md"></a>Coleção Levels (ADO MD)


**Aplica-se ao:** Access 2013, Office 2013

Contém os objetos [Level](level-object-ado-md.md) que formam uma hierarquia.

## <a name="remarks"></a>Comentários

**Levels** é uma coleção padrão do ADO. Com as propriedades e os métodos de uma coleção, você pode executar os seguintes procedimentos:

  - Obter o número de objetos na coleção com a propriedade [Count](count-property-ado.md).

  - Retornar um objeto da coleção com a propriedade padrão [Item](item-property-ado.md).

  - Atualizar os objetos na coleção do provedor com o método [Refresh](refresh-method-ado.md).
