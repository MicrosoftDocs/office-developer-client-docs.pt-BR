# <a name="version-property-ado"></a>Propriedade Version (ADO)


**Aplica-se a**: Access 2013 | Office 2013

Indica o número da versão do ADO.

## <a name="return-value"></a>Valor de retorno

Retorna um valor **String** que indica a versão.

## <a name="remarks"></a>Comentários

Utilize a propriedade **Version** para retornar o número da versão da implementação do ADO.

A versão do provedor estará disponível como uma propriedade dinâmica na coleção [Properties](properties-collection-ado.md).
