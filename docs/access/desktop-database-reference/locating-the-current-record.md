# <a name="locating-the-current-record"></a>Localizando o registro atual


**Aplica-se a**: Access 2013 | Office 2013

A posição atual do cursor no **Recordset** delineia a posição do registro atual. Supondo que o comando emitido retorne resultados, o cursor é colocado automaticamente no primeiro registro quando o método **Open** do **Recordset** é chamado. Portanto, com o **Recordset** de exemplo, o cursor estaria no primeiro registro, "Uncle Bob's Organic Dried Pears".
