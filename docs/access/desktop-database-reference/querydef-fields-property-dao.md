# <a name="querydeffields-property-dao"></a>Propriedade QueryDef.Fields (DAO)


**Aplica-se a**: Access 2013, o Office 2013

Retorna uma coleção **[Fields](fields-collection-dao.md)** que representa todos os objetos **[Field](field-object-dao.md)** armazenados para o objeto especificado. Somente leitura.

## <a name="syntax"></a>Sintaxe

*expressão* . Campos

*expressão* Uma variável que representa um objeto **QueryDef** .
