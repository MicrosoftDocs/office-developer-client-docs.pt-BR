# <a name="positions-collection-ado-md"></a>Coleção Positions (ADO MD)


**Aplica-se a**: Access 2013, o Office 2013

Contém os objetos [Position](position-object-ado-md.md) que definem um ponto em um eixo.

## <a name="remarks"></a>Comentários

**Positions** é uma coleção padrão do ADO. Com as propriedades e os métodos de uma coleção, você pode executar os seguintes procedimentos:

  - Obter o número de objetos na coleção com a propriedade [Count](count-property-ado.md).

  - Retornar um objeto da coleção com a propriedade padrão [Item](item-property-ado.md).

  - Atualizar os objetos na coleção do provedor com o método [Refresh](refresh-method-ado.md).
