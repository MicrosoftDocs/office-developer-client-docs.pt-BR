# <a name="cancelrecordchange-macro-action"></a>Ação da macro CancelarAlteraçãodeRegistro


**Aplica-se a**: Access 2013, o Office 2013

Você pode usar a ação **CancelarAlteraçãodeRegistro** para cancelar alterações aplicadas a um registro em um bloco de dados **[CriarRegistro](createrecord-data-block.md)** ou **[EditarRegistro](editrecord-data-block.md)** antes que as alterações sejam submetidas.


> [!NOTE]
> [!OBSERVAçãO] A ação **CancelarAlteraçãodeRegistro** está disponível somente em Macros de Dados.



## <a name="remarks"></a>Comentários

Quando você chama a ação **CancelarAlteraçãodeRegistro**, o bloco de dados **CriarRegistro** ou **EditarRegistro** é encerrado imediatamente.
