# <a name="recordsetlastupdated-property-dao"></a>Propriedade Recordset.LastUpdated (DAO)


**Aplica-se ao:** Access 2013, Office 2013

Retorna a data e a hora da alteração mais recente feita em uma tabela base. Somente leitura **Variant**.

## <a name="syntax"></a>Sintaxe

*expressão* . LastUpdated

*expression* Uma variável que representa um objeto **Recordset**.

## <a name="remarks"></a>Comentários

As configurações de data e hora derivadas do computador no qual a tabela base foi criada ou atualizada pela última vez.
