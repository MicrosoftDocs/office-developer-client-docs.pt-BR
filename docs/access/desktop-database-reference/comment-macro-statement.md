# <a name="comment-macro-statement"></a>Instrução de macro Comentário


**Aplica-se a**: Access 2013, o Office 2013

A instrução **Comentário** pode ser usada para fornecer explicações em uma macro.

## <a name="remarks"></a>Comentários

Um **Comentário** pode ter até 1.000 caracteres. No entanto, por padrão, somente a primeira linha do comentário é exibida.
