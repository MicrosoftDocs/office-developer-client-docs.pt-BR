## <a name="database-requirements"></a>Requisitos do banco de dados

Para usar este exemplo, você deve ter:

  - Um servidor de banco de dados operacional com Microsoft® SQL Server versão 6.5 ou posterior.

  - Privilégios para criar o banco de dados e preenchê-lo com os dados de exemplo.

  - Verificação dos dados preenchidos pelo Enterprise Manager ou pelos utilitários ISQL (chamado Query Analyser no SQL Server 7.0).

Se você não tiver privilégios, o administrador do banco de dados poderá precisar configurar o sistema e dar-lhe permissão de acesso ao banco de dados ou configurar o banco de dados para você.
