# <a name="databaserelations-property-dao"></a>Propriedade Database. Relations (DAO)


**Aplica-se ao:** Access 2013, Office 2013

Retorna uma coleção **Relations** que contém todos os objetos **Relation** armazenados para o banco de dados especificado. Somente leitura.

## <a name="syntax"></a>Sintaxe

*expressão* . Relations

*expressão* Uma variável que representa um objeto **Database** .
