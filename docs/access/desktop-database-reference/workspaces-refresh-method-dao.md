# <a name="workspacesrefresh-method-dao"></a>Método Workspaces. Refresh (DAO)


**Aplica-se ao:** Access 2013, Office 2013

Não aceito para este objeto.

## <a name="syntax"></a>Sintaxe

*expressão* . Atualizado

*expressão* Uma variável que representa um **** objeto Workspaces.

## <a name="remarks"></a>Comentários

Você não pode usar o método **Refresh** com as coleções que não são persistentes, como as coleções **Connections**, **Databases**, **Recordsets**, **Workspaces** ou **QueryDefs** de um objeto **Connection**.
