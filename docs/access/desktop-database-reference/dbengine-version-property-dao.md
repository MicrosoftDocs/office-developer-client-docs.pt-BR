# <a name="dbengineversion-property-dao"></a>Propriedade DBEngine. Version (DAO)


**Aplica-se ao:** Access 2013, Office 2013

Retorna a versão do DAO atualmente em uso. **String** somente leitura.

## <a name="syntax"></a>Sintaxe

*expressão* . Versão

*expressão* Uma variável que representa um objeto **DBEngine** .

## <a name="remarks"></a>Comentários

O valor retornado é uma cadeia de caracteres que avalia um número de versão no formulário "major.minor". Por exemplo, "3.0". O número de versão do produto consiste no número de versão (3), em um ponto e no número de liberação (0).
