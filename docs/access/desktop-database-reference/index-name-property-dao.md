# <a name="indexname-property-dao"></a>Propriedade Index.Name (DAO)


**Aplica-se ao:** Access 2013, Office 2013

Retorna ou define o nome do objeto especificado. **String** de leitura/gravação se o objeto não foi acrescentado a uma coleção. **String** somente leitura se o objeto foi acrescentado a uma coleção.

## <a name="syntax"></a>Sintaxe

*expressão* . Tdomínio

*expressão* Uma variável que representa um objeto **index** .

## <a name="remarks"></a>Comentários

O comprimento máximo do nome do objeto **Index** é de 64 caracteres.
