# <a name="sql-aggregate-functions-sql"></a>Funções SQL agregadas (SQL)


**Aplica-se a**: Access 2013, o Office 2013

O uso das funções SQL agregadas permite determinar várias estatísticas nos conjuntos de valores. Use essas funções em uma consulta e expressões agregadas na propriedade **SQL** de um objeto **QueryDef** ou durante a criação de um objeto **Recordset** baseado em uma consulta SQL.
