# <a name="sql-aggregate-functions-sql"></a>Funções SQL agregadas (SQL)

**Aplica-se ao**: Access 2013, Office 2013

O uso das funções SQL agregadas permite determinar várias estatísticas nos conjuntos de valores. Use essas funções em uma consulta e expressões agregadas na propriedade **SQL** de um objeto **QueryDef** ou durante a criação de um objeto **Recordset** baseado em uma consulta SQL.

- [Função Avg](https://docs.microsoft.com/office/vba/access/concepts/criteria-expressions/avg-function-microsoft-access-sql)
- [Função Count](https://docs.microsoft.com/office/vba/access/concepts/criteria-expressions/count-function-microsoft-access-sql)
- [Funções First, Last](https://docs.microsoft.com/office/vba/access/concepts/miscellaneous/first-last-functions-microsoft-access-sql)
- [Funções Min, Max](https://docs.microsoft.com/office/vba/access/concepts/criteria-expressions/min-max-functions-microsoft-access-sql)
- [Funções StDev, StDevP](https://docs.microsoft.com/office/vba/access/concepts/criteria-expressions/stdev-stdevp-functions-microsoft-access-sql)
- [Função Sum](https://docs.microsoft.com/office/vba/access/concepts/criteria-expressions/sum-function-microsoft-access-sql)
- [Funções Var, VarP](https://docs.microsoft.com/office/vba/access/concepts/criteria-expressions/var-varp-functions-microsoft-access-sql)
