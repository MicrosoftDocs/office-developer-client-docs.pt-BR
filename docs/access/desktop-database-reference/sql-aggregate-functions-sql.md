# <a name="sql-aggregate-functions-sql"></a>Funções agregadas do SQL (SQL)


**Aplica-se a**: Access 2013, o Office 2013

O uso das funções SQL agregadas permite determinar várias estatísticas nos conjuntos de valores. Use essas funções em uma consulta e expressões agregadas na propriedade **SQL** de um objeto **QueryDef** ou durante a criação de um objeto **Recordset** baseado em uma consulta SQL.

- [Função AVG](https://msdn.microsoft.com/library/ff822755\(v=office.15\))

- [Função Count](https://msdn.microsoft.com/library/ff844748\(v=office.15\))

- [Funções First, Last](https://msdn.microsoft.com/library/ff197381\(v=office.15\))

- [Funções min, Max](https://msdn.microsoft.com/library/ff194490\(v=office.15\))

- [Funções StDev, StDevP](https://msdn.microsoft.com/library/ff197043\(v=office.15\))

- [Função SUM](https://msdn.microsoft.com/library/ff844764\(v=office.15\))

- [Funções var e VarP](https://msdn.microsoft.com/library/ff192105\(v=office.15\))
