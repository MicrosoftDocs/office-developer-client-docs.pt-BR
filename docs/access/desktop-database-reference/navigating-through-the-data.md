Agora que você executou um comando na fonte de dados e determinou que o conjunto de resultados contém dados, é possível navegar nos resultados usando os métodos e as propriedades fornecidos pelo objeto **Recordset**. 

Os tópicos a seguir descrevem como usar esses métodos e propriedades no **Recordset** de exemplo:

- [Outras maneiras de mover-se em um Recordset](more-ways-to-move-in-a-recordset.md)
- [Usando indicadores](using-bookmarks.md)
- [Usando páginas](using-pages.md)
- [Posicionamento no Recordset](recordset-positioning.md)
