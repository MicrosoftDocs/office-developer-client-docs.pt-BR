# <a name="exitforeachrecord-macro-action"></a>Ação da macro SairparaCadaRegistro


**Aplica-se ao:** Access 2013, Office 2013

Use a ação **SairparaCadaRegistro** para sair imediatamente de um bloco de dados **[ParaCadaRegistro](foreachrecord-data-block.md)**.


> [!NOTE]
> [!OBSERVAçãO] A ação **SairparaCadaRegistro** está disponível somente em Macros de Dados.
