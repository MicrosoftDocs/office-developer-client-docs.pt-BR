# <a name="leveldepth-property-ado-md"></a>Propriedade LevelDepth (ADO MD)


**Aplica-se a**: Access 2013 | Office 2013

Indica o número de níveis entre a raiz da hierarquia e um membro.

## <a name="remarks"></a>Comentários

Use a propriedade **LevelDepth** para determinar a distância do objeto [Member](member-object-ado-md.md)do nível raiz da hierarquia. O **LevelDepth** de um membro no nível raiz é 0. Isso corresponde à propriedade [Depth](depth-property-ado-md.md) de um objeto [Level](level-object-ado-md.md).
