# <a name="ltrim-function-access-custom-web-app"></a>Função LTrim (aplicativo Web personalizado do Access)

Retorna uma expressão Character depois de remover espaços em branco à esquerda.
  
> [!IMPORTANT]
> A Microsoft não recomenda mais criar e usar aplicativos Web do Access no SharePoint. Como alternativa, use o [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) para criar soluções de negócios sem código para a Web e dispositivos móveis. 
  
## <a name="syntax"></a>Sintaxe

 **LTrim** (*TextName*) 
  
A função **LTrim** contém o argumento a seguir. 
  
|**Nome do Argumento**|**Descrição**|
|:-----|:-----|
| *TextExpression*  <br/> |Uma expressão de cadeia de caracteres  <br/> |
