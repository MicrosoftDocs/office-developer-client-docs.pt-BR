# <a name="gotorecord-macro-action-access-custom-web-app"></a>Ação de Macro IrParaRegistro (aplicativo da web personalizado do Access)

Você pode usar a ação **IrParaRegistro** para tornar o registro especificado o registro atual em um modo de exibição. 
  
> [!IMPORTANT]
> [!IMPORTANTE] A Microsoft não recomenda mais criar e usar aplicativos Web do Access no SharePoint. Como alternativa, use o [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) para criar soluções de negócios sem código para a Web e dispositivos móveis. 
  
## <a name="setting"></a>Configuração

A ação **IrParaRegistro** tem os seguintes argumentos. 
  
|**Argumento da ação**|**Descrição**|
|:-----|:-----|
|**Objeto Record** <br/> |O registro para tornar o registro atual. Selecione **anterior**, **próximo**, **primeiro**ou **último**. O padrão é **próximo**.  <br/> |
