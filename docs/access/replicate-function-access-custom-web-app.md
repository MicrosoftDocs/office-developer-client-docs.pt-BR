# <a name="replicate-function-access-custom-web-app"></a>Replicar função (aplicativo da web personalizado do Access)

Repete um valor de texto em um número de vezes especificado.
  
> [!IMPORTANT]
> A Microsoft não recomenda mais criar e usar aplicativos Web do Access no SharePoint. Como alternativa, use o [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) para criar soluções de negócios sem código para a Web e dispositivos móveis. 
  
## <a name="syntax"></a>Sintaxe

 **Replicar** (*TextExpression*, *NumberOfTimes*) 
  
A função **replicar** contém os seguintes argumentos. 
  
|**Nome do argumento**|**Descrição**|
|:-----|:-----|
| *TextExpression*  <br/> |Uma expressão de texto.  <br/> |
| *NumberOfTimes*  <br/> |Um inteiro que especifica o número de vezes para repetir a *TextExpression* .  <br/> |
