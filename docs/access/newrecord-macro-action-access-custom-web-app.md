# <a name="newrecord-macro-action-access-custom-web-app"></a>Ação de macro NewRecord (aplicativo Web personalizado do Access)

Cria um novo registro em branco na exibição atual.
  
> [!IMPORTANT]
> A Microsoft não recomenda mais criar e usar aplicativos Web do Access no SharePoint. Como alternativa, use o [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) para criar soluções de negócios sem código para a Web e dispositivos móveis. 
  
## <a name="remarks"></a>Comentários

Você pode usar a **ação NewRecord** para adicionar um registro em branco no final do modo de exibição para que possa inserir novos dados. 
  
A **ação NewRecord** é equivalente a usar o **botão Adicionar** na barra de ações para adicionar um registro.
