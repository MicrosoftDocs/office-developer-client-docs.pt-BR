# <a name="and-access-custom-web-app"></a>E (aplicativo Web personalizado do Access)

Combina duas expressões booleanas e retorna TRUE quando ambas as expressões são TRUE. 
  
> [!IMPORTANT]
> A Microsoft não recomenda mais criar e usar aplicativos Web do Access no SharePoint. Como alternativa, use o [Microsoft PowerApps](https://powerapps.microsoft.com/en-us/) para criar soluções de negócios sem código para a Web e dispositivos móveis. 
  
## <a name="syntax"></a>Sintaxe

 *Boolean_expression* **E** *Boolean_expression* 
  
## <a name="return-value"></a>Valor de retorno

Retorna TRUE quando as duas expressões são TRUE.
  
## <a name="remarks"></a>Comentários

Quando mais de um operador lógico é usado em uma instrução, os operadores e são avaliados primeiro. Você pode alterar a ordem de avaliação usando parênteses.
