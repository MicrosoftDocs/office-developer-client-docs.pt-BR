### <a name="to-create-an-infopath-2003-compatible-form-template-in-infopath-with-visual-studio-tools-for-applications"></a><span data-ttu-id="330c9-108">Para criar um modelo de formulário compatíveis com o InfoPath 2003 no InfoPath com o Visual Studio Tools for Applications</span><span class="sxs-lookup"><span data-stu-id="330c9-108">To create an InfoPath 2003-compatible form template in InfoPath with Visual Studio Tools for Applications</span></span>

1. <span data-ttu-id="330c9-109">Inicie o Designer do InfoPath.</span><span class="sxs-lookup"><span data-stu-id="330c9-109">Start the InfoPath Designer.</span></span>
    
2. <span data-ttu-id="330c9-110">Clique na guia **arquivo** e, em seguida, clique em **Opções de formulário**.</span><span class="sxs-lookup"><span data-stu-id="330c9-110">Click the **File** tab, and then click **Form Options**.</span></span>
    
3. <span data-ttu-id="330c9-111">Na categoria **compatibilidade** , selecione o **Editor de formulários do InfoPath 2003**.</span><span class="sxs-lookup"><span data-stu-id="330c9-111">In the **Compatibility** category, select **InfoPath 2003 Editor Form**.</span></span>
    
4. <span data-ttu-id="330c9-112">Na categoria **de programação** , selecione **c# (compatível com o InfoPath 2003)** ou **Visual Basic (compatível com o InfoPath 2003)** na lista suspensa **linguagem de código do modelo de formulário** .</span><span class="sxs-lookup"><span data-stu-id="330c9-112">In the **Programming** category, select either **C# (InfoPath 2003 Compatible)** or **Visual Basic (InfoPath 2003 Compatible)** from the **Form template code language** drop-down list.</span></span>
