# <a name="create-a-form-template-using-the-infopath-2003-object-model"></a>Criar um modelo de formulário usando o modelo de objeto do InfoPath 2003

Os procedimentos neste tópico descrevem como criar um modelo de formulário que funciona com o modelo de objeto compatível com o InfoPath 2003.
  
> [!IMPORTANT]
> Além dos procedimentos abaixo, você também deve clicar na guia **arquivo** , clicar em **salvar como**e, em seguida, selecionar **modelo de formulário do InfoPath 2003** na caixa **salvar como tipo** , para salvar o modelo de formulário no formato de arquivo compatível com o InfoPath 2003. Além disso, para abrir modelos de formulário compatíveis com o InfoPath 2003 criados com o InfoPath, todos os usuários do InfoPath 2003 devem ter o .NET Framework 2,0 ou posterior instalado em seus computadores. 
  
### <a name="to-create-an-infopath-2003-compatible-form-template-in-infopath-with-visual-studio-tools-for-applications"></a>Para criar um modelo de formulário compatível com o InfoPath 2003 no InfoPath com o Visual Studio Tools for Applications

1. Inicie o designer do InfoPath.
    
2. Clique na guia **Arquivo** e depois em **Opções de Formulário**.
    
3. Na categoria **compatibilidade** , selecione **formulário do Editor do InfoPath 2003**.
    
4. Na categoria **programação** , selecione **C# (compatível com InfoPath 2003)** ou **Visual Basic (compatível com o InfoPath 2003)** na lista suspensa **linguagem de código de modelo de formulário** . 
    
5. Clique em **OK**.
    
6. Crie seu modelo de formulário e, em seguida, adicione manipuladores de eventos no Visual Studio 2012, conforme descrito em [Adicionar um manipulador de eventos usando o modelo de objeto do InfoPath 2003](how-to-add-an-event-handler-using-the-infopath-2003-object-model.md).
