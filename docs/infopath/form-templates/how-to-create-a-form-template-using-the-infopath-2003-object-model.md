# <a name="create-a-form-template-using-the-infopath-2003-object-model"></a>Criar um modelo de formulário usando o modelo de objeto do InfoPath 2003

Os procedimentos neste tópico descrevem como criar um modelo de formulário que funciona com o modelo de objeto do InfoPath 2003 compatíveis.
  
> [!IMPORTANT]
> Além dos procedimentos a seguir, você deve também clicar na guia **arquivo** , clique em **Salvar como**e, em seguida, selecione o **Modelo de formulário do InfoPath 2003** na caixa **Salvar como tipo** para salvar o modelo de formulário para o formato de arquivo compatíveis com o InfoPath 2003. Além disso, para abrir os modelos de formulário compatíveis com o InfoPath 2003 criados com o InfoPath, todos os usuários do InfoPath 2003 devem ter o .NET Framework 2.0 ou posterior instalado em seus computadores. 
  
### <a name="to-create-an-infopath-2003-compatible-form-template-in-infopath-with-visual-studio-tools-for-applications"></a>Para criar um modelo de formulário compatíveis com o InfoPath 2003 no InfoPath com o Visual Studio Tools for Applications

1. Inicie o Designer do InfoPath.
    
2. Clique na guia **arquivo** e, em seguida, clique em **Opções de formulário**.
    
3. Na categoria **compatibilidade** , selecione o **Editor de formulários do InfoPath 2003**.
    
4. Na categoria **de programação** , selecione **c# (compatível com o InfoPath 2003)** ou **Visual Basic (compatível com o InfoPath 2003)** na lista suspensa **linguagem de código do modelo de formulário** . 
    
5. Clique em **OK**.
    
6. Projete seu modelo de formulário e, em seguida, adicionar manipuladores de eventos no Visual Studio 2012, conforme descrito em [Adicionar um manipulador de eventos usando o modelo de objeto do InfoPath 2003](how-to-add-an-event-handler-using-the-infopath-2003-object-model.md).
